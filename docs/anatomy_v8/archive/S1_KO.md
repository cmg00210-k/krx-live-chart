# 제1장: 데이터와 API — 원천 자료

> **Stage 색상:** Slate Blue `#2C3E5C`
>
> CheeseStock의 모든 연산은 데이터에서 출발한다. 본 장에서는 단순히 어떤 데이터가
> 시스템에 유입되는지를 나열하는 것이 아니라, 각 데이터 원천이 **왜 학술적으로
> 필수적인지** — 즉, 제2장의 어떤 이론적 프레임워크가 해당 데이터의 존재를
> 요구하는지를 체계적으로 논증한다.

---

## 1.1 설계 철학: 이론 주도형 데이터 선정

일반적인 차트 플랫폼은 가용한 API를 기회주의적으로 수집한다. 그러나 CheeseStock은
이를 역전시킨다. 각 데이터 원천은 `core_data/` 라이브러리에 수록된 47개 학술 문서 중
특정 이론이 입력값으로 **요구**하기 때문에 추가되었다.

**원칙:** 이론적 소비자가 없는 데이터 파이프라인은 존재하지 않는다.

이로써 다음과 같은 추적 가능한 체인이 형성된다:

```
학술 이론 (제2장) ──requires──→ 데이터 원천 (제1장)
      ──feeds──→ 기술적 분석 공식 (제3장)
      ──renders──→ 차트 (제4장) ──delivers──→ 웹사이트 (제5장)
```

---

## 1.2 데이터 원천 목록과 학술적 근거

### 1.2.1 OHLCV 캔들 데이터 — pykrx / KRX Open API

OHLCV(시가, 고가, 저가, 종가, 거래량)는 기술적 분석의 근간이 되는 데이터이다.
다우이론(Dow Theory)의 가격 추세 분석, Nison(1991)의 캔들스틱 패턴 인식,
Bollinger(2001)의 변동성 밴드 산출 등 제3장의 모든 지표가 이 데이터를 소비한다.

| 학술 이론 | 학문 분야 | core_data 문서 | 필요 데이터 |
|-----------|----------|---------------|------------|
| 다우이론 | 기술적 분석 | doc 06 | 가격 추세, 고점/저점, 거래량 확인 |
| Nison 캔들스틱 | 일본식 기술적 분석 | doc 16 | 시가, 고가, 저가, 종가 — 몸통/그림자 비율 산출 |
| 볼린저밴드 | 통계학 | doc 02 | 종가의 이동평균 + 표준편차 |
| GARCH/EWMA | 계량경제학 | doc 02, 34 | 로그수익률 r_t = ln(P_t/P_{t-1}) |
| Hurst 지수 | 프랙탈 수학 | doc 01 | 종가 시계열의 R/S 분석 |
| ATR 정규화 | 위험 측정 | doc 06 | 참고범위(True Range): H, L, C_{t-1} |
| OBV | 행동재무학 | doc 04 | 종가 + 거래량 → 매집/분산 판단 |

OHLCV가 없으면 제3장의 어떤 공식도 실행할 수 없다. 이것이 기초 데이터 계층이다.

### 1.2.2 기업 재무제표 — DART (금융감독원)

금융감독원(FSS)의 전자공시시스템 DART는 상장기업의 재무제표를 제공한다.
현재 2,607개(전체 2,736개의 95%) 종목에 대한 재무데이터를 보유하고 있다.

| 학술 이론 | 학문 분야 | core_data 문서 | 필요 재무 항목 |
|-----------|----------|---------------|---------------|
| DCF 가치평가 | 기업재무 | doc 14 | 매출액, 당기순이익, 총자산, 자본총계 |
| PER/PBR/PSR | 재무관리 | doc 14 | EPS, BPS, 주당매출 |
| Merton 부도거리 | 신용위험 | doc 47 | 총자산, 총부채, 자본 → DD 산출 |
| Jensen-Meckling 대리인 | 대리인이론 | doc 33 | 잉여현금흐름, 부채비율 |
| ROE/ROA | 재무분석 | doc 14 | 당기순이익, 자본총계, 총자산 |

**데이터 신뢰도 3단계 체계:**

| 등급 | 원천 태그 | 표시 방식 | 이론 적용 가능성 |
|------|----------|----------|----------------|
| **1등급** | `'dart'` | 전체 지표 표시 | 이론 적용 가능 — 실제 재무데이터 |
| **2등급** | `'hardcoded'` | 전체 표시 + 경고 | 이론 적용 가능 — 수동 검증 데이터 |
| **3등급** | `'seed'` | 모든 지표 "—" | 이론 적용 **불가** — 의사난수 생성 데이터 |

> **인식론적 근거:** 시드 데이터에 Merton 부도거리(DD)나 DCF 모형을 적용하면
> 이론적으로 무의미한 결과가 산출된다. 이는 데이터-이론 계약의 위반이다.

### 1.2.3 거시경제 지표 — ECOS (한국은행)

한국은행 경제통계시스템(ECOS)은 기준금리, 국고채 수익률, 소비자물가지수 등
거시경제 변수를 제공한다.

| 학술 이론 | 학문 분야 | core_data 문서 | ECOS 데이터 |
|-----------|----------|---------------|------------|
| IS-LM 모형 | 거시경제학 | doc 30 | 기준금리(r), 통화량 |
| 테일러 준칙 | 거시 정책 | doc 30 | 정책금리 대 인플레이션 갭 |
| 수익률곡선 분석 | 채권 이론 | doc 35, 44 | 국고채 3년, 10년 수익률 |
| 경기순환 국면 | 섹터로테이션 | doc 29 | 선행종합지수(CLI) |
| 신용스프레드 | 신용위험 | doc 35, 47 | AA-/BBB- 회사채 수익률 |

ECOS 데이터는 `appWorker.js`의 거시복합점수(MCS v2)에 직접 입력되어,
패턴 신뢰도를 ±5% 범위에서 조정한다(제3장 3.4절 CONF-3, CONF-7 참조).

### 1.2.4 미국 거시경제 — FRED (연방준비은행)

미국 경제 데이터는 한국 시장에 대한 글로벌 맥락을 제공한다.
VIX(변동성지수), 미국 기준금리, 10년물 국채 수익률, 달러인덱스(DXY) 등이
교차시장 상관관계 분석(doc 28)의 핵심 입력값이다.

### 1.2.5 통계 데이터 — KOSIS

통계청(KOSIS)의 경기동행지수, 소비자기대지수, 소매판매 등은
Stovall(1996)의 섹터로테이션 이론에서 경기 국면 판별의 기초가 된다.

### 1.2.6 시장 미시구조 — KRX Open API

투자자별 매매동향, 공매도 현황, 파생상품 데이터는 Kyle(1985)의 정보거래자 모형,
Miller(1977)의 의견불일치 모형, Grossman-Stiglitz(1980)의 정보 역설 등
시장미시구조 이론의 실증적 검증에 필수적이다.

### 1.2.7 파생상품 데이터

VKOSPI(변동성지수), 선물 베이시스, 옵션 분석 데이터는
Black-Scholes-Merton 모형(doc 26, 45), 보유비용모형(doc 27),
분산위험프리미엄(VRP, doc 34) 등 파생상품 이론의 입력값이다.

### 1.2.8 실시간 시세 — 키움증권 OCX

WebSocket을 통한 실시간 데이터는 일중(intraday) 분석을 가능하게 한다.
Kyle(1985)의 가격발견 모형은 틱(tick) 수준에서도 적용되며,
시장미시구조 이론(doc 18, 36)은 본질적으로 일중 동학을 연구 대상으로 한다.

**이중 모드 아키텍처:** WebSocket 모드(키움증권 실시간)와 파일 모드(정적 JSON)
모두 **동일한** 분석 파이프라인을 실행한다. 데이터 원천만 다를 뿐,
적용되는 이론은 모드와 무관하다.

## 1.3 데이터 흐름도

```
┌─────────────────────────────────────────────────────────────┐
│                  제1장: 데이터 원천                           │
│                                                              │
│  OHLCV ──┬──→ 통계학 (시계열)      ──→ 기술적 지표           │
│          └──→ 물리학 (멱법칙)      ──→ 꼬리위험              │
│                                                              │
│  DART  ────→ 금융학 (가치평가)     ──→ 펀더멘털 분석          │
│                                                              │
│  ECOS  ──┬──→ 경제학 (IS-LM)      ──→ 거시 신뢰도 조정       │
│          └──→ 금융학 (수익률곡선)   ──→ 채권 신호             │
│                                                              │
│  FRED  ────→ 교차시장 (VIX, DXY)   ──→ 글로벌 체제           │
│                                                              │
│  KOSIS ────→ 경제학 (경기순환)     ──→ 섹터로테이션           │
│                                                              │
│  KRX   ──┬──→ 미시구조 (Kyle)      ──→ 수급 신호             │
│  수급    └──→ 심리학 (군집행동)    ──→ 센티먼트              │
│                                                              │
│  파생  ──┬──→ 금융학 (BSM)         ──→ 변동성 체제           │
│          └──→ 금융학 (보유비용)    ──→ 베이시스 분석          │
└─────────────────────────────────────────────────────────────┘
```

## 1.4 데이터 신뢰도와 인식론적 정합성

| 신뢰 등급 | 원천 태그 | 이론 적용 | 근거 |
|-----------|----------|----------|------|
| 완전 | `'dart'` | 가능 | 금융감독원 공시 데이터 |
| 검증 | `'hardcoded'` | 가능 (경고 표시) | 수동 검증 데이터 (삼성전자/SK하이닉스) |
| 없음 | `'seed'` | **불가** | 의사난수 — 모든 지표 "—" 표시 |
| 표본 | `'sample'` | **불가** | 플레이스홀더 — null 처리 |
| 미가용 | `'unavailable'` | **불가** | API 장애 — 우아한 성능 저하 |

데이터 경과(staleness) 검증: 30일 이상 미갱신 데이터는 `_staleDataSources`에
등록되어 신뢰도 조정 로직에서 자동 배제된다.

## 1.5 참조 연결표

| 데이터 원천 | 학문 분야 (제2장) | 공식 ID (제3장) | JS 함수 |

| OHLCV 종가 | 통계학 → 시계열 | I-01 ~ I-08 | calcMA, calcEMA, calcBB, calcRSI, calcMACD |

*제1장은 경험적 기반을 수립한다. CheeseStock에 유입되는 모든 데이터 포인트는

데이터 없이 구현되는 이론도 없다.*
