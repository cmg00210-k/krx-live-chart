1. Executive Summary

CheeseStock(cheesestock.co.kr)은 한국 주식시장(KOSPI/KOSDAQ) 전문 기술적 분석 웹 플랫폼으로, 30종 이상의 캔들/차트 패턴 자동 감지, 28개 개별 + 6개 복합 시그널 엔진, WLS 회귀 기반 수익률 예측, HTS급 9-Layer Canvas 시각화를 제공합니다.

빌드 시스템 없이(No webpack, no bundler) 순수 HTML/CSS/JavaScript 16개 모듈로 구성되며, TradingView Lightweight Charts v4.2.3을 차트 엔진으로 사용합니다. Cloudflare Pages에 정적 배포되어 무료/비로그인으로 운영됩니다.

3.3 Quality Scoring System (0-100)

각 감지된 패턴에 다요인 가중 합산 품질 점수를 산출합니다:

FactorWeightMeasures
Body25%Body size relative to ATR
Volume25%Volume / VMA(20) ratio
Trend20%Prior trend strength & direction alignment
Shadow15%Shadow conformity to theoretical ideal
Extra15%Additional confirmation factors (default 0.3)

Confluence Bonus: 패턴의 손절/목표가가 지지/저항 수준과 ATR 이내에 있으면 confidence += 3×strength.

Stop Loss: close ± ATR×2 (기본). 마루보주: low/high ± ATR×1.5.

Target: Bulkowski measured move — pattern height × 1.0.
