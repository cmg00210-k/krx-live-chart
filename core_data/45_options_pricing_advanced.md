### 핵심 주장 요약

```
1. CRR 이항트리는 BSM의 이산 버전이 아니라, BSM의 논리적 기반이다
   → Δt → 0에서 BSM으로 수렴 (기하브라운운동의 이산 근사)
   → 미국형 옵션, 배당, 경로의존성까지 자연스럽게 확장

2. 미국형 옵션의 조기행사는 "최적 정지 문제"이다
   → 각 노드에서 행사 vs 보유의 비교 → 자유경계 문제 (Doc 10 §2 연결)
   → KOSPI200 옵션은 유럽형이므로 이론적 완전성 차원의 학습

3. 풋-콜 패리티는 옵션 시장의 "열역학 제1법칙"이다
   → 위반 시 무위험 차익거래 기회 (전환/역전환)
   → 실무에서 거래비용·공매도 제약으로 no-arbitrage band 존재

4. 이색옵션은 한국 ELS/DLS 구조화 상품의 이론적 기반이다
   → 배리어 + 디지털 + 풋의 조합으로 step-down 구조 설명
   → 2024년 홍콩H지수 ELS 사태의 이론적 분석

5. Heston 확률변동성 모형은 BSM의 상수 σ 가정을 해제한 최초의 실용적 모형이다
   → ρ < 0 → 변동성 스큐의 이론적 기초 (Doc 34 §2 VRP 연결)
   → 특성함수 접근법으로 반닫힌해(semi-closed form) 도출

6. Dupire Local Vol은 시장 관측 IV surface와 무차익 일관적인 유일한 확산 모형이다
   → SVI (Doc 37 §2)는 IV surface의 "피팅", Dupire는 "가격결정 모형" — 역할 구분

7. 실물옵션은 NPV를 넘어서는 경영 유연성의 가치를 정량화한다
   → Doc 43 §5.5의 개관을 수학적으로 형식화
```

## 1. CRR 이항트리 (Cox-Ross-Rubinstein Binomial Tree)

### 1.1 모형의 직관과 구조

이항트리(Binomial Tree) 모형은 옵션 가격결정의 가장 직관적이면서도
강력한 이산시간(discrete-time) 프레임워크이다. Cox, Ross & Rubinstein
(1979)은 다음 질문에서 출발한다:

```
"주가가 한 기간에 u배 오르거나 d배 내리는 세상에서,
 옵션의 공정 가격은 무엇인가?"
```

이 단순한 이산 모형이 충분히 많은 기간으로 분할되면(Δt → 0),
연속시간 BSM 모형으로 수렴한다는 것이 CRR의 핵심 정리이다.

### 1.2 모형 파라미터

한 기간의 길이를 Δt = T/N (T: 만기, N: 분할 수)이라 하자.

```
상승 배율:  u = e^(σ√Δt)                               … (1.1)
하락 배율:  d = 1/u = e^(-σ√Δt)                         … (1.2)
위험중립 확률: p = (e^(rΔt) - d) / (u - d)              … (1.3)

여기서:
  σ: 연율화 변동성 (annualized volatility)
  r: 연속복리 무위험이자율 (continuously compounded risk-free rate)
  T: 만기까지 잔존기간 (년)
  N: 이항트리 분할 수 (number of steps)
  Δt: 한 단계의 시간 간격 = T/N
```

**파라미터 설정의 근거:**

u = e^(σ√Δt), d = 1/u로 설정하는 이유는 이 선택이 다음 세 조건을
동시에 만족시키기 때문이다:

```
조건 1 (Recombining): u·d = 1
  → 상승 후 하락 = 하락 후 상승 (경로 독립 → 노드 수 감소)
  → N-step tree의 종점 노드: N+1개 (비결합 시 2^N개)

조건 2 (모멘트 매칭): 이산 모형의 기대값과 분산이 연속 모형과 일치
  E[S_{t+Δt}/S_t] = p·u + (1-p)·d = e^(rΔt)           … (1.4)
  Var[ln(S_{t+Δt}/S_t)] = σ²·Δt + O(Δt²)              … (1.5)

조건 3 (위험중립): 할인된 주가가 마팅게일
  e^(-rΔt) · [p·u·S + (1-p)·d·S] = S                  … (1.6)
```

**무차익 조건 확인:**

0 < p < 1이 보장되려면 d < e^(rΔt) < u가 성립해야 한다.

```
d < e^(rΔt) < u
e^(-σ√Δt) < e^(rΔt) < e^(σ√Δt)

r·Δt < σ√Δt (오른쪽)
→ r < σ/√Δt

N이 충분히 크면 (Δt 충분히 작으면) 항상 성립.
실무에서 N ≥ 30이면 문제없음.
```

### 1.3 역방향 유도 (Backward Induction)

이항트리에서 옵션 가격은 만기(terminal nodes)에서 시작하여 역방향으로
계산한다. 시점 t, 노드 (i, j)에서의 옵션 가치를 V(i, j)라 하자
(i: 시간 인덱스, j: 상승 횟수).

**만기 시점 (i = N)의 페이오프:**

```
콜옵션: V(N, j) = max(S·u^j·d^(N-j) - K, 0)           … (1.7)
풋옵션: V(N, j) = max(K - S·u^j·d^(N-j), 0)           … (1.8)

j = 0, 1, 2, ..., N
```

**중간 노드 (유럽형):**

```
V(i, j) = e^(-rΔt) · [p · V(i+1, j+1) + (1-p) · V(i+1, j)]   … (1.9)

해석: 한 기간 후의 기대값을 무위험이자율로 할인
      = 위험중립 가격결정의 이산 버전
```

**중간 노드 (미국형):**

```
V(i, j) = max(exercise_value, continuation_value)

exercise_value   = max(S(i,j) - K, 0)  [콜]  또는  max(K - S(i,j), 0)  [풋]
continuation_value = e^(-rΔt) · [p · V(i+1, j+1) + (1-p) · V(i+1, j)]

→ 미국형은 §2에서 상세 논의
```

### 1.4 수치 예: KOSPI200 콜옵션 3-Step Tree

**문제 설정:**

```
기초자산: KOSPI200 지수, S₀ = 370
행사가격: K = 380 (OTM 콜)
변동성: σ = 18% (연율)
무위험이자율: r = 3.5% (연속복리, CD 91일 기준)
만기: T = 0.25년 (3개월, 1분기)
분할: N = 3

Δt = 0.25/3 = 0.08333 (≈ 1개월)
```

**파라미터 계산:**

```
u = e^(0.18 × √0.08333)
  = e^(0.18 × 0.28868)
  = e^(0.05196)
  = 1.05334                                             … (1.10)

d = 1/u = 1/1.05334 = 0.94937                           … (1.11)

e^(rΔt) = e^(0.035 × 0.08333)
        = e^(0.002917)
        = 1.00292                                        … (1.12)

p = (1.00292 - 0.94937) / (1.05334 - 0.94937)
  = 0.05355 / 0.10397
  = 0.51505                                              … (1.13)

확인: 0 < 0.51505 < 1 ✓
(1-p) = 0.48495
```

**주가 트리 (S 노드):**

```
시점:    t=0          t=1          t=2          t=3
                                             S(3,3)=432.11
                                S(2,2)=410.30
                   S(1,1)=389.74             S(3,2)=389.74
        S(0,0)=370              S(2,1)=370.00
                   S(1,0)=351.27             S(3,1)=351.27
                                S(2,0)=333.43
                                             S(3,0)=316.56

검증: S(3,3) = 370 × u³ = 370 × 1.05334³ = 370 × 1.16786 = 432.11
      S(3,2) = 370 × u²d = 370 × u (∵ u·d=1) = 370 × 1.05334 = 389.74
      S(3,1) = 370 × ud² = 370 × d = 370 × 0.94937 = 351.27
      S(3,0) = 370 × d³ = 370 × 0.85558 = 316.56
```

**옵션 가치 트리 (유럽형 콜):**

```
만기 노드 (i=3):
  V(3,3) = max(432.11 - 380, 0) = 52.11
  V(3,2) = max(389.74 - 380, 0) = 9.74
  V(3,1) = max(351.27 - 380, 0) = 0
  V(3,0) = max(316.56 - 380, 0) = 0

i=2 노드:
  V(2,2) = e^(-0.002917) × [0.51505 × 52.11 + 0.48495 × 9.74]
         = 0.99709 × [26.84 + 4.72]
         = 0.99709 × 31.56
         = 31.47

  V(2,1) = e^(-0.002917) × [0.51505 × 9.74 + 0.48495 × 0]
         = 0.99709 × 5.02
         = 5.00

  V(2,0) = e^(-0.002917) × [0.51505 × 0 + 0.48495 × 0]
         = 0

i=1 노드:
  V(1,1) = e^(-0.002917) × [0.51505 × 31.47 + 0.48495 × 5.00]
         = 0.99709 × [16.21 + 2.42]
         = 0.99709 × 18.63
         = 18.58

  V(1,0) = e^(-0.002917) × [0.51505 × 5.00 + 0.48495 × 0]
         = 0.99709 × 2.58
         = 2.57

i=0 (현재):
  V(0,0) = e^(-0.002917) × [0.51505 × 18.58 + 0.48495 × 2.57]
         = 0.99709 × [9.57 + 1.25]
         = 0.99709 × 10.82
         = 10.79
```

**결과:**

```
KOSPI200 콜옵션 가격 (3-step CRR) = 10.79 포인트

KRW 환산: 10.79 × 250,000원 = 2,697,500원 (프리미엄)

→ BSM 공식으로 직접 계산하면:
  d₁ = [ln(370/380) + (0.035 + 0.18²/2)×0.25] / (0.18×√0.25)
     = [-0.02667 + 0.01281] / 0.09
     = -0.1540
  d₂ = -0.1540 - 0.09 = -0.2440
  C_BSM = 370×N(-0.1540) - 380×e^(-0.035×0.25)×N(-0.2440)
        = 370×0.4388 - 376.67×0.4036
        = 162.36 - 152.07
        = 10.29

CRR 3-step (10.79) vs BSM (10.29): 차이 0.50 (4.9%)
→ N을 증가시키면 수렴 (§1.5에서 분석)
```

### 1.5 CRR의 BSM 수렴 — 증명 스케치

**정리 (CRR Convergence):**
N → ∞ (Δt → 0)일 때, CRR 이항트리의 유럽형 옵션 가격은
BSM 가격으로 수렴한다.

```
BSM = CRR의 Δt → 0 극한 (Doc 26 §1 참조)
```

**증명 스케치:**

N-step CRR에서 콜옵션 가격은 이항분포의 합으로 표현된다:

```
C_N = e^(-rT) Σ_{j=a}^{N} C(N,j) · p^j · (1-p)^(N-j) · (S·u^j·d^(N-j) - K)

여기서 a = min{j : S·u^j·d^(N-j) > K} (ITM 경계)         … (1.14)
       C(N,j) = N! / (j!(N-j)!)  (이항계수)
```

이를 정리하면:

```
C_N = S · Σ_{j=a}^{N} C(N,j) · p'^j · (1-p')^(N-j)
    - K·e^(-rT) · Σ_{j=a}^{N} C(N,j) · p^j · (1-p)^(N-j)

여기서 p' = p·u·e^(-rΔt)  (보조 확률)                     … (1.15)
```

핵심 수렴 논거:

```
단계 1: 이항분포 → 정규분포 (중심극한정리)
  N → ∞일 때, 이항분포 B(N, p)와 B(N, p')는 정규분포로 수렴.

단계 2: 정규분포의 누적분포함수 → N(d₁), N(d₂)
  Σ_{j=a}^{N} C(N,j) · p'^j · (1-p')^(N-j) → N(d₁)
  Σ_{j=a}^{N} C(N,j) · p^j · (1-p)^(N-j)  → N(d₂)

  여기서 d₁, d₂는 BSM의 표준 정의.                        … (1.16)

단계 3: 수렴 속도
  |C_N - C_BSM| = O(1/√N)  (odd-even 진동 포함)
  Richardson 외삽법 적용 시: O(1/N)으로 개선

  실무 기준:
    N = 50:   오차 < 1%
    N = 200:  오차 < 0.1%
    N = 1000: 기계적 정밀도 수준
```

**수렴의 직관적 의미:**

```
CRR 이항트리                     BSM 연속시간 모형
─────────────                    ──────────────
이산적 상승/하락 (u, d)     →    기하브라운운동 dS = μSdt + σSdW
이항분포 B(N, p)            →    정규분포 N(0, σ²T)
이산적 할인 e^(-rΔt)        →    연속 할인 e^(-rT)
역방향 유도 (N개 단계)      →    PDE 풀이 (BSM 편미분방정식)
위험중립확률 p               →    위험중립측도 Q (Girsanov 정리)
```

### 1.6 배당 처리

KOSPI200 지수 옵션의 경우, 지수를 구성하는 종목들의 배당을 연속
배당수익률 q로 근사한다.

**연속 배당수익률 모형:**

```
u, d는 동일. 위험중립확률만 수정:

p_div = (e^((r-q)Δt) - d) / (u - d)                     … (1.17)

q: 연속배당수익률 (KOSPI200의 경우 q ≈ 1.5-2.5%)

검증: q = 0이면 p_div = p (원래 공식)
```

**이산 배당 모형 (개별 주식 옵션용):**

```
배당 시점 직전에 주가를 조정:
  S_ex = S_cum - D  (현금배당 D만큼 하락)

또는 배당수익률 방식:
  S_ex = S_cum × (1 - δ)  (δ: 배당수익률)

이산 배당은 트리를 비결합(non-recombining)으로 만들 수 있으므로,
실무에서는 배당 시점 전후로 트리를 분리하거나,
S → S - PV(D)로 사전 조정한 뒤 결합 트리를 사용한다.    … (1.18)
```

### 1.7 CRR 변형과 확장

```
Jarrow-Rudd (1983):       등확률 모형
  u = e^((r-σ²/2)Δt + σ√Δt)
  d = e^((r-σ²/2)Δt - σ√Δt)
  p = 0.5 (정확히)
  → 트리 중심이 forward price에 정렬

Tian (1993):              3차 모멘트 매칭
  u, d, p를 왜도까지 매칭하도록 설정
  → fat-tail 효과 일부 반영

Leisen-Reimer (1996):     향상된 수렴
  Peizer-Pratt inversion으로 p, p' 직접 계산
  → O(1/N²) 수렴 속도 (CRR의 O(1/√N) 대비 대폭 개선)

삼항트리 (Trinomial Tree, Boyle 1986):
  상승(u), 유지(m=1), 하락(d) 세 경로
  p_u, p_m, p_d 세 확률 (합 = 1)
  → 명시적 유한차분법(explicit finite difference)과 동치
  → 2차원 이항트리로 경로의존 옵션 처리 가능                … (1.19)
```

## 2. 미국형 옵션 (American Options)

### 2.1 조기행사의 경제학

미국형 옵션(American option)은 만기 이전 어느 시점에서든 행사할 수
있는 권리를 가진다. 이 "조기행사 프리미엄"은 미국형과 유럽형 가격의
차이로 정의된다.

```
조기행사 프리미엄 (Early Exercise Premium, EEP):

EEP ≡ V_American - V_European ≥ 0                        … (2.1)

유럽형이 미국형의 하한: V_American ≥ V_European
  (행사 기회가 더 많으므로 가치가 더 낮을 수 없음)

특수한 경우:
  - 무배당 미국형 콜: EEP = 0 (조기행사 불합리)
  - 배당 있는 미국형 콜: EEP > 0 (배당 직전 행사 가능)
  - 미국형 풋: EEP > 0 (항상, 배당 유무 무관)
```

**무배당 미국형 콜의 조기행사 불합리성 — 증명:**

```
정리: 배당이 없는 미국형 콜옵션을 조기행사하는 것은 최적이 아니다.

증명:
  임의의 시점 t에서 조기행사 시 수취액 = S_t - K
  
  한편, 유럽형 콜의 하한:
    C_E ≥ S_t - K·e^(-r(T-t)) > S_t - K  (∵ r > 0)      … (2.2)
  
  그런데 V_American ≥ C_E이므로:
    V_American > S_t - K = 행사 수취액
  
  → 행사보다 보유(또는 시장 매도)가 항상 유리 ∎

직관: 조기행사 시 두 가지를 잃는다:
  (1) 시간가치: K원을 지금 지불 vs 나중에 지불 → 이자 이득 K·(1-e^(-r(T-t)))
  (2) 보험가치: ITM → OTM 전환 시 손실 제한 (옵션 보유 → max 손실 = 프리미엄)
```

### 2.2 미국형 풋의 최적행사경계

미국형 풋은 항상 양의 조기행사 프리미엄을 갖는다. 이는 "자유경계
문제(free boundary problem)"를 형성한다.

```
최적행사경계 (Optimal Exercise Boundary):

각 시점 t에서 임계 주가 S*(t)가 존재하여:
  S < S*(t) → 즉시 행사 (풋 가치 = K - S)
  S > S*(t) → 보유 계속 (풋 가치 > K - S)

S*(t)의 특성:
  - S*(T) = K (만기 시점에서는 ATM에서 경계)
  - S*(0) < K (시간이 남아있으면 경계가 낮아짐)
  - t → T일 때 S*(t) → K (위에서 수렴)
  - r → ∞이면 S*(t) → K (시간가치 상실 → 즉시행사 선호)   … (2.3)

연속시간에서의 조건:
  ∂V/∂t + (1/2)σ²S²∂²V/∂S² + rS·∂V/∂S - rV = 0
  (S > S*(t)인 영역에서)

  경계 조건:
  V(S*(t), t) = K - S*(t)        (value matching)
  ∂V/∂S|_{S=S*(t)} = -1          (smooth pasting)         … (2.4)

→ Doc 10 §2-3의 HJB 방정식과 최적 정지 문제의 금융 적용
```

### 2.3 이항트리에서의 미국형 옵션 계산

미국형 옵션은 이항트리에서 자연스럽게 처리된다. 각 노드에서
"행사" vs "보유"를 비교하면 된다.

```
미국형 풋 — 역방향 유도:

만기 노드:
  V(N, j) = max(K - S·u^j·d^(N-j), 0)                   … (2.5)

중간 노드 (핵심 — 유럽형과의 유일한 차이):
  continuation = e^(-rΔt) × [p·V(i+1,j+1) + (1-p)·V(i+1,j)]
  exercise     = max(K - S(i,j), 0)
  V(i, j) = max(continuation, exercise)                   … (2.6)

exercise > continuation인 노드:
  → 이 노드에서 조기행사가 최적
  → 이 노드들의 집합이 최적행사경계 S*(t)의 이산 근사
```

**수치 예: KOSPI200 미국형 풋 (동일 파라미터)**

```
S₀ = 370, K = 380, σ = 18%, r = 3.5%, T = 0.25, N = 3
(§1.4와 동일 파라미터, 풋으로 변경)

만기 노드 (풋):
  V(3,3) = max(380 - 432.11, 0) = 0
  V(3,2) = max(380 - 389.74, 0) = 0
  V(3,1) = max(380 - 351.27, 0) = 28.73
  V(3,0) = max(380 - 316.56, 0) = 63.44

i=2 노드:
  V(2,2): cont = 0.99709 × [0.51505×0 + 0.48495×0] = 0
          exer = max(380 - 410.30, 0) = 0
          V(2,2) = max(0, 0) = 0

  V(2,1): cont = 0.99709 × [0.51505×0 + 0.48495×28.73]
              = 0.99709 × 13.93 = 13.89
          exer = max(380 - 370, 0) = 10
          V(2,1) = max(13.89, 10) = 13.89 [보유]

  V(2,0): cont = 0.99709 × [0.51505×28.73 + 0.48495×63.44]
              = 0.99709 × [14.80 + 30.77] = 0.99709 × 45.57 = 45.44
          exer = max(380 - 333.43, 0) = 46.57
          V(2,0) = max(45.44, 46.57) = 46.57 [행사!] ← 조기행사 최적

i=1 노드:
  V(1,1): cont = 0.99709 × [0.51505×0 + 0.48495×13.89]
              = 0.99709 × 6.74 = 6.72
          exer = max(380 - 389.74, 0) = 0
          V(1,1) = max(6.72, 0) = 6.72 [보유]

  V(1,0): cont = 0.99709 × [0.51505×13.89 + 0.48495×46.57]
              = 0.99709 × [7.15 + 22.58] = 0.99709 × 29.73 = 29.64
          exer = max(380 - 351.27, 0) = 28.73
          V(1,0) = max(29.64, 28.73) = 29.64 [보유]

i=0:
  V(0,0): cont = 0.99709 × [0.51505×6.72 + 0.48495×29.64]
              = 0.99709 × [3.46 + 14.37] = 0.99709 × 17.83 = 17.78
          exer = max(380 - 370, 0) = 10
          V(0,0) = max(17.78, 10) = 17.78 [보유]
```

**결과 비교:**

```
미국형 풋 가격 = 17.78
유럽형 풋 가격 (BSM) = 17.45 (풋-콜 패리티로 계산: §3에서 검증)
조기행사 프리미엄 EEP = 17.78 - 17.45 = 0.33 (1.9%)

조기행사 발생 노드: V(2,0) — S = 333.43 (ITM 깊음)
→ K - S = 46.57을 즉시 수취하는 것이 보유가치 45.44보다 유리
→ 깊은 ITM 풋에서 시간가치보다 이자 수익(K원 즉시 수취)이 큼
```

### 2.4 배당 직전 콜의 조기행사

무배당 미국형 콜은 조기행사가 불합리하지만, 배당이 있으면 달라진다.

```
배당 직전 행사 조건 (Discrete Dividend D):

콜옵션 보유자는 배당 직전(ex-dividend date 전날)에
행사 여부를 판단한다.

행사 시 수취: S_cum - K (배당 전 주가에서 행사)
보유 시 가치: C(S_cum - D, K, T-t_ex, σ, r)
             (배당락 후 주가에 대한 옵션 가치)

행사가 최적일 조건:
  D > C(S_cum - D, K, T-t_ex, σ, r) - (S_cum - D - K)
  ≈ D > K × (1 - e^(-r(T-t_ex)))                        … (2.7)

직관: 배당(D)이 행사가에 대한 이자 비용보다 크면 행사
  → 고배당 주식 + 깊은 ITM + 잔여만기 짧을 때 발생

수치 예:
  S = 400, K = 350 (깊은 ITM), D = 5, T-t_ex = 0.08 (약 1개월), r = 3.5%
  임계값 = 350 × (1 - e^(-0.035×0.08)) = 350 × 0.00280 = 0.98
  D = 5 > 0.98 → 조기행사 합리적
```

### 2.5 한국 시장 적용

```
KOSPI200 옵션:
  유럽형(European-style) — 조기행사 불가
  → 미국형 가격결정 이론이 직접 적용되지 않음
  → 그러나 이항트리 프레임워크는 유럽형에도 동일하게 작동
     (단지 중간 노드에서 행사 비교를 하지 않을 뿐)

ELW (주식워런트):
  유럽형(European-style) — 발행 조건에 따라 결정
  → 대부분의 한국 ELW는 유럽형

미국형이 적용되는 한국 상품:
  (1) 일부 장외 옵션 (OTC)
  (2) 전환사채(CB)의 전환권 — 미국형 콜옵션으로 분석
  (3) 실물옵션 (§6에서 상세 논의)

이론적 완전성:
  미국형 가격결정은 유럽형의 상위 집합(superset)이다.
  유럽형 = 미국형에서 "조기행사 금지" 제약을 추가한 것.
  → 모형 일반성을 위해 미국형을 먼저 이해하는 것이 바람직.
```

## 3. Put-Call Parity 심층 (풋-콜 패리티)

### 3.1 패리티 관계의 도출

풋-콜 패리티는 동일한 기초자산·행사가·만기를 가진 유럽형 콜과 풋의
가격 관계를 규정하는 무차익 조건(no-arbitrage condition)이다.

```
풋-콜 패리티 (배당수익률 q 포함):

C - P = S·e^(-qT) - K·e^(-rT)                           … (3.1)

여기서:
  C: 유럽형 콜 가격
  P: 유럽형 풋 가격
  S: 현재 기초자산 가격
  K: 행사가격
  r: 무위험이자율 (연속복리)
  q: 연속배당수익률
  T: 만기까지 잔존기간

q = 0인 경우 (무배당):
  C - P = S - K·e^(-rT)                                  … (3.2)
```

**복제 포트폴리오에 의한 증명:**

```
포트폴리오 A: 유럽형 콜 + K·e^(-rT)원 투자 (만기 시 K원)
포트폴리오 B: 유럽형 풋 + e^(-qT)주의 기초자산 (배당 재투자 → 만기 시 1주)

만기(T) 시점의 페이오프:

  S_T > K인 경우:
    A: (S_T - K) + K = S_T
    B: 0 + S_T = S_T

  S_T ≤ K인 경우:
    A: 0 + K = K    ← 아, 이것은 틀림. 수정:
    
  재도출:
  포트폴리오 A: 유럽형 콜 + K원짜리 무위험채권
    만기 가치: max(S_T - K, 0) + K = max(S_T, K)

  포트폴리오 B: 유럽형 풋 + 기초자산 1주
    만기 가치: max(K - S_T, 0) + S_T = max(S_T, K)

  A와 B의 만기 가치가 항상 동일:
    → 무차익 조건에 의해 현재 가치도 동일                    … (3.3)

  C + K·e^(-rT) = P + S·e^(-qT)
  ∴ C - P = S·e^(-qT) - K·e^(-rT)  ∎
```

### 3.2 패리티 검증 — §1.4 수치 예 확인

```
§1.4에서 계산한 유럽형 콜:
  C_CRR = 10.79, C_BSM = 10.29

BSM 기준으로 풋 가격 확인 (q = 0 가정):
  P = C - S + K·e^(-rT)
    = 10.29 - 370 + 380 × e^(-0.035 × 0.25)
    = 10.29 - 370 + 380 × 0.99127
    = 10.29 - 370 + 376.68
    = 16.97                                               … (3.4)

검증 (BSM 풋 직접 계산):
  P = K·e^(-rT)·N(-d₂) - S·N(-d₁)
    = 376.68 × N(0.2440) - 370 × N(0.1540)
    = 376.68 × 0.5964 - 370 × 0.5612
    = 224.64 - 207.64
    = 17.00 (반올림 차이)

C - P = 10.29 - 17.00 = -6.71
S - K·e^(-rT) = 370 - 376.68 = -6.68 (반올림 차이로 근사 일치) ✓
```

### 3.3 전환 차익거래 (Conversion Arbitrage)

풋-콜 패리티가 위반되면 무위험 차익거래 기회가 발생한다.

```
정의:
  synthetic forward = long call + short put (동일 K, T)

전환 (Conversion):
  조건: C - P > S·e^(-qT) - K·e^(-rT) + 거래비용
  → 콜이 패리티 대비 과대평가 (또는 풋이 과소평가)

  전략:
    Long put + Long stock + Short call                     … (3.5)
    → 만기 시 페이오프: K (S_T 무관)
    → 초기 비용: P + S - C
    → 수익: K·e^(-rT) - (P + S - C)
         = (C - P) - (S - K·e^(-rT)) > 0

역전환 (Reversal):
  조건: C - P < S·e^(-qT) - K·e^(-rT) - 거래비용
  → 콜이 패리티 대비 과소평가 (또는 풋이 과대평가)

  전략:
    Long call + Short stock + Short put                    … (3.6)
    → 만기 시 페이오프: -K (S_T 무관, 주식 반환 후 K 지급/수취)
    → 초기 수취: S - P - C (공매도 대금 + 풋 매도 수취)
    → 수익: (S - K·e^(-rT)) - (C - P) > 0
```

### 3.4 실무적 패리티 밴드

완전한 무차익 균형은 이론적 이상이다. 실무에서는 거래비용, 공매도
제약, 배당 불확실성으로 인해 패리티 "밴드"가 존재한다.

```
패리티 밴드:

|C - P - (S·e^(-qT) - K·e^(-rT))| ≤ ε                   … (3.7)

ε의 구성요소:
  (1) 매수·매도 호가 스프레드 (bid-ask spread)
  (2) 증권거래세 (한국: 0.18% 매도 시)
  (3) 공매도 비용 (대차수수료: Doc 40 §5 참조)
  (4) 마진/증거금 비용
  (5) 배당 추정 오차

KRX에서의 경험적 밴드:
  KOSPI200 옵션: ε ≈ 0.3-0.5 포인트 (ATM 기준)
  → 기관 투자자/시장조성자가 적극적으로 차익거래 수행
  → 개인 투자자에게는 거래비용으로 인해 비실현적

→ Doc 37 §6 UOA(Unusual Options Activity) 참조:
  패리티 위반의 지속적 확대는 정보 비대칭 신호일 수 있음.
  다만, 위반 탐지 로직 자체는 Doc 37의 영역이므로
  본 문서에서는 이론적 기초만 제시한다.
```

### 3.5 미국형 풋-콜 부등식

```
미국형 옵션에서는 등식이 아닌 부등식이 성립:

S - K ≤ C_A - P_A ≤ S - K·e^(-rT)                       … (3.8)

증명 스케치:
  왼쪽 부등식: 미국형 풋의 조기행사 프리미엄으로 인해
    P_A > P_E이므로 C_A - P_A < C_E - P_E = S - K·e^(-rT)

  오른쪽 부등식: 미국형 콜의 조기행사 프리미엄 ≥ 0이므로
    C_A ≥ C_E이고, 조기행사 시 최소 S - K 수취 가능

→ 미국형 풋의 EEP가 클수록 (깊은 ITM + 고금리)
  부등식 밴드가 넓어진다.
```

## 4. 이색옵션 (Exotic Options)

### 4.1 개관

이색옵션(Exotic Options)은 표준형(Vanilla) 유럽형/미국형 옵션의
확장이다. 페이오프 구조가 복잡하거나, 경로에 의존하거나, 추가 조건이
부여된 옵션을 총칭한다.

```
이색옵션 분류:

경로 독립 (Path-Independent):
  - Digital/Binary: 고정 금액 페이오프
  - 갭 옵션 (Gap Option): 트리거 ≠ 행사가
  - 선택 옵션 (Chooser Option): 미래 시점에 콜/풋 선택

경로 의존 (Path-Dependent):
  - Barrier: 기초자산이 배리어 도달 여부에 따른 조건부
  - Asian: 평균 가격 기반 페이오프
  - Lookback: 최대/최소 가격 기반 페이오프

복합/다중자산:
  - 복합 옵션 (Compound): 옵션에 대한 옵션
  - 레인보우 (Rainbow): 다중 기초자산
  - 스프레드 (Spread): 두 자산 가격 차이 기반

한국 시장 관련:
  ELS(Equity-Linked Securities)는 Barrier + Digital + Put의 조합
  DLS(Derivative-Linked Securities)는 원자재/금리 기초
```

### 4.2 Barrier 옵션 (배리어 옵션)

기초자산 가격이 특정 배리어(barrier) 수준에 도달하면 옵션이 활성화
(knock-in)되거나 소멸(knock-out)되는 구조이다.

**4종류의 기본 배리어:**

```
Down-and-Out (하방 소멸):
  S가 배리어 H (H < S₀) 아래로 내려가면 옵션 소멸
  조건: min_{0≤t≤T} S(t) > H

Down-and-In (하방 활성):
  S가 배리어 H 아래로 내려가야 옵션 활성화
  조건: min_{0≤t≤T} S(t) ≤ H

Up-and-Out (상방 소멸):
  S가 배리어 H (H > S₀) 위로 올라가면 옵션 소멸
  조건: max_{0≤t≤T} S(t) < H

Up-and-In (상방 활성):
  S가 배리어 H 위로 올라가야 옵션 활성화
  조건: max_{0≤t≤T} S(t) ≥ H
```

**In-Out 패리티:**

```
Knock-In + Knock-Out = Vanilla                           … (4.1)

Down-and-In Call + Down-and-Out Call = Vanilla Call
Up-and-In Put + Up-and-Out Put = Vanilla Put

→ 하나만 알면 다른 하나는 패리티로 계산 가능
```

**Rubinstein-Reiner (1991) 해석해 — Down-and-Out 콜:**

```
H < K인 경우 (배리어가 행사가 아래):

C_do = S·e^(-qT)·[N(x₁) - (H/S)^(2λ)·N(x₃)]
     - K·e^(-rT)·[N(x₁-σ√T) - (H/S)^(2λ-2)·N(x₃-σ√T)]  … (4.2)

여기서:
  x₁ = [ln(S/K)] / (σ√T) + λ·σ√T
  x₃ = [ln(H²/(S·K))] / (σ√T) + λ·σ√T
  λ = (r - q + σ²/2) / σ²

H ≥ K인 경우 (배리어가 행사가 이상):
  C_do = S·e^(-qT)·[N(x₂) - (H/S)^(2λ)·N(x₄)]
       - K·e^(-rT)·[N(x₂-σ√T) - (H/S)^(2λ-2)·N(x₄-σ√T)]
       + Rebate항  (배리어 도달 시 환불금)                    … (4.3)

  x₂ = [ln(S/H)] / (σ√T) + λ·σ√T
  x₄ = [ln(H/S)] / (σ√T) + λ·σ√T

검증: H → 0이면 C_do → C_vanilla (배리어가 의미없으므로)
검증: H → S이면 C_do → 0 (즉시 소멸)
```

**이항트리에서의 배리어 옵션:**

```
단순 접근: 각 노드에서 S(i,j) vs H 비교
  S(i,j) ≤ H → V(i,j) = Rebate (또는 0)
  S(i,j) > H → 일반 역방향 유도

문제: 이산 모니터링 편향 (specification error)
  → 트리의 노드가 배리어를 정확히 지나지 않으면 오차 발생
  → Broadie, Glasserman & Kou (1997) 보정:
    H_eff = H × e^(±β₁·σ√Δt)                             … (4.4)
    β₁ = -ζ(1/2)/√(2π) ≈ 0.5826
    (+: down barrier, -: up barrier)
```

### 4.3 Asian 옵션 (아시안 옵션)

페이오프가 기초자산의 특정 기간 평균 가격에 기반하는 경로의존
옵션이다.

**두 유형:**

```
고정행사가 Asian (Fixed-Strike):
  콜: max(A(0,T) - K, 0)
  풋: max(K - A(0,T), 0)

부유행사가 Asian (Floating-Strike):
  콜: max(S_T - A(0,T), 0)
  풋: max(A(0,T) - S_T, 0)

여기서 A(0,T)는 평균 가격:

산술 평균:  A_arith = (1/n) Σ_{i=1}^{n} S(t_i)           … (4.5)
기하 평균:  A_geom = (Π_{i=1}^{n} S(t_i))^(1/n)          … (4.6)
```

**기하평균 해석해 (Kemna-Vorst 1990):**

```
기하평균 Asian 콜의 가치:

로그 기하평균: ln(A_geom) = (1/n) Σ ln(S(t_i))

GBM 하에서 ln(S(t_i))는 정규분포 → 합도 정규분포
→ 기하평균의 분포를 명시적으로 계산 가능

C_geom = e^(-rT) · E_Q[max(A_geom - K, 0)]

BSM과 동일한 형태의 해:
  C_geom = e^(-rT)[F_A · N(d₁_A) - K · N(d₂_A)]         … (4.7)

여기서:
  F_A = S·exp((r̃ - q̃)T̃) (기하평균의 선도가격)
  r̃ = (r + q + σ²/6) / 2  (수정 이자율)
  q̃ = 0
  T̃ = T
  σ_A = σ / √3  (기하평균의 변동성 축소)
  d₁_A = [ln(F_A/K) + σ_A²T/2] / (σ_A√T)
  d₂_A = d₁_A - σ_A√T

핵심: σ_A = σ/√3 < σ → 평균화 효과로 변동성 감소
      → Asian 옵션이 Vanilla보다 저렴
```

**산술평균 — 해석해 부재:**

```
산술평균은 로그정규 확률변수의 합이므로 분포를 닫힌 형태로
표현할 수 없다. 세 가지 접근법:

(1) 몬테카를로 시뮬레이션 (MC):
    경로를 다수 생성하여 평균 페이오프의 기대값 추정
    제어변량(control variate): 기하평균 해석해를 활용
    → C_arith ≈ C_MC + (C_geom_analytic - C_geom_MC)    … (4.8)

(2) 모멘트 매칭 (Turnbull-Wakeman 1991):
    산술평균의 1차·2차 모멘트를 로그정규로 근사
    → BSM 형태의 근사 공식 사용

(3) 편미분방정식 (PDE):
    상태변수 (S, I, t)에서 I = Σ S(t_i): 누적 합
    → 3차원 PDE, 유한차분법으로 수치 풀이
```

### 4.4 Lookback 옵션 (룩백 옵션)

페이오프가 기초자산의 관측 기간 최대/최소 가격에 기반한다.
"Buy at the low, sell at the high" 전략을 옵션으로 구현한 것이다.

**부유행사가 Lookback (Goldman-Sosin-Gatto 1979):**

```
부유행사가 Lookback 콜:
  페이오프 = S_T - min_{0≤t≤T} S(t)
  → "최저가에 매수, 만기에 매도"

부유행사가 Lookback 풋:
  페이오프 = max_{0≤t≤T} S(t) - S_T
  → "최고가에 매도, 만기에 매수"

Goldman-Sosin-Gatto (1979) 해석해 — 부유행사가 콜:

C_lookback = S·e^(-qT)·N(a₁) - S·e^(-qT)·(σ²/(2(r-q)))·N(-a₁)
           - m·e^(-rT)·N(a₂) + m·e^(-rT)·(σ²/(2(r-q)))·(S/m)^(-2(r-q)/σ²)·N(a₃)

여기서:                                                    … (4.9)
  m = min_{0≤t≤T} S(t) (현재까지의 최소값, 초기에는 m = S₀)
  a₁ = [ln(S/m) + (r - q + σ²/2)T] / (σ√T)
  a₂ = a₁ - σ√T
  a₃ = [ln(S/m) - (r - q - σ²/2)T] / (σ√T)

특성: Lookback > Vanilla (항상)
      → 최대/최소 보장이므로 프리미엄이 상당히 비쌈
```

**고정행사가 Lookback:**

```
고정행사가 Lookback 콜:
  페이오프 = max(max_{0≤t≤T} S(t) - K, 0)
  → "최고가가 행사가보다 높으면 그 차이만큼 수취"

고정행사가 Lookback 풋:
  페이오프 = max(K - min_{0≤t≤T} S(t), 0)
  → "최저가가 행사가보다 낮으면 그 차이만큼 수취"

→ 고정행사가 Lookback은 배리어 옵션과 부유행사가 Lookback의
  조합으로 분해 가능 (Conze & Viswanathan 1991)           … (4.10)
```

### 4.5 Digital/Binary 옵션 (디지털 옵션)

페이오프가 0 또는 고정 금액인 가장 단순한 이색옵션이다.

```
Cash-or-Nothing 콜:
  페이오프 = Q  (S_T > K일 때)
           = 0  (S_T ≤ K일 때)

  가격: V_CON_call = Q · e^(-rT) · N(d₂)                 … (4.11)

Cash-or-Nothing 풋:
  페이오프 = Q  (S_T < K일 때)
           = 0  (S_T ≥ K일 때)

  가격: V_CON_put = Q · e^(-rT) · N(-d₂)                 … (4.12)

Asset-or-Nothing 콜:
  페이오프 = S_T  (S_T > K일 때)
           = 0    (S_T ≤ K일 때)

  가격: V_AON_call = S · e^(-qT) · N(d₁)                 … (4.13)

Vanilla과의 관계:
  Vanilla Call = Asset-or-Nothing Call - K × Cash-or-Nothing Call
  C = S·e^(-qT)·N(d₁) - K·e^(-rT)·N(d₂)
    = V_AON_call - K · (V_CON_call / Q) (Q=1일 때)        … (4.14)

Delta와의 관계:
  ∂C/∂K = -e^(-rT)·N(d₂) = -V_CON_call/(Q·e^(-rT))·e^(-rT)
  → 행사가에 대한 미분이 Cash-or-Nothing의 가격과 직결

  Breeden-Litzenberger (1978):
  ∂²C/∂K² = e^(-rT)·f_Q(K)
  → 이중 미분이 위험중립 확률밀도함수를 복원
  → Doc 26 §2.4에서 상세 논의                              … (4.15)
```

### 4.6 한국 ELS 구조: Step-Down Barrier + Digital Coupon

한국 시장에서 가장 대중적인 구조화 상품인 ELS(Equity-Linked
Securities)는 이색옵션의 복합체이다.

```
대표적 3년 만기 Step-Down ELS 구조:

기초자산: KOSPI200 + HSCEI (또는 S&P500)
만기: 3년 (6회 조기상환 관찰일: 6/12/18/24/30/36개월)

Step-Down 배리어:
  6개월:  95% (기초자산이 초기가의 95% 이상이면 조기상환)
  12개월: 90%
  18개월: 85%
  24개월: 80%
  30개월: 75%
  36개월: 70%                                             … (4.16)

상환 구조:
  조기상환 시: 원금 + 연 X% 쿠폰 (예: 연 6-8%)
  만기 시 (조기상환 미발생):
    - 기초자산 ≥ 70%: 원금 + 쿠폰
    - 기초자산 < 70% (Knock-In):
      원금 × (기초자산 수익률) → 원금 손실 가능

이색옵션 분해:
  ELS = 원금 + Σ(Digital Coupon_i) - Down-and-In Put

  Digital Coupon_i:
    관찰일 i에서 S(t_i)/S(0) ≥ Barrier_i이면 쿠폰 지급
    = Cash-or-Nothing (discrete monitoring)

  Down-and-In Put:
    만기까지 한 번이라도 S(t)/S(0) < 70%이면 풋 활성화
    활성화된 풋의 행사가 = 100% (원금)
    → Knock-In 발생 시 원금 손실 위험 노출                   … (4.17)
```

**2024년 홍콩H지수 ELS 사태의 이론적 분석:**

```
사건: 2023-2024년 HSCEI 급락 (8,000 → 5,300 수준)
      → Knock-In 배리어(통상 50-60%) 대거 도달
      → 투자자 원금 손실 (최대 40-50%)

이론적 교훈:
(1) Worst-of 구조: 다중 기초자산 중 최악의 성과에 연동
    → 상관관계 ρ가 낮을수록 Knock-In 확률 급증
    → ρ의 비대칭성: 하락장에서 상관관계가 수렴 (Doc 28 §3 참조)

(2) 스큐 미반영: 발행 시 BSM (상수 σ) 가격결정
    → 실제 스큐 고려 시 Down-and-In Put 가치가 더 높았음
    → 발행사에게 유리한 가격 설정 (→ Doc 37 §2 SVI 참조)

(3) 모형 위험(Model Risk):
    GBM 가정 하에서 30-40% 하락 확률은 매우 낮지만
    실제로는 꼬리 위험이 과소평가됨 (Doc 12 EVT 참조)
```

## 5. Heston (1993) 확률변동성 모형

### 5.1 모형 설정

BSM의 가장 심각한 가정 위배는 "일정한 변동성"이다. Heston (1993)은
변동성 자체를 확률과정으로 모형화한 최초의 실용적 해를 제시했다.

```
Heston (1993) 확률변동성 모형:

주가 동학:
  dS(t) = μ·S(t)·dt + √v(t)·S(t)·dW₁(t)               … (5.1)

분산 동학 (CIR 과정):
  dv(t) = κ(θ - v(t))·dt + ξ·√v(t)·dW₂(t)              … (5.2)

상관관계:
  Cov(dW₁, dW₂) = ρ·dt                                  … (5.3)

파라미터:
  v(t): 순간분산 (instantaneous variance), σ(t)² 해당
  κ:    평균회귀속도 (mean-reversion speed)
        → 분산이 장기 수준으로 복귀하는 속도
        → 큰 κ → 빠른 회귀, 작은 κ → 느린 회귀
  θ:    장기분산 (long-run variance)
        → v(t)가 장기적으로 수렴하는 수준
        → √θ = 장기 변동성
  ξ:    변동성의 변동성 (volatility of volatility, vol of vol)
        → 분산 과정 자체의 불확실성
  ρ:    주가-변동성 상관계수
        → 보통 ρ < 0 (주가 하락 시 변동성 상승: "leverage effect")
        → KOSPI200: ρ ≈ -0.5 ~ -0.7 (경험적)

  μ:    주가의 드리프트 (위험중립 측도에서는 r - q로 대체)
```

### 5.2 Feller 조건

분산 v(t)가 CIR 과정을 따를 때, v(t) = 0에 도달하면 √v(t) = 0이
되어 확산항이 사라진다. Feller 조건은 v(t)가 0에 도달하지 않을
충분조건이다.

```
Feller 조건 (1951):

  2κθ > ξ²                                               … (5.4)

물리적 의미:
  - 평균회귀력(κθ)이 변동성의 불확실성(ξ²/2)보다 강하면
    분산이 0에 도달하지 않음
  - 위반 시 v(t) = 0에 도달 가능 → 반사(reflecting) 처리 필요

경험적 값 (KOSPI200 기준):
  κ ≈ 2-5 (반감기 ≈ ln(2)/κ ≈ 0.14-0.35년)
  θ ≈ 0.03-0.06 (장기 변동성 ≈ 17-24%)
  ξ ≈ 0.3-0.8

Feller 체크:
  κ=3, θ=0.04, ξ=0.5:  2×3×0.04 = 0.24 > 0.25? No (위반!)
  κ=3, θ=0.04, ξ=0.4:  2×3×0.04 = 0.24 > 0.16? Yes ✓
  κ=5, θ=0.04, ξ=0.5:  2×5×0.04 = 0.40 > 0.25? Yes ✓

→ 시장 데이터에서 칼리브레이션하면 Feller 조건이 종종 위반된다.
  이 경우 v(t)가 0에 도달할 수 있지만, 옵션 가격 공식 자체는
  여전히 유효하다 (Andersen & Piterbarg 2007).              … (5.5)
```

### 5.3 특성함수와 반닫힌해

Heston 모형의 강력한 점은 주가의 로그에 대한 **특성함수(characteristic
function)**를 명시적으로 구할 수 있다는 것이다. 이로부터 옵션 가격을
Fourier 역변환으로 계산한다.

```
로그주가: x = ln(S)

위험중립 측도 Q 하에서의 특성함수:

φ(u; x, v, τ) = E_Q[e^{iu·x_T} | x_t = x, v_t = v]

         = exp(C(u,τ) + D(u,τ)·v + iu·x)                … (5.6)

여기서 τ = T - t (잔존만기), 그리고:

C(u,τ) = (r-q)·iu·τ + (κθ/ξ²)·[(κ-ρξiu-d)τ - 2·ln((1-g·e^(-dτ))/(1-g))]

D(u,τ) = [(κ-ρξiu-d)/ξ²] · [(1-e^(-dτ))/(1-g·e^(-dτ))]

d = √((ρξiu - κ)² + ξ²(iu + u²))

g = (κ - ρξiu - d) / (κ - ρξiu + d)                     … (5.7)
```

**Gil-Pelaez (1951) Fourier 역변환에 의한 옵션 가격:**

```
유럽형 콜 가격:

C = S·e^(-qτ)·P₁ - K·e^(-rτ)·P₂                        … (5.8)

여기서:

P_j = 1/2 + (1/π) ∫₀^∞ Re[e^(-iu·ln(K)} · φ_j(u) / (iu)] du

φ₁(u) = φ(u-i; x, v, τ) / φ(-i; x, v, τ)
       (리스크조정 특성함수)

φ₂(u) = φ(u; x, v, τ)
       (위험중립 특성함수)                                 … (5.9)

수치 적분:
  ∫₀^∞ → ∫₀^{u_max} (충분히 큰 u_max, 보통 100-200)
  Gauss-Laguerre 구적법 또는 Simpson 규칙 적용
  → 32-64 포인트로 충분한 정밀도 달성 가능
```

**BSM과의 관계:**

```
BSM 특성함수:
  φ_BSM(u) = exp(iu(r-q-σ²/2)τ - u²σ²τ/2)

Heston에서 ξ → 0 (vol of vol → 0)이면:
  v(t) → θ (분산이 상수)
  φ_Heston → φ_BSM (with σ² = θ)

→ BSM은 Heston의 특수한 경우 (ξ = 0)                      … (5.10)
```

### 5.4 파라미터와 시장 현상의 관계

```
ρ < 0 (레버리지 효과):
  주가 하락 → 변동성 상승 (dW₁ < 0일 때 dW₂ > 0 경향)
  → OTM 풋의 IV 상승 → 변동성 스큐(skew) 생성
  → ρ가 더 음수일수록 스큐가 가파름

  → Doc 34 §2 VRP와의 관계:
    Heston의 ρ < 0이 VRP(IV > RV)의 이론적 기초를 제공.
    위험중립 측도(Q)에서의 분산 기대값이 실제 측도(P)보다 높아지는
    메커니즘을 ρ가 설명한다. VRP 공식 자체는 Doc 34 참조.

ξ (vol of vol):
  ξ가 클수록 → 변동성 미소(smile)가 뚜렷
  ξ → 0이면 → BSM (평평한 IV)
  → 스큐의 "폭"을 결정

κ (평균회귀속도):
  κ가 클수록 → 변동성 충격의 지속성이 낮음
  → 장기 옵션 IV < 단기 옵션 IV (평균회귀 효과)
  → 변동성 기간구조의 형태를 결정                           … (5.11)

θ (장기분산):
  → IV surface의 전반적 수준(level)을 결정
  → √θ ≈ ATM 장기 IV
```

### 5.5 칼리브레이션 (Calibration)

시장에서 관측되는 IV surface를 Heston 모형으로 피팅하여 파라미터
(v₀, κ, θ, ξ, ρ)를 추정한다.

```
칼리브레이션 절차:

Step 1: 시장 데이터 수집
  옵션 가격 C_mkt(K_i, T_j) (행사가 i, 만기 j 조합)
  또는 IV σ_mkt(K_i, T_j)

Step 2: 목적함수 설정
  min_{v₀,κ,θ,ξ,ρ} Σ_{i,j} w_{ij} · (C_model(K_i,T_j; Θ) - C_mkt(K_i,T_j))²

  w_{ij}: 가중치 (ATM에 높은 가중, 유동성 기반)
  Θ = (v₀, κ, θ, ξ, ρ)                                   … (5.12)

Step 3: 최적화
  비선형 최소자승법: Levenberg-Marquardt, differential evolution,
  또는 particle swarm optimization
  
  제약 조건:
    v₀ > 0, κ > 0, θ > 0, ξ > 0
    -1 < ρ < 0 (보통, 레버리지 효과)
    Feller 조건 2κθ > ξ² (선택적 — 위반 허용하기도 함)

Step 4: 검증
  (a) IV surface 잔차 분석: |σ_model - σ_mkt| < 0.5%이면 양호
  (b) 날짜별 안정성: 파라미터가 급변하면 과적합 의심
  (c) 외삽 합리성: 관측 범위 밖의 가격이 합리적인지 확인

KOSPI200 경험적 파라미터 범위:
  v₀ ≈ 0.02-0.08 (현재 분산, √v₀ ≈ 14-28%)
  κ  ≈ 1-8
  θ  ≈ 0.03-0.06
  ξ  ≈ 0.3-0.8
  ρ  ≈ -0.7 ~ -0.3                                       … (5.13)
```

### 5.6 Heston 모형의 한계와 확장

```
한계:
(1) 단일 인자: 변동성이 하나의 확률과정만으로 구동
    → 변동성 기간구조의 유연성이 부족
(2) 대칭적 vol-of-vol: ξ가 상수
    → 극단적 스큐 상황에서 피팅 어려움
(3) 점프 미포함: 연속 확산만 가정
    → 급격한 변동성 점프 설명 불가

확장 모형:
  Bates (1996): Heston + 주가 점프 (Merton jump 추가)
    dS = ... + J·dN(t)  (J: 점프 크기, N: Poisson 과정)
    → Doc 34 §3 점프-확산 모형과 결합

  Double Heston: 2개의 독립적 분산 과정
    dv₁ = κ₁(θ₁-v₁)dt + ξ₁√v₁ dW₃
    dv₂ = κ₂(θ₂-v₂)dt + ξ₂√v₂ dW₄
    σ²(t) = v₁(t) + v₂(t)
    → 단기/장기 변동성 성분 분리

  SABR (Hagan et al. 2002): α(변동성), β(CEV 지수), ρ, ν(vol-of-vol)
    → FX 옵션, 금리 옵션에서 표준 모형
    → KOSPI200 옵션에도 적용 가능

  3/2 모형 (Lewis 2000):
    dv = κv(θ-v)dt + ξv^(3/2)dW₂
    → vol-of-vol이 분산 수준에 비례 → 극단 스큐 설명력 향상
                                                           … (5.14)
```

## 6. 실물옵션 (Real Options)

### 6.1 NPV에서 실물옵션으로

Doc 43 §5.5에서 실물옵션의 개관과 유형을 소개했다. 본 절에서는
각 유형의 수학적 형식화를 제시한다.

```
핵심 통찰 (Doc 43 §5.5 요약):

  프로젝트의 "확장된 가치" = 정적 NPV + 옵션 가치

  정적 NPV: "지금 투자" vs "영원히 투자 안 함"의 이분법
  옵션 가치: 미래에 의사결정을 변경할 수 있는 유연성의 가치

  → 옵션 가치 ≥ 0 (항상)
  → 불확실성이 클수록 (σ↑) 옵션 가치 증가 (금융옵션과 동일)
  → 되돌릴 수 없는(irreversible) 투자일수록 대기의 가치가 큼
```

### 6.2 연기옵션 (Option to Delay)

McDonald & Siegel (1986)의 최적 투자 타이밍 모형이다.

```
모형 설정:

프로젝트 가치 V(t)가 GBM을 따른다:
  dV = α·V·dt + σ·V·dW                                   … (6.1)

투자비용: I (상수, 또는 결정적 함수)
NPV(t) = V(t) - I

무한 만기 미국형 콜옵션으로 해석:
  F(V) = max_τ E[e^(-rτ)(V(τ) - I)]                     … (6.2)

  여기서 τ는 정지시간(stopping time), r은 할인율

해 (영구 미국형 콜):

  F(V) = { (V/V*)^β · (V* - I)    if V < V*
         { V - I                    if V ≥ V*             … (6.3)

  최적 투자 임계값:
  V* = β/(β-1) · I                                        … (6.4)

  β = 1/2 - (r-δ)/σ² + √[(r-δ)/σ² - 1/2]² + 2r/σ²)    … (6.5)

  여기서 δ = r - α (편의수익률, convenience yield)

핵심 결과:
  V* > I (항상, β/(β-1) > 1)
  → NPV > 0 (즉, V > I)이더라도 V < V*이면 투자를 미루는 것이 최적
  → "positive NPV rule is not sufficient"

수치 예 (반도체 설비 투자):
  I = 1조원, σ = 30%, r = 5%, δ = 3%
  β = 0.5 - 0.02/0.09 + √(0.02/0.09-0.5)² + 2×0.05/0.09
    = 0.5 - 0.222 + √(0.222-0.5)² + 1.111)
    = 0.5 - 0.222 + √(0.0772 + 1.111)
    = 0.5 - 0.222 + √1.188
    = 0.278 + 1.090
    = 1.368

  → 주의: β ≈ 1.368이면 β/(β-1) = 1.368/0.368 = 3.72
  → V* = 3.72 × 1조 = 3.72조원
  → NPV = V - I > 0이려면 V > 1조이면 되지만,
    실물옵션 고려 시 V > 3.72조여야 투자 최적!
  → 대기의 가치가 매우 크다 (불확실성 σ = 30%이 높기 때문)    … (6.6)
```

### 6.3 폐기옵션 (Option to Abandon)

운영 중인 프로젝트를 잔존가치(salvage value) A에 매도(포기)할 수
있는 미국형 풋옵션이다.

```
모형:
  운영가치 V(t): GBM (dV = αVdt + σVdW)
  잔존가치 A: 상수 (또는 감가상각 함수)

  F_abandon(V) = max(A - V + 운영 계속 가치, 0)

미국형 풋 유비:
  기초자산 = V(t) (프로젝트 운영가치)
  행사가 = A (잔존가치)
  → §2의 미국형 풋 프레임워크와 동일                        … (6.7)

최적 폐기 경계:
  V < V*_abandon이면 즉시 폐기
  V*_abandon < A (항상)

  → V*_abandon에서 smooth pasting 조건:
    ∂F/∂V|_{V=V*} = -1                                   … (6.8)

이항트리 구현:
  각 기간에 운영 현금흐름 CF를 수취하면서
  폐기 여부를 결정하는 역방향 유도

  V(i,j) = max(
    A_i,                                               [폐기]
    CF·Δt + e^(-rΔt)·[p·V(i+1,j+1) + (1-p)·V(i+1,j)]  [계속]
  )                                                       … (6.9)
```

### 6.4 확장옵션 (Option to Expand)

초기 투자 후, 시장 상황이 좋으면 규모를 확대할 수 있는 유럽형/
미국형 콜옵션이다.

```
모형:
  초기 프로젝트: 가치 V, 투자비용 I
  확장 시: 가치 (1+γ)V, 추가 투자비용 I_E

  확장 옵션의 가치 = max(γV - I_E, 0)
  → 콜옵션: 기초자산 = γV, 행사가 = I_E                    … (6.10)

BSM 적용 (유럽형 확장옵션):
  C_expand = γ·V·e^(-δT)·N(d₁) - I_E·e^(-rT)·N(d₂)

  d₁ = [ln(γV/I_E) + (r-δ+σ²/2)T] / (σ√T)
  d₂ = d₁ - σ√T                                          … (6.11)

프로젝트의 확장된 가치:
  V_total = NPV + C_expand
  → NPV < 0이더라도, C_expand가 충분히 크면 투자 정당화
  → R&D, 파일럿 프로젝트, 전략적 투자의 이론적 근거
```

### 6.5 전환옵션 (Option to Switch)

Kulatilaka (1993)의 이중연료 산업 보일러 모형을 일반화한 것이다.

```
모형:
  두 가지 운영 모드: 모드 A (가치 V_A(t))와 모드 B (가치 V_B(t))
  전환비용: K_AB (A→B), K_BA (B→A)

  V_A(t)와 V_B(t)가 상관된 GBM을 따른다:
    dV_A = μ_A V_A dt + σ_A V_A dW_A
    dV_B = μ_B V_B dt + σ_B V_B dW_B
    Corr(dW_A, dW_B) = ρ_AB                               … (6.12)

  전환옵션 가치:
    F_switch = max(V_B - V_A - K_AB, 0)  (현재 A 운영 중)
    → 교환 옵션 (Margrabe 1978)의 확장

Margrabe (1978) 교환 옵션 공식 (전환비용 = 0):
  C_exchange = V_B·N(d₁) - V_A·N(d₂)
  d₁ = [ln(V_B/V_A) + σ_AB²T/2] / (σ_AB√T)
  d₂ = d₁ - σ_AB√T
  σ_AB = √(σ_A² + σ_B² - 2ρ_AB·σ_A·σ_B)                 … (6.13)
```

### 6.6 복합옵션 (Compound Option)과 순차적 투자

제약/바이오 산업의 임상시험처럼 순차적 투자 결정이 이루어지는
경우, 실물옵션은 "옵션에 대한 옵션"(compound option)으로 모형화된다.

```
2단계 복합 콜옵션:

Phase I: 비용 I₁ 투자 → Phase II 진입 권리(옵션) 획득
Phase II: 비용 I₂ 투자 → 프로젝트 가치 V 획득

가치 구조:
  F₁ = max(F₂(V) - I₁, 0)  (Phase I의 가치)
  F₂ = max(V - I₂, 0)       (Phase II의 가치)             … (6.14)

Geske (1979) 해석해 — 복합 유럽형 콜:
  C_compound = V·M(a₁, b₁; ρ_t) - I₂·e^(-rT₂)·M(a₂, b₂; ρ_t)
             - I₁·e^(-rT₁)·N(a₂)

  M(): 이변량 정규 누적분포함수
  ρ_t = √(T₁/T₂)
  a₁, a₂, b₁, b₂는 V*, I₁, I₂, T₁, T₂에 의존            … (6.15)

바이오 산업 적용 (한국 예시):
  Phase I (전임상 → Phase I 임상):
    비용 I₁ ≈ 100-300억원, 기간 T₁ ≈ 2년, 성공확률 ≈ 60%
  Phase II (Phase I → Phase II 임상):
    비용 I₂ ≈ 300-800억원, 기간 T₂ ≈ 3년, 성공확률 ≈ 40%
  Phase III → 상업화:
    비용 I₃ ≈ 1,000-3,000억원, 기간 T₃ ≈ 3년, 성공확률 ≈ 65%

  → 3단계 복합옵션: F₁(F₂(F₃(V)))
  → 각 단계에서 중단 = 포기옵션 행사
  → 바이오 기업의 높은 PER은 이 복합옵션 가치를 반영              … (6.16)
```

### 6.7 실물옵션과 금융옵션의 대응관계

```
| 금융옵션 변수 | 실물옵션 변수 | 비고 |
|--------------|-------------|------|
| 기초자산 가격 S | 프로젝트 현금흐름의 현재가치 V | 관측 어려움 |
| 행사가격 K | 투자비용 I | 비교적 확정적 |
| 만기 T | 투자 기회의 유효 기간 | 경쟁에 의해 축소 가능 |
| 변동성 σ | 프로젝트 가치의 불확실성 | 추정 난이도 높음 |
| 무위험이자율 r | 할인율 (WACC 또는 r) | |
| 배당수익률 q/δ | 대기의 기회비용 (경쟁 진입, 현금흐름 상실) | |

실무적 난점:
  (1) V의 관측: 금융옵션은 S를 시장에서 관측하지만,
      실물옵션의 V는 추정치 (DCF에 의존)
  (2) σ의 추정: 프로젝트 변동성을 어떻게 측정할 것인가?
      → 유사 기업 주가 변동성, 시뮬레이션, 경영진 판단
  (3) 완전복제 불가: 금융옵션의 위험중립 가격결정은 복제 가능성에 기반
      실물 프로젝트는 시장에서 거래되지 않으므로 복제 불가
      → 균형 가격결정(equilibrium pricing) 또는
        MAD (Marketed Asset Disclaimer) 접근법
        (Copeland & Antikarov 2001)                        … (6.17)
```

### 6.8 실물옵션과 최적 제어의 관계

```
연결:
  실물옵션 = 최적 정지 문제(optimal stopping problem)의 금융 적용
  
  → Doc 10 §1 HJB 방정식:
    sup_u [L·V + U(u)] = 0
    여기서 L은 미분 작용소, U는 효용 함수

  실물옵션에서:
    제어 변수 u = {투자, 대기, 포기, 확장} (이산적 선택)
    상태 변수 = (V(t), t)
    가치 함수 F(V, t):

    max{행사 가치, 대기 가치} = max{(V-I), (∂F/∂t + (1/2)σ²V²∂²F/∂V² + (r-δ)V∂F/∂V - rF + ...)}

  → 자유경계 문제: §2.2의 미국형 풋과 수학적으로 동치        … (6.18)

차이점:

## 7. Local Volatility (Dupire 1994, 로컬 변동성)

### 7.1 BSM에서 Local Volatility으로

BSM은 변동성 σ가 상수라고 가정하지만, 시장에서 관측되는 IV는
행사가(K)와 만기(T)에 따라 다르다 (변동성 미소/스큐). 이 불일치를
해소하는 두 가지 접근이 있다:

```
(1) 확률변동성 (Stochastic Volatility):
    σ 자체를 확률과정으로 모형화 (§5 Heston)
    → 두 개의 확률인자 (S, v) → 불완전시장

(2) 로컬 변동성 (Local Volatility):
    σ를 S와 t의 결정적 함수 σ_loc(S, t)로 설정
    → 하나의 확률인자 (S) → 완전시장 (유일한 마팅게일 측도)
    → 시장의 모든 유럽형 옵션 가격과 정확히 일치 가능         … (7.1)
```

### 7.2 Dupire 공식

Dupire (1994)는 유럽형 옵션의 시장 가격으로부터 로컬 변동성
함수를 추출하는 공식을 도출했다.

```
Dupire (1994) Local Volatility Formula:

σ_loc(K, T)² = [∂C/∂T + (r-q)·K·∂C/∂K + q·C]
               / [(1/2)·K²·∂²C/∂K²]                     … (7.2)

여기서:
  C = C_mkt(K, T): 시장에서 관측된 유럽형 콜 가격 (행사가 K, 만기 T)
  ∂C/∂T: 콜 가격의 만기에 대한 편미분 (캘린더 스프레드)
  ∂C/∂K: 콜 가격의 행사가에 대한 편미분
  ∂²C/∂K²: 콜 가격의 행사가에 대한 이중 편미분 (버터플라이 스프레드)

물리적 해석:
  분자 ∂C/∂T: 시간 가치 (theta에 대응)
  분모 K²·∂²C/∂K²: 볼록성 (gamma에 대응)

  → "시간 감쇠를 감마로 나눈 것" = 로컬 분산
```

**Dupire 공식의 도출 스케치:**

```
전제: 기초자산이 다음 확산 과정을 따른다고 가정:
  dS = (r-q)·S·dt + σ_loc(S,t)·S·dW                     … (7.3)

Fokker-Planck 방정식 (전방 Kolmogorov 방정식):
  유럽형 콜 가격 C(K,T)는 다음을 만족:

  ∂C/∂T = (1/2)·σ_loc(K,T)²·K²·∂²C/∂K² - (r-q)·K·∂C/∂K - q·C

이 PDE를 σ_loc²에 대해 풀면 (7.2) 식이 된다.

경계 조건:
  C(K,0) = max(S₀ - K, 0)  (만기 T=0에서 내재가치)
  C(0,T) = S₀·e^(-qT)      (K → 0)
  C(K,T) → 0                (K → ∞)                      … (7.4)
```

### 7.3 IV로 표현한 Dupire 공식

실무에서는 옵션 가격 대신 내재변동성(IV)으로 표현하는 것이 편리하다.

```
σ_iv = σ_iv(K, T)로 표현된 IV surface가 주어졌을 때:

σ_loc(K,T)² = σ_iv² + 2σ_iv·T·∂σ_iv/∂T + 2(r-q)K·T·σ_iv·∂σ_iv/∂K
              ──────────────────────────────────────────────────────
              [1 + K·d₁·√T·∂σ_iv/∂K]²
              + K²·T·σ_iv·[∂²σ_iv/∂K² - d₁·(∂σ_iv/∂K)²·√T]

여기서 d₁ = [ln(S/K) + (r-q+σ_iv²/2)T] / (σ_iv√T)       … (7.5)

→ SVI (Doc 37 §2)로 IV surface를 먼저 매끄럽게 피팅한 뒤,
  위 공식으로 수치 미분하여 Local Vol surface를 구하는 것이
  표준적 워크플로우이다.

주의:
  ∂²C/∂K² < 0이면 σ_loc² < 0 → 무차익 조건 위반
  → IV surface가 무차익 조건을 만족해야 Local Vol이 존재
  → Doc 37 §2의 SVI 무차익 조건이 여기서 중요한 이유
```

### 7.4 수치적 구현

```
실무적 구현 절차:

Step 1: 시장 IV 데이터 수집
  옵션 호가에서 (K_i, T_j, σ_iv(K_i, T_j)) 추출

Step 2: IV surface 보간/피팅
  SVI (Doc 37 §2) 또는 스플라인 보간
  → 매끄러운 σ_iv(K, T) 함수 구성

Step 3: 수치 미분
  ∂C/∂T ≈ [C(K, T+ΔT) - C(K, T-ΔT)] / (2ΔT)
  ∂C/∂K ≈ [C(K+ΔK, T) - C(K-ΔK, T)] / (2ΔK)
  ∂²C/∂K² ≈ [C(K+ΔK, T) - 2C(K,T) + C(K-ΔK, T)] / (ΔK²)  … (7.6)

Step 4: σ_loc(K, T) 계산 (Dupire 공식 적용)

Step 5: 가격결정
  로컬 변동성 하의 옵션 가격을 유한차분법(FDM) 또는
  몬테카를로(MC)로 계산
  → 이색옵션, 경로의존 옵션에 특히 유용

정밀도 이슈:
  (a) 수치 미분의 불안정성: ΔK, ΔT가 작을수록 노이즈 증폭
  (b) 데이터 부족: ITM/OTM 깊은 영역에서 유동성 부족
      → 외삽에 대한 정규화(regularization) 필요
  (c) 캘린더 차익: T₁ < T₂에서 ∂C/∂T < 0이면 위반
      → IV surface 피팅 단계에서 방지해야 함
```

### 7.5 Heston과 Local Vol의 관계

```
Gyöngy (1986) Lemma:

  "확률변동성 모형의 마진(marginal) 분포는
   특정 로컬 변동성 모형과 동일하다."

  구체적으로:
  Heston 모형이 생성하는 S_T의 분포와 동일한 분포를 생성하는
  로컬 변동성 함수가 존재한다:

  σ_loc(K,T)² = E[v(T) | S(T) = K]                       … (7.7)

  즉, 로컬 변동성은 S(T) = K 조건부 기대분산이다.

함의:
(1) 유럽형 옵션에 대해 두 모형이 동일한 가격을 생성
(2) 그러나 경로의존 옵션에서는 가격이 다를 수 있음
    → 로컬 변동성: 결정적 σ(S,t) → 미래 스큐가 평탄해짐
    → 확률변동성: 확률적 v(t) → 미래 스큐가 유지됨
    → "Local Vol flattens forward skew" 문제
(3) 실무에서 두 모형을 결합한 SLV(Stochastic Local Volatility)를 사용
    → σ(S,t) × √v(t) 형태                                … (7.8)
```

### 7.6 SVI와 Dupire의 역할 분담

```
SVI (Doc 37 §2):
  목적: IV surface의 파라미터화 (parametric fitting)
  입력: 시장 IV 데이터
  출력: 매끄러운 IV 함수 σ_iv(K, T)
  용도: 보간, 시각화, 무차익 검증, 리스크 보고

Dupire Local Vol (본 문서 §7):
  목적: 가격결정 모형 (pricing engine)
  입력: SVI (또는 기타 방법)로 피팅된 IV surface
  출력: σ_loc(K, T) 함수
  용도: 이색옵션 가격결정, 헤징, 시나리오 분석

워크플로우:
  시장 데이터 → SVI 피팅 → IV surface → Dupire → Local Vol → Pricing

→ SVI는 "데이터 처리", Dupire는 "모형" — 역할이 다르므로 중복이 아니다.
```

```
§1 CRR 이항트리:

  V(i,j) = e^(-rΔt)[p·V(i+1,j+1) + (1-p)·V(i+1,j)]

§2 미국형 옵션:
  V(i,j) = max(exercise, continuation)
  EEP = V_American - V_European ≥ 0

§3 풋-콜 패리티:
  C - P = S·e^(-qT) - K·e^(-rT)

§4 이색옵션:
  In-Out Parity: C_in + C_out = C_vanilla
  Asian σ_A = σ/√3 (기하평균)
  ELS = 원금 + Σ(Digital Coupon) - Down-and-In Put

§5 Heston:
  dS = μS dt + √v S dW₁
  dv = κ(θ-v)dt + ξ√v dW₂, Corr = ρ
  Feller: 2κθ > ξ²

§6 실물옵션:
  V* = [β/(β-1)]·I (최적 투자 임계값)
  프로젝트 가치 = NPV + 옵션 가치

§7 Dupire Local Vol:
  σ_loc² = (∂C/∂T + (r-q)K·∂C/∂K + qC) / ((1/2)K²·∂²C/∂K²)

§8 KRX:
