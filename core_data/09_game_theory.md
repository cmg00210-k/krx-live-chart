# 09. 게임 이론 — Game Theory in Financial Markets

> 시장은 참여자들의 전략적 상호작용이다.
> 게임 이론은 "상대방이 무엇을 할 것인가"를 수학적으로 모형화한다.

---

## 1. 기초 프레임워크

### 1.1 게임의 구성요소

```
게임 G = (N, S, u)

N = {1, 2, ..., n}  — 참여자(player) 집합
S = S₁ × S₂ × ... × Sₙ  — 전략(strategy) 공간
u = (u₁, u₂, ..., uₙ)  — 보수(payoff) 함수

uᵢ: S → ℝ  (참여자 i의 보수)
```

금융 시장의 게임 모형화:
- 참여자: 개인투자자, 기관, 알고리즘 트레이더, 마켓메이커
- 전략: 매수, 매도, 관망, 포지션 크기, 타이밍
- 보수: 수익률 (타인의 전략에 의존)

### 1.2 내쉬 균형 (Nash Equilibrium)

John Nash (1950), *Equilibrium Points in N-Person Games*, PNAS
(1994 노벨 경제학상)

```
전략 프로필 s* = (s₁*, s₂*, ..., sₙ*)이 내쉬 균형이려면:

∀i, ∀sᵢ ∈ Sᵢ:
  uᵢ(sᵢ*, s₋ᵢ*) ≥ uᵢ(sᵢ, s₋ᵢ*)

→ 어떤 참여자도 자신만 전략을 변경하여 보수를 개선할 수 없는 상태
```

**존재 정리** (Nash, 1950):
유한 게임에는 (혼합 전략을 포함하면) 내쉬 균형이 반드시 존재한다.

금융 적용:
- 시장 가격 = 모든 참여자 전략의 내쉬 균형
- 기술적 분석 전략이 널리 알려지면 → 균형 이동 → 수익 소멸
- 이것이 Andrew Lo의 적응적 시장 가설의 게임 이론적 표현

## 2. 시장의 게임 구조

### 2.1 제로섬 vs 비제로섬

```
제로섬 게임: Σᵢ uᵢ(s) = 0  ∀s
  → 선물/옵션 시장 (한쪽의 이익 = 다른쪽의 손실)
  → 폰 노이만 미니맥스 정리 적용

비제로섬 게임: Σᵢ uᵢ(s) ≠ 0
  → 주식 시장 (경제 성장으로 전체 파이 확대 가능)
  → 그러나 단기 트레이딩은 근사적으로 제로섬
```

### 2.2 죄수의 딜레마와 시장 붕괴

```
         참여자 B
         보유    매도
참여자 A
보유    (3,3)   (0,5)
매도    (5,0)   (1,1)

내쉬 균형: (매도, 매도) = (1,1)
파레토 최적: (보유, 보유) = (3,3)
```

금융 적용:
- 시장 패닉 시 모든 참여자가 개별적으로 합리적인 매도 선택
- 그러나 집단적으로는 비합리적 (가격 폭락)
- 흑삼병 패턴 = 죄수의 딜레마의 연쇄적 실현

### 2.3 반복 게임 (Repeated Games)

```
무한 반복 게임에서의 민중 정리 (Folk Theorem):

할인 인자 δ가 충분히 크면 (참여자가 충분히 인내하면),
내쉬 균형에서 협력(보유)이 유지될 수 있다.

조건: δ ≥ (5-3)/(5-1) = 0.5
```

금융 적용:
- 장기 투자자(δ 큼): 패닉에 동참하지 않음 → 시장 안정화
- 단기 트레이더(δ 작음): 즉시 반응 → 변동성 증가
- 기관 투자자의 존재가 시장 안정에 기여하는 이론적 근거

## 3. 경매 이론 (Auction Theory)

### 3.1 주식시장 = 이중 경매 (Double Auction)

```
매수자: 매수호가(bid) 제출
매도자: 매도호가(ask) 제출
체결: bid ≥ ask 일 때

연속 이중 경매 (Continuous Double Auction):
  실시간으로 호가가 제출되고 체결되는 시장 구조
  → KOSPI, KOSDAQ, NYSE 등 대부분의 현대 거래소
```

William Vickrey (1961), 경매 이론 선구자 (1996 노벨 경제학상)

### 3.2 정보 비대칭과 역선택

George Akerlof (1970), *The Market for Lemons* (2001 노벨 경제학상)

```
내부 정보 보유자 → 정보 우위로 거래
외부 투자자 → 역선택 위험에 노출

스프레드 = 마켓메이커의 역선택 비용 보상
```

Kyle (1985), *Continuous Auctions and Insider Trading*:
```
가격 충격 함수: ΔP = λ · (주문 흐름)

λ = 정보 비대칭 정도의 측도
λ 큼: 정보 비대칭 심각 → 스프레드 넓음
λ 작음: 정보 효율적 시장 → 스프레드 좁음
```

## 4. 메커니즘 디자인 (Mechanism Design)

Leonid Hurwicz, Eric Maskin, Roger Myerson (2007 노벨 경제학상)

```
역게임 이론: 원하는 결과를 달성하는 게임 규칙을 설계

시장 설계 문제:
  - 호가 단위(tick size): 너무 크면 비효율, 너무 작으면 노이즈
  - 매매 중단(circuit breaker): 급락 시 거래 일시 중지
  - 공매도 규제: 하락 가속 방지
```

금융 적용:
- KRX의 가격 제한폭(±30%)은 메커니즘 디자인의 결과
- 거래 중단 규칙은 죄수의 딜레마 연쇄를 차단하는 메커니즘
- 기술적 분석: 이러한 규칙이 만드는 인위적 지지/저항선 존재

## 5. 진화 게임 이론 (Evolutionary Game Theory)

### 5.1 복제자 동학 (Replicator Dynamics)

Taylor & Jonker (1978), "Evolutionary Stable Strategies and Game Dynamics",
Mathematical Biosciences, 40(1-2), 145-156 — 복제자 동학 방정식의 최초 정식화.

John Maynard Smith, *Evolution and the Theory of Games* (1982) — 진화적 안정 전략(ESS)의 체계화.

```
복제자 동학 방정식 (Taylor & Jonker, 1978):

전략 i의 비율 변화:

ẋᵢ = xᵢ · [fᵢ(x) - f̄(x)]

xᵢ: 전략 i를 사용하는 참여자 비율
fᵢ: 전략 i의 적합도(fitness) = 기대 보수
f̄: 전체 평균 적합도 = Σⱼ xⱼ · fⱼ(x)

→ 평균보다 높은 보수의 전략은 확산, 낮은 전략은 도태
→ 내쉬 균형의 동학적 기초를 제공: 복제자 동학의 정적점(rest point)은
  내쉬 균형과 밀접하게 연관된다 (Weibull 1995).
```

### 5.2 진화적 안정 전략 (ESS)

```
전략 s*가 ESS이려면:

1) u(s*, s*) ≥ u(s, s*)  ∀s  (내쉬 균형 조건)
2) u(s*, s*) = u(s, s*) ⟹ u(s*, s) > u(s, s)  (안정성)
```

금융 적용:
- 추세 추종 전략 vs 역추세 전략의 진화적 경쟁
- 한 전략이 지배적이면 → 상대 전략의 수익 증가 → 전환
- 시장의 추세/횡보 교대 = 전략 생태계의 진화적 순환
- → 기술적 분석의 유효성이 시기에 따라 변하는 근거

### 5.3 다중 전략 생태계

```
시장 참여자 유형:
  - 추세 추종자 (Trend Followers): MA 교차, 적삼병 매매
  - 역추세자 (Contrarians): RSI 과매수/과매도 매매
  - 가치 투자자 (Fundamentalists): 내재가치 기반
  - 노이즈 트레이더 (Noise Traders): 무작위

Brock & Hommes (1998), *Heterogeneous Beliefs and Routes to Chaos
in a Simple Asset Pricing Model*:
  → 이질적 참여자 모형에서 카오스적 가격 변동 발생
```

## 6. 시그널링과 선별 (Signaling & Screening)

### 6.1 스펜스 시그널링

Michael Spence (1973), *Job Market Signaling* (2001 노벨 경제학상)

금융 적용:
```
거래량 급증 = 정보 보유자의 시그널
  → "나는 중요한 정보를 가지고 있다"

적삼병 + 거래량 증가 = 강한 시그널
적삼병 + 거래량 감소 = 약한 시그널 (노이즈 가능성)
```

### 6.2 기술적 패턴의 시그널 가치

```
패턴의 정보 내용:

I(패턴) = -log₂ P(패턴|무작위)

적삼병 발생 확률 (무작위):
  P ≈ (0.5)³ × 조건부 확률 ≈ 0.02~0.05

I(적삼병) ≈ -log₂(0.03) ≈ 5 bits

→ 적삼병은 약 5비트의 정보를 전달
→ 이 정보의 경제적 가치 = 패턴 후 초과수익
```
