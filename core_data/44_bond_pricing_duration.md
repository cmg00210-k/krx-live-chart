## 2. 쿠폰채 가격결정 (Coupon Bond Pricing)

### 2.1 기본 가격결정 공식

채권의 가격은 미래에 발생하는 모든 현금흐름(Cash Flow)의 현재가치
(Present Value)의 합이다. 고정금리 쿠폰채의 경우:

```
          n
P = Σ   C / (1+y)^t  +  FV / (1+y)^n
        t=1

여기서:
  P   = 채권 가격 (Bond Price)
  C   = 기간당 쿠폰 지급액 (Coupon Payment per Period)
      = FV × (연간 쿠폰율 / 연간 지급 횟수)
  FV  = 액면가 (Face Value, Par Value) — 한국: 보통 10,000원
  y   = 기간당 만기수익률 (Yield to Maturity per Period)
  n   = 잔존 쿠폰 지급 횟수 (Number of Remaining Periods)
```

**수치 예시: 국고채 3년물**

```
조건:
  액면가 FV  = 10,000원
  표면금리    = 3.00% (연 2회 이자 지급, 한국 국고채 표준)
  만기       = 3년 (n = 6 반기)
  시장수익률  = 3.50% (연환산)
  기간당 YTM = 3.50% / 2 = 1.75%
  기간당 쿠폰 = 10,000 × 3.00% / 2 = 150원

계산:
  P = 150/(1.0175)^1 + 150/(1.0175)^2 + 150/(1.0175)^3
    + 150/(1.0175)^4 + 150/(1.0175)^5 + 150/(1.0175)^6
    + 10,000/(1.0175)^6

  = 147.42 + 144.88 + 142.38 + 139.91 + 137.48 + 135.09 + 9,006.32

  P = 9,853.48원

→ 할인채(Discount Bond): 쿠폰율(3.00%) < YTM(3.50%) → P < FV
```

### 2.2 가격-수익률 관계의 기본 성질

채권 가격과 수익률의 관계는 다음 5가지 성질을 갖는다
(Malkiel, 1962, "Expectations, Bond Prices, and the Term Structure
of Interest Rates," QJE):

```
성질 1: 가격과 수익률은 역의 관계 (Inverse Relationship)
  y↑ → P↓,  y↓ → P↑
  → 모든 현금흐름의 할인율이 변하므로

성질 2: 가격 변동의 비대칭성 (Asymmetric Price Change)
  동일한 Δy에 대해: |ΔP(y↓)| > |ΔP(y↑)|
  → 수익률 하락 시 가격 상승폭 > 수익률 상승 시 가격 하락폭
  → 이것이 볼록성(Convexity)의 원천 (§6에서 상세히 다룸)

성질 3: 만기가 길수록 가격 민감도가 크다
  동일 쿠폰, 동일 Δy: 장기채 |ΔP/P| > 단기채 |ΔP/P|

성질 4: 만기에 따른 민감도 증가율은 체감한다
  10Y→20Y의 민감도 증가분 < 20Y→30Y의 민감도 증가분

성질 5: 쿠폰이 낮을수록 가격 민감도가 크다
  동일 만기, 동일 Δy: 저쿠폰채 |ΔP/P| > 고쿠폰채 |ΔP/P|
  → 극단: 제로쿠폰채가 가장 민감
```

### 2.3 클린 가격과 더티 가격 (Clean Price vs Dirty Price)

채권 거래에서 호가(Quoted Price)와 결제 가격(Settlement Price)이 다르다:

```
Dirty Price = Clean Price + Accrued Interest (경과이자)

여기서:
  Clean Price      = 호가 가격 (시장에서 고시되는 가격)
  Accrued Interest = 지난 이자 지급일부터 결제일까지 발생한 이자
  Dirty Price      = 실제 결제 금액 (= Invoice Price)
```

**경과이자(Accrued Interest, AI) 계산:**

```
AI = C × (결제일 - 직전 이자지급일) / (다음 이자지급일 - 직전 이자지급일)

   = C × d / D

여기서:
  C = 기간당 쿠폰 금액
  d = 직전 이자지급일로부터 경과일수
  D = 이자 기간의 총 일수 (일수 산정 관례에 따름)
```

**왜 클린 가격을 호가하는가?**

```
문제: 더티 가격은 시간이 지남에 따라 자연적으로 증가 (AI 축적)
  → 수익률 변동 없이도 가격이 매일 변동
  → 가격 변동이 금리 변동 때문인지, 시간 경과 때문인지 구분 불가

해결: 클린 가격으로 호가
  → 금리 변동에 의한 순수 가격 변동만 관찰 가능
  → 톱니파(sawtooth) 패턴 제거

이자 지급일의 관계:
  이자 지급 직전: AI 최대 (≈ C), Dirty ≈ Clean + C
  이자 지급 직후: AI = 0, Dirty = Clean
  → 이자 지급일에 Clean 가격이 C만큼 점프하는 것은 정상
```

### 2.4 일수 산정 관례 (Day Count Conventions)

경과이자 계산에서 "일수"를 세는 방법은 시장마다 다르다:

```
┌─────────────────────────────────────────────────────┐
│              주요 일수 산정 관례                      │
├──────────────┬──────────────────────────────────────┤
│ 한국 국고채    │ Actual/365                           │
│ (KTB)        │ 실제 경과일수 / 365                   │
│              │ → 윤년도 365 사용 (Actual/365 Fixed)   │
├──────────────┼──────────────────────────────────────┤
│ 한국 회사채    │ Actual/365 또는 30/360               │
│              │ 발행 조건에 따라 상이                   │
├──────────────┼──────────────────────────────────────┤
│ 미국 국채     │ Actual/Actual                        │
│ (UST)        │ 실제 경과일수 / 실제 이자기간 일수      │
│              │ → 반기: 약 181~184일                  │
├──────────────┼──────────────────────────────────────┤
│ 미국 회사채    │ 30/360                              │
│              │ 1개월 = 30일, 1년 = 360일             │
│              │ 2월도 30일로 간주                      │
├──────────────┼──────────────────────────────────────┤
│ 유로 시장     │ Actual/360                           │
│ (머니마켓)    │ 실제 경과일수 / 360                   │
└──────────────┴──────────────────────────────────────┘
```

**30/360 관례 세부 규칙:**

```
30/360 일수 = 360 × (Y₂ - Y₁) + 30 × (M₂ - M₁) + (D₂ - D₁)

특수 규칙:
  - D₁ = 31 → D₁ = 30
  - D₂ = 31 이고 D₁ ≥ 30 → D₂ = 30

예: 2024-03-15 ~ 2024-09-15
  = 360 × 0 + 30 × 6 + 0 = 180일

예: 2024-01-31 ~ 2024-04-30
  D₁ = 31 → 30, D₂ = 30
  = 360 × 0 + 30 × 3 + 0 = 90일
```

### 2.6 제로쿠폰채 (Zero-Coupon Bond)

중간 이자 지급이 없는 채권:

```
P = FV / (1+y)^n

여기서:
  P   = 현재 가격
  FV  = 만기 상환금 (= 액면가)
  y   = 기간당 수익률
  n   = 잔존 기간 수

특성:
  (1) 항상 할인 발행: P < FV (y > 0일 때)
  (2) 이자소득 = 0, 자본이득 = FV - P
  (3) 경과이자 = 0 → Clean Price = Dirty Price
  (4) 듀레이션 = 잔존 만기 (§4에서 증명)
```

**수치 예시: 1년 제로쿠폰채**

```
FV = 10,000원, y = 3.50% (연환산), n = 1
P = 10,000 / (1.035)^1 = 9,661.84원

→ 투자 수익 = 10,000 - 9,661.84 = 338.16원
→ 수익률 = 338.16 / 9,661.84 = 3.50% ✓
```

### 2.7 파, 프리미엄, 디스카운트 (Par, Premium, Discount)

```
쿠폰율과 YTM의 관계에 따라:

┌──────────────────────────────────────────────────┐
│ 쿠폰율 > YTM  →  프리미엄채 (P > FV)             │
│   예: 4% 쿠폰, 3% YTM → P = 10,283원             │
│   투자자: "높은 쿠폰 → 더 비싸게 매수"             │
│                                                  │
│ 쿠폰율 = YTM  →  파 채권 (P = FV)                │
│   예: 3% 쿠폰, 3% YTM → P = 10,000원             │
│   기준: 시장 금리와 쿠폰이 일치                     │
│                                                  │
│ 쿠폰율 < YTM  →  디스카운트채 (P < FV)            │
│   예: 2% 쿠폰, 3% YTM → P = 9,719원              │
│   투자자: "낮은 쿠폰 → 더 싸게 매수"               │
└──────────────────────────────────────────────────┘
```

### 2.8 풀-투-파 효과 (Pull-to-Par Effect)

```
만기가 가까워지면 채권 가격은 액면가에 수렴한다.

프리미엄채: P(t) ↓ → FV  (시간 경과에 따라 가격 하락)
디스카운트채: P(t) ↑ → FV  (시간 경과에 따라 가격 상승)
파 채권:     P(t) = FV     (변화 없음)

수학적 설명:
  n → 0에 따라:
  P = Σ C/(1+y)^t + FV/(1+y)^n → FV  (쿠폰 항 소멸, 할인 계수 → 1)

실무적 의미:
  (1) 프리미엄채 보유 시 "자본손실" 발생 (쿠폰으로 보상)
  (2) 디스카운트채 보유 시 "자본이득" 발생 (낮은 쿠폰 보상)
  (3) 총 수익(total return)은 YTM으로 수렴 (재투자 가정 하에)
```

## 3. 만기수익률 (Yield to Maturity, YTM) 계산

### 3.1 YTM의 정의

만기수익률(Yield to Maturity, YTM)은 **채권의 시장가격과 미래 현금흐름의
현재가치를 일치시키는 단일 할인율**이다:

```
         n
P = Σ   C / (1+y)^t  +  FV / (1+y)^n
        t=1

위 등식을 만족하는 y = YTM

직관: "이 채권을 현재 가격에 매수하여 만기까지 보유하면
      얻게 되는 연환산 수익률"
```

### 3.2 YTM의 내재 가정

YTM에는 두 가지 중요한 가정이 내재되어 있다:

```
가정 1: 모든 쿠폰을 YTM과 동일한 수익률로 재투자한다
  → 현실에서는 재투자 시점의 시장금리로 재투자
  → 금리 하락 시: 실현 수익률 < YTM (재투자 수익 감소)
  → 금리 상승 시: 실현 수익률 > YTM (재투자 수익 증가)
  → 이것이 "재투자 위험(Reinvestment Risk)"

가정 2: 만기까지 보유한다
  → 중도 매각 시: 매각가격은 그 시점 시장금리에 의존
  → 금리 상승 시 매각가격 하락 → 자본손실
  → 이것이 "가격 위험(Price Risk)" 또는 "금리 위험(Interest Rate Risk)"

핵심 통찰 (Fisher & Weil, 1971):
  재투자 위험과 가격 위험은 반대 방향으로 작용한다.
  → 금리 상승: 재투자 이익 ↑, 매각가격 ↓
  → 금리 하락: 재투자 이익 ↓, 매각가격 ↑
  → 듀레이션 매칭(§8)은 이 두 위험을 상쇄시키는 전략이다.
```

### 3.3 Newton-Raphson 수치해법

YTM은 해석적(Analytical) 해가 없으므로 수치 해법으로 구한다.
Newton-Raphson 방법이 가장 표준적이다:

```
목적: f(y) = 0을 만족하는 y를 찾는다

f(y) = Σ_{t=1}^{n} C/(1+y)^t + FV/(1+y)^n - P = 0

f'(y) = -Σ_{t=1}^{n} t·C/(1+y)^(t+1) - n·FV/(1+y)^(n+1)

반복 공식:
  y_{k+1} = y_k - f(y_k) / f'(y_k)

초기값: y₀ = 쿠폰율 / 기간 (합리적 시작점)
수렴: 보통 4-6회 반복으로 소수점 이하 8자리까지 수렴
종료 조건: |f(y_k)| < ε (예: ε = 1e-10)
```

**수치 예시: Newton-Raphson 적용**

```
채권: FV = 10,000원, 쿠폰 3% (반기), 3년 만기, P = 9,853.48원
→ C = 150원, n = 6

초기값: y₀ = 0.015 (쿠폰율/2 = 0.03/2)

[반복 1]
f(0.015) = 150·Σ_{1..6} 1/(1.015)^t + 10000/(1.015)^6 - 9853.48
         = 150 × 5.6972 + 10000 × 0.9145 - 9853.48
         = 854.58 + 9145.42 - 9853.48
         = 146.52

f'(0.015) = -(1·150/(1.015)^2 + 2·150/(1.015)^3 + ... + 6·10150/(1.015)^7)
           ≈ -57,183

y₁ = 0.015 - 146.52/(-57,183) = 0.015 + 0.00256 = 0.01756

[반복 2]
f(0.01756) ≈ -3.45
y₂ = 0.01756 - (-3.45)/(-56,800) ≈ 0.01750

[반복 3]
f(0.01750) ≈ 0.002 → 수렴

결과: y_반기 = 0.01750, YTM(연환산) = 3.50%
```

### 3.4 이분법 (Bisection Method)

Newton-Raphson보다 느리지만 수렴이 보장되는 대안:

```
알고리즘:
  1. 초기 구간 [a, b] 설정: f(a) > 0, f(b) < 0 (또는 반대)
     → a = 0% (하한), b = 쿠폰율 × 2 (상한) 등
  2. 중점 c = (a + b) / 2
  3. f(c)의 부호에 따라 구간 축소:
     f(c) > 0 → 새 구간 [c, b]
     f(c) < 0 → 새 구간 [a, c]
  4. |b - a| < ε 또는 |f(c)| < ε까지 반복

수렴 속도: 선형 수렴 (매 반복 오차 1/2 감소)
  → 소수점 이하 8자리: 약 27회 반복 필요
  → Newton-Raphson(2차 수렴)의 4-6회보다 느림

장점: 초기 추정이 나빠도 항상 수렴
단점: 느림, f'(y) 불필요
```

## 4. 듀레이션 (Duration)

### 4.1 Macaulay Duration (매컬리 듀레이션)

Frederick Macaulay (1938)가 제안한 원래의 듀레이션 개념이다.
**현금흐름의 가중평균 수취 시점**으로 정의된다:

```
                  1       n       CF_t
D_mac = ─── · Σ  t · ─────────
                  P      t=1    (1+y)^t

         Σ_{t=1}^{n}  t · PV(CF_t)
       = ─────────────────────────
         Σ_{t=1}^{n}  PV(CF_t)

여기서:
  t        = 현금흐름 수취 시점 (기간 단위)
  CF_t     = t 시점의 현금흐름
  PV(CF_t) = CF_t / (1+y)^t = t 시점 현금흐름의 현재가치
  P        = 채권 가격 = Σ PV(CF_t)
  y        = 기간당 YTM

단위: 기간 (periods) 또는 년 (years)
→ 반기 기준 듀레이션을 연 기준으로 변환: D_years = D_periods / 2
```

**직관:**

```
듀레이션 = "평균적으로 몇 년 후에 투자 원금을 회수하는가?"

비유:
  시소(seesaw)의 받침점 위치
  → 각 시점의 현금흐름을 시소 위의 무게라 하면,
     듀레이션은 시소가 균형 잡히는 지점

예: 3년 만기, 5% 쿠폰 국고채
  현금흐름 무게 분포:  [500, 500, 10500] (액면 10,000)
  PV 무게 분포:       [476, 454, 9070]
  가중평균 시점 ≈ 2.86년
  → 만기(3년)보다 짧음: 쿠폰이 받침점을 앞으로 당김
```

### 4.2 Macaulay Duration 수치 예시

```
채권: FV = 10,000원, 쿠폰 4% (반기), 5년 만기, YTM = 3.50%
→ C = 200원 (반기), n = 10 (반기), y = 1.75% (반기)

┌──────┬───────┬────────────┬──────────────┬───────────────┐
```

### 4.3 듀레이션의 특수 사례

```
(1) 제로쿠폰채:
  현금흐름이 만기 시점 하나뿐
  → D_mac = n (만기)
  → 듀레이션 = 잔존 만기

  증명: D = (1/P) × n × FV/(1+y)^n = (1/P) × n × P = n  ∎

(2) 영구채 (Perpetuity, 콘솔):
  무한 시점까지 C 지급, 원금 상환 없음

  D_mac = (1+y) / y

  예: y = 5% → D = 1.05/0.05 = 21.0년
  예: y = 3% → D = 1.03/0.03 = 34.3년

  증명: P = C/y, D = (1/P) × Σ_{t=1}^∞ t·C/(1+y)^t
        Σ t·x^t = x/(1-x)^2 (x = 1/(1+y))
        D = (C/P) × (1+y)/y^2 = (y/1) × (1+y)/y^2 = (1+y)/y  ∎

(3) 변동금리채 (FRN):
  이자 리셋 직후: D ≈ 다음 리셋까지의 기간
  → 분기 리셋 FRN: D ≈ 0.25년
  → 금리 변동에 거의 면역 (금리 리스크 ≈ 0)
```

### 4.4 Modified Duration (수정 듀레이션)

금리 민감도의 실용적 지표:

```
              D_mac
D_mod = ─────────────
           1 + y/m

여기서:
  D_mac = Macaulay Duration
  y     = 연환산 YTM
  m     = 연간 복리 횟수 (반기 = 2, 분기 = 4)

대안적 표현:
  D_mod = D_mac / (1 + y_period)  (y_period = 기간당 YTM)
```

**가격 민감도 공식 (1차 근사):**

```
ΔP/P ≈ -D_mod × Δy

여기서:
  ΔP/P = 가격 변화율 (% 변동)
  Δy   = 수익률 변화 (절대값, 예: 50bp = 0.005)
  부호: D_mod > 0이므로, Δy > 0 → ΔP < 0 (역의 관계)

유도:
  P = Σ CF_t / (1+y)^t
  dP/dy = -Σ t·CF_t / (1+y)^(t+1)
        = -[1/(1+y)] × Σ t·CF_t / (1+y)^t
        = -[1/(1+y)] × P × D_mac

  → (1/P) × dP/dy = -D_mac / (1+y) = -D_mod

  → ΔP/P ≈ -D_mod × Δy  (1차 Taylor 전개)
```

**수치 예시:**

```
앞의 5년 채권: D_mac = 4.544년, y = 3.50%, m = 2

D_mod = 4.544 / (1 + 0.035/2) = 4.544 / 1.0175 = 4.466

시나리오: 수익률이 3.50% → 4.00% (+50bp)
  ΔP/P ≈ -4.466 × 0.005 = -0.0223 = -2.23%
  P_new ≈ 10,206.22 × (1 - 0.0223) = 9,978.62원

실제 계산 (가격 공식 재적용):
  P(4.00%) = 10,000.00원 (파 채권이 됨)
  실제 ΔP/P = (10,000 - 10,206.22) / 10,206.22 = -2.02%

오차: -2.23% vs -2.02% = 0.21%p (볼록성 무시에 의한 오차, §6에서 보정)
```

### 4.5 Dollar Duration (달러 듀레이션)

절대 금액으로 표시한 금리 민감도:

```
Dollar Duration = P × D_mod

또는 수익률 변동 Δy에 대한 절대 가격 변동:

ΔP ≈ -P × D_mod × Δy = -DD × Δy

예: P = 10,206원, D_mod = 4.466
  DD = 10,206 × 4.466 = 45,580원/100%

  → 수익률 +1bp (0.01%): ΔP ≈ -45,580 × 0.0001 = -4.56원
  → 수익률 +50bp (0.50%): ΔP ≈ -45,580 × 0.005 = -227.90원
```

### 4.6 듀레이션의 5가지 성질

```
성질 1: 만기 증가 → 듀레이션 증가 (체감적)
  ┌──────────┬────────────────────────┐
  │ 만기     │ D_mod (쿠폰 3%, y=3.5%)│
  ├──────────┼────────────────────────┤
  │  1년     │  0.98                  │
  │  3년     │  2.86                  │
  │  5년     │  4.47                  │
  │ 10년     │  8.38                  │
  │ 20년     │ 14.78                  │
  │ 30년     │ 19.65                  │
  │ 50년     │ 25.23                  │
  │ 영구채   │ 29.28 (=(1+y)/y/y_adj) │
  └──────────┴────────────────────────┘
  → 30→50년 증가 (5.58) < 10→30년 증가 (11.27): 체감

성질 2: 쿠폰 증가 → 듀레이션 감소
  높은 쿠폰 → 더 많은 현금흐름이 일찍 도달
  → 가중평균 시점이 앞으로 이동
  → 극단: 쿠폰 0%(제로) → D = 만기 (최대)

성질 3: 수익률 증가 → 듀레이션 감소
  높은 할인율 → 먼 미래 현금흐름의 PV 비중 감소
  → 가까운 쿠폰의 상대적 비중 증가
  → 가중평균 시점이 앞으로 이동

성질 4: 제로쿠폰채의 듀레이션 = 만기
  중간 현금흐름 없음 → 모든 무게가 만기 시점에 집중

성질 5: 포트폴리오 듀레이션 = 가중평균
  D_portfolio = Σ w_i × D_i
  w_i = (P_i × N_i) / Σ(P_j × N_j) = 시가 기준 비중

  예: 3년채(40%) D=2.86 + 10년채(60%) D=8.38
  D_p = 0.4 × 2.86 + 0.6 × 8.38 = 6.17
```

## 5. DV01 (Dollar Value of a Basis Point)

### 5.1 DV01의 정의

DV01(Dollar Value of a Basis Point)은 수익률이 1 베이시스 포인트(1bp =
0.01%) 변동할 때의 채권 가격 변동액이다:

```
DV01 = P × D_mod × 0.0001

또는 수치적 정의:

DV01 = [P(y - 0.5bp) - P(y + 0.5bp)] / 1

여기서:
  P  = 채권 가격 (액면 10,000원 기준 또는 실제 투자금액)
  D_mod = 수정 듀레이션
  0.0001 = 1bp = 0.01%

동의어:
  PVBP (Price Value of a Basis Point)
  BPV  (Basis Point Value)
```

**수치 예시:**

```
5년 국고채 (FV = 10,000원):
  P = 10,206원, D_mod = 4.466

  DV01 = 10,206 × 4.466 × 0.0001 = 4.56원 / 1bp

해석: 수익률이 1bp 변동하면 채권 가격은 약 4.56원 변동
     수익률이 10bp 변동하면: 약 45.6원 변동
     수익률이 100bp(1%) 변동하면: 약 456원 변동

액면 1억원 기준:
  DV01 = 1억 × 4.466 × 0.0001 = 44,660원 / 1bp
  → 10Y 국고채: DV01 ≈ 83,800원/1bp (액면 1억)
  → 30Y 국고채: DV01 ≈ 196,500원/1bp (액면 1억)
```

### 5.2 DV01 헤지 비율

```
목표: 포트폴리오의 금리 위험을 제거 (DV01 = 0)

헤지 비율:
  N_hedge = -DV01_portfolio / DV01_hedge_instrument

예: 10Y 국고채 포트폴리오 (액면 100억)
  DV01_portfolio = 83,800 × 100 = 838만원/1bp

  KTB 10Y 선물 1계약 DV01 ≈ 8만원/1bp (대략)
  N_futures = -8,380,000 / 80,000 = -104.8 → 105계약 매도

교차 헤지(Cross-Hedge):
  DV01이 다른 두 수단 간 헤지:
  N_B = -(DV01_A / DV01_B) × ρ_AB

  여기서 ρ_AB = 수익률 변동의 상관계수
  → 상관이 완벽(=1)하지 않으면 "기저 위험(Basis Risk)" 존재
```

### 5.3 KTB 선물 DV01

```

```

### 5.4 전환계수와 최저비용인도채권 (CF & CTD)

KTB 선물의 결제 시 여러 채권이 인도 가능하다. 전환계수(Conversion Factor,
CF)는 상이한 쿠폰의 채권을 동일 기준으로 비교하기 위한 조정 계수이다:

```
전환계수 (CF):
  CF_i = 채권 i의 이론가 / 액면 (수익률을 표준 쿠폰율로 가정)
  → CF > 1: 쿠폰 > 표준 (프리미엄)
  → CF < 1: 쿠폰 < 표준 (디스카운트)
  → CF = 1: 쿠폰 = 표준 (파)

인도 비용 (Delivery Cost):
  Cost_i = P_i - CF_i × Futures_Price
  → 인도자(매도자)는 Cost가 가장 낮은 채권을 인도
  → 이것이 CTD (Cheapest-to-Deliver)

CTD 결정 법칙 (근사):
  - 수익률 > 표준 쿠폰: 장기, 저쿠폰 채권이 CTD
  - 수익률 < 표준 쿠폰: 단기, 고쿠폰 채권이 CTD

의미:
  선물 가격 ≈ CTD 가격 / CF_CTD
  선물의 DV01 ≈ CTD의 DV01 / CF_CTD
  → CTD가 바뀌면 선물의 금리 민감도가 급변할 수 있음
```

## 6. 볼록성 (Convexity)

### 6.1 볼록성의 정의

듀레이션이 가격-수익률 관계의 1차 근사(기울기)라면,
볼록성은 2차 근사(곡률)이다:

```
수학적 정의:

              1       d²P
Convexity = ─── × ────
              P      dy²

이산형 계산 공식:

          1       n     t(t+1) · CF_t
C = ─────────── · Σ   ─────────────────
     P·(1+y)²   t=1     (1+y)^t

여기서:
  CF_t = t 시점 현금흐름
  y    = 기간당 YTM
  n    = 총 기간 수
  P    = 채권 가격

단위: 연²(years²) 또는 기간²
→ 반기 기준 볼록성을 연 기준으로 변환: C_annual = C_semi / 4
```

**수치적 근사 (모형에 독립적):**

```
              P(y-Δy) + P(y+Δy) - 2·P(y)
C_approx = ──────────────────────────────
                   P(y) · (Δy)²

여기서 Δy = 작은 수익률 변동 (보통 1bp ~ 10bp 사용)

예: P(3.49%) = 10,210.78, P(3.50%) = 10,206.22, P(3.51%) = 10,201.67

C = (10,210.78 + 10,201.67 - 2 × 10,206.22) / (10,206.22 × (0.0001)^2)
  = 0.01 / (10,206.22 × 1e-8)
  ≈ 97.98
→ 이 값은 반기 기준 → 연환산: 97.98 / 4 ≈ 24.5 (연²)
```

### 6.2 듀레이션-볼록성 가격 근사 (2차 Taylor 전개)

```
ΔP/P ≈ -D_mod × Δy  +  (1/2) × C × (Δy)²
        ──────────      ───────────────────
         1차 항            2차 항 (볼록성 보정)

풀어 쓰면:
  P_new ≈ P × [1 - D_mod × Δy + (1/2) × C × (Δy)²]
```

**수치 비교: 듀레이션만 vs 듀레이션+볼록성**

```
5년 국고채: P = 10,206원, D_mod = 4.466, C = 24.5

시나리오 1: Δy = +50bp (3.50% → 4.00%)
  듀레이션만:    ΔP/P = -4.466 × 0.005 = -2.233%
  +볼록성:      ΔP/P = -2.233% + 0.5 × 24.5 × (0.005)² = -2.233% + 0.031% = -2.202%
  실제:         ΔP/P = -2.02%
  → 볼록성 보정으로 오차 축소 (0.21%p → 0.18%p)

시나리오 2: Δy = -50bp (3.50% → 3.00%)
  듀레이션만:    ΔP/P = -4.466 × (-0.005) = +2.233%
  +볼록성:      ΔP/P = +2.233% + 0.5 × 24.5 × (0.005)² = +2.233% + 0.031% = +2.264%
  실제:         ΔP/P = +2.27%
  → 볼록성이 비대칭성을 포착

시나리오 3: Δy = +200bp (대규모 변동)
  듀레이션만:    ΔP/P = -4.466 × 0.02 = -8.932%
  +볼록성:      ΔP/P = -8.932% + 0.5 × 24.5 × (0.02)² = -8.932% + 0.490% = -8.442%
  실제:         ΔP/P = -8.17%
  → 큰 변동에서 볼록성 보정이 더 중요해짐
```

### 6.3 양의 볼록성 (Positive Convexity)

```
모든 옵션 미내재(option-free) 채권은 양의 볼록성을 갖는다:

                    가격
                    ↑
                    │     .*
                    │   .*   *. 실제 가격-수익률 곡선 (볼록)
                    │  *       *.
                    │.*          *..
                    │*              *...
                    │                  *....
           --------+*---------+----------→ 수익률
                    │*        │
                    │ *.      │ 듀레이션 직선 (접선)
                    │   *.    │
                    │     *.  │
                    │       *.│

양의 볼록성의 의미:
  실제 곡선이 접선(듀레이션 직선) 위에 있다
  → 수익률 하락 시: 실제 가격 > 듀레이션 예측 가격 (더 오름)
  → 수익률 상승 시: 실제 가격 > 듀레이션 예측 가격 (덜 내림)
  → "좋은 비대칭": 이익 > 손실

투자자의 관점:
  동일한 듀레이션이라면, 볼록성이 높은 채권이 유리
  → 볼록성은 "공짜 점심(free lunch)"이라는 표현
  → 실제로는 프리미엄을 지불 (수익률이 약간 낮음)
```

### 6.4 음의 볼록성 (Negative Convexity)

```
내재옵션이 있는 채권에서 발생:

(1) 콜러블 채권 (Callable Bond):
  발행자가 조기 상환 가능 → 수익률 하락 시 조기 상환 확률 ↑
  → 가격 상승이 콜 가격(Call Price)에 의해 제한
  → 수익률 하락 시: 가격 상승 < 옵션미내재 채권
  → 낮은 수익률 영역에서 음의 볼록성

                    가격
                    ↑
          Call Price│- - - - -..*...*...*  옵션미내재
                    │       .*
                    │     .*  콜러블 (음의 볼록성 구간)
                    │   .*
                    │  *   ← 여기서부터 양의 볼록성
                    │.*
           --------+*---------→ 수익률
                    │

(2) MBS (Mortgage-Backed Securities):
  주택담보대출자의 조기상환 옵션
  → 금리 하락 시 대환(refinancing) 증가
  → 원금 조기 회수 → 유효 만기 단축
  → 가격 상승 제한 → 음의 볼록성

(3) 한국 시장:
  콜러블 회사채: 발행 3-5년 후 콜 옵션 (보통)
  구조화채권(DLS/ELS 연계): 복잡한 옵션 구조
  → 옵션 내재 채권은 유효듀레이션(§10)으로 분석
```

### 6.5 볼록성의 결정 요인

```
볼록성이 증가하는 경우:

(1) 만기 증가:
  장기채는 현금흐름 분산(dispersion)이 크다
  → t(t+1) 항이 큰 값을 가짐

(2) 쿠폰 감소:
  현금흐름이 만기에 집중 → 분산 증가
  → 제로쿠폰채가 동일 만기에서 최대 볼록성

(3) 수익률 감소:
  먼 미래 현금흐름의 PV 비중 증가
  → 분산 증가 → 볼록성 증가

볼록성의 규모감:
  ┌──────────┬────────┬──────────┐
  │ 만기     │ D_mod  │ Convexity│
  ├──────────┼────────┼──────────┤
  │  1년     │  0.98  │    1.4   │
  │  3년     │  2.86  │   10.8   │
  │  5년     │  4.47  │   24.5   │
  │ 10년     │  8.38  │   84.0   │
  │ 20년     │ 14.78  │  273.0   │
  │ 30년     │ 19.65  │  520.0   │
  └──────────┴────────┴──────────┘
  (3% 쿠폰, 3.5% YTM 기준, 근사치)
```

## 7. 주요금리 듀레이션 (Key Rate Duration, KRD)

### 7.1 한계: 평행이동 가정

Macaulay/Modified Duration의 근본적 한계:

```
가정: 수익률 곡선이 평행하게 이동 (parallel shift)
  → 모든 만기의 수익률이 동일한 Δy만큼 변동
  → 1Y, 3Y, 5Y, 10Y, 30Y 수익률이 동시에 +50bp

현실:
  → 수익률 곡선은 비평행 이동(non-parallel shift)이 일반적
  → Bull Steepening: 장기금리 하락 > 단기금리 하락
  → Bear Flattening: 단기금리 상승 > 장기금리 상승
  → Twist: 단기 ↑ 동시에 장기 ↓ (또는 반대)
  → Butterfly: 중기 이동 ≠ 단기/장기 이동

문제:
  동일한 Modified Duration이라도 비평행 이동에 대한 반응이 다를 수 있다.
  예: D_mod = 7인 바벨(Barbell)과 불릿(Bullet) 포트폴리오는
      평행 이동에는 동일 반응, 트위스트에는 정반대 반응
```

### 7.2 Key Rate Duration (Ho, 1992)

Thomas Ho (1992)가 제안한 주요금리 듀레이션:

```
정의:
  KRD_τ = -(1/P) × (∂P / ∂y_τ)

여기서:
  y_τ  = 만기 τ 지점의 수익률
  ∂P/∂y_τ = τ 지점 수익률만 1bp 변동 시 가격 변화

방법론:
  1. 기준 수익률 곡선에서 특정 만기(τ)의 수익률만 1bp 상승
  2. 나머지 만기는 보간(interpolation)으로 조정
  3. 변경된 곡선으로 채권 가격 재계산
  4. KRD_τ = -(P_shifted - P_base) / (P_base × 0.0001)

핵심 성질:
  Σ KRD_τ ≈ D_mod  (평행 이동 시 합 = Modified Duration)
  → KRD는 D_mod를 만기별로 분해한 것
```

### 7.3 표준 KRD 격자점 (Tenor Points)

```
한국 국고채 기준 표준 격자점:

  1Y — 2Y — 3Y — 5Y — 7Y — 10Y — 20Y — 30Y — 50Y

미국 Treasury 기준:
  3M — 6M — 1Y — 2Y — 3Y — 5Y — 7Y — 10Y — 20Y — 30Y

이동 전파 (Shift Propagation):
  τ 지점의 1bp 이동은 인접 격자점에 선형 보간으로 전파

  예: 5Y 지점 +1bp 이동 시
    3Y: +0bp (영향 없음)
    4Y: +0.5bp (5Y와 3Y 사이 보간)
    5Y: +1.0bp (전체 이동)
    6Y: +0.6bp (5Y와 7Y 사이 보간)
    7Y: +0bp (영향 없음)

  → "텐트(tent)" 형태의 이동 함수
```

### 7.4 KRD 프로파일 비교

```
동일 D_mod = 7.0인 세 포트폴리오:

(A) 불릿(Bullet) — 7년 단일 채권 중심:
  KRD: [0, 0, 0, 0.5, 6.0, 0.5, 0, 0]
       1Y  2Y 3Y  5Y   7Y  10Y 20Y 30Y

(B) 바벨(Barbell) — 2년 + 30년:
  KRD: [0, 2.0, 0, 0, 0, 0, 0, 5.0]
       1Y   2Y  3Y 5Y 7Y 10Y 20Y  30Y

(C) 래더(Ladder) — 고른 분산:
  KRD: [0.5, 0.8, 1.0, 1.2, 1.0, 1.0, 0.8, 0.7]
       1Y   2Y   3Y   5Y   7Y  10Y  20Y  30Y

시나리오 분석:

시나리오 1: 평행 이동 +50bp
  A: -7.0 × 50bp = -350bp → 세 포트폴리오 동일
  B: -7.0 × 50bp = -350bp
  C: -7.0 × 50bp = -350bp

시나리오 2: 장단기 스프레드 확대 (2Y: 0bp, 30Y: +50bp)
  A: -0.5 × 0 - 6.0 × 25bp* - 0.5 × 50bp ≈ -175bp
  B: -2.0 × 0 - 5.0 × 50bp = -250bp
  C: -0.8 × 0 - ... - 0.7 × 50bp ≈ -125bp
  → 바벨이 가장 큰 손실 (장기 노출 높음)

시나리오 3: 버터플라이 (2Y: +25, 7Y: -10, 30Y: +25)
  A: -0.5×25 - 6.0×(-10) - 0.5×25 = +35bp (이익!)
  B: -2.0×25 - 5.0×25 = -175bp (손실)
  C: 분산 효과로 완충
  → 불릿이 나비 이동에 유리

* 보간에 의한 근사값
```

### 7.5 KRD 기반 헤지

```
목표: 특정 만기 구간의 노출을 조절

예: 10Y 노출만 제거하고 3Y 노출 유지

현재 포트폴리오 KRD:
  3Y: +2.5, 10Y: +4.0 (나머지 0)

헤지 수단: KTB 10Y 선물 1계약의 KRD
  3Y: +0.1, 10Y: +0.08 (KRD_10Y / 계약)

필요 매도 계약수: N = -4.0 / 0.08 = 50계약

헤지 후 KRD:
  3Y: 2.5 - 50 × 0.1 = -2.5 (부작용!)
  10Y: 4.0 - 50 × 0.08 = 0.0 ✓

문제: 10Y 헤지가 3Y에 부수적 영향
→ 해결: 두 수단(3Y선물 + 10Y선물) 동시 사용
→ 연립방정식으로 해결 (§8.1과 연결)
```

## 8. 면역전략 (Immunization)

### 8.1 고전적 면역 (Classical Immunization)

Frank Redington (1952)이 보험 수리 맥락에서 제안:

```
목적: 금리 변동에도 부채를 충족할 수 있는 자산 포트폴리오 구성

3가지 조건:
  (1) PV(자산) = PV(부채)        ← 현재가치 일치
  (2) D(자산) = D(부채)          ← 듀레이션 일치
  (3) C(자산) ≥ C(부채)          ← 볼록성 우위

직관:
  듀레이션을 맞추면 → 평행 이동에 대해 1차 면역
  볼록성 우위이면  → 어떤 방향의 이동이든 순자산 ≥ 0

증명 (개략):
  순자산 S = A - L (자산 - 부채)
  dS/dy = 0     (D_A = D_L이면 → 1차 미분 = 0)
  d²S/dy² > 0   (C_A > C_L이면 → 2차 미분 > 0)
  → S(y) ≥ S(y₀) = 0 for all y near y₀
  → 금리가 어느 방향으로 변해도 순자산이 감소하지 않음
```

**수치 예시: 연금 부채 면역**

```
부채: 5년 후 100억원 일시 지급
  PV(부채) = 100억 / (1.035)^5 = 84.19억원
  D(부채) = 5.0년 (단일 지급이므로 = 만기)

자산 구성 (2종 국고채):
  채권 A: 3년 만기, D_A = 2.86년
  채권 B: 10년 만기, D_B = 8.38년

듀레이션 매칭:
  w_A × 2.86 + w_B × 8.38 = 5.0
  w_A + w_B = 1

연립방정식 풀이:
  w_A × 2.86 + (1 - w_A) × 8.38 = 5.0
  w_A × (2.86 - 8.38) = 5.0 - 8.38
  w_A × (-5.52) = -3.38
  w_A = 0.612 (61.2%)
  w_B = 0.388 (38.8%)

투자금액:
  채권 A: 84.19 × 0.612 = 51.52억원
  채권 B: 84.19 × 0.388 = 32.67억원
  합계: 84.19억원 = PV(부채) ✓

포트폴리오 듀레이션:
  D_p = 0.612 × 2.86 + 0.388 × 8.38 = 5.00 ✓
```

### 8.2 면역의 한계

```
(1) 평행 이동 가정:
  고전적 면역은 수익률 곡선의 평행 이동만 보호
  → 비평행 이동(twist, butterfly)에는 불완전
  → KRD 매칭(§7)으로 보완 가능

(2) 즉시적 이동 가정:
  금리가 "한 번에" 변한다고 가정
  → 실제로는 연속적으로 변동
  → 경과 시간에 따른 듀레이션 변화 미반영

(3) 듀레이션 드리프트:
  시간 경과 → 자산/부채 듀레이션이 서로 다른 속도로 변화
  → 정기적 리밸런싱 필요

(4) 거래비용:
  리밸런싱마다 매매 비용 발생
  → 너무 잦은 리밸런싱은 비용 과다
  → 실무: 분기 리밸런싱이 표준, 월간은 정밀 운용
```

### 8.3 현금흐름 매칭 (Cash Flow Matching / Dedication)

```
정의: 자산의 현금흐름을 부채의 현금흐름과 정확히 일치시키는 전략

방법:
  부채: [L₁, L₂, ..., L_n] (각 시점 지급액)
  자산: 채권 조합의 현금흐름이 [L₁, L₂, ..., L_n]과 일치

역방향 구축 (Backward Construction):
  1. 가장 먼 미래 부채 L_n에 맞는 채권 매수
  2. 이 채권의 중간 쿠폰을 다음 부채에서 차감
  3. 잔여 부채에 맞는 다음 채권 매수
  4. 반복

장점:
  → 금리 위험 완전 제거 (재투자 위험도 없음)
  → 리밸런싱 불필요
  → 면역보다 "확실한" 전략

단점:
  → 비용이 높음 (정확한 매칭을 위해 프리미엄 지불)
  → 유연성 부족 (부채 변경 시 재구축 필요)
  → 적합한 채권이 존재하지 않을 수 있음
  → 초과 수익 기회 포기
```

### 8.4 바벨 vs 불릿 전략 비교

```
동일 듀레이션, 동일 투자금의 두 전략:

바벨(Barbell): 단기채 + 장기채
  예: 2년(50%) + 20년(50%) → D ≈ 11년
  특성: 높은 볼록성, 높은 곡선 위험

불릿(Bullet): 중기채 집중
  예: 10년(100%) → D ≈ 11년
  특성: 낮은 볼록성, 낮은 곡선 위험

┌──────────────────────┬──────────┬──────────┐
│ 시나리오              │ 바벨     │ 불릿     │
├──────────────────────┼──────────┼──────────┤
│ 평행 이동 ±100bp     │ 유리     │ 불리     │
│  (볼록성 우위)        │ (볼록↑)  │ (볼록↓)  │
├──────────────────────┼──────────┼──────────┤
│ 곡선 기울기 확대      │ 불리     │ 유리     │
│  (장기↑, 단기 유지)   │ (장기 손)│ (중기 少) │
├──────────────────────┼──────────┼──────────┤
│ 곡선 기울기 축소      │ 유리     │ 불리     │
│  (장기↓, 단기 유지)   │ (장기 익)│          │
├──────────────────────┼──────────┼──────────┤
│ 나비 이동            │ 불리     │ 유리     │
│  (양끝↑, 중간↓)      │ (양끝 손)│ (중간 익) │
└──────────────────────┴──────────┴──────────┘

실무 선택 기준:
  → 금리 방향에만 베팅: 바벨 (볼록성 우위)
  → 곡선 형태에 베팅: 선택적
  → 보수적 운용: 불릿 (곡선 위험 최소)
```

### 8.5 조건부 면역 (Contingent Immunization)

Leibowitz & Weinberger (1982)가 제안:

```
핵심 아이디어:
  적극적 운용(active management)과 면역(immunization)의 혼합

작동 방식:
  1. 목표 수익률(Target Rate) 설정 — 면역으로 확보 가능한 최소 수익률
  2. 현재 포트폴리오 가치가 "면역 가능 가치(Safety Net)" 이상이면
     → 적극적 운용 (듀레이션 조절, 섹터 교체 등)
  3. 포트폴리오 가치가 Safety Net에 근접하면
     → 즉시 면역 전략으로 전환 (Lock-in)

정의:
  Safety Cushion = 현재 포트폴리오 가치 - 면역 필요 가치
  면역 필요 가치 = PV(부채) at 현재 시장 수익률

  Safety Cushion > 0 → 적극적 운용 가능
  Safety Cushion ≈ 0 → 면역으로 전환 (trigger)
  Safety Cushion < 0 → 이론상 불가 (trigger 실패)

장점:
  → 면역의 안전성 + 적극적 운용의 초과수익 가능성
  → 최악의 경우에도 최소 수익률 확보

단점:
  → 적극적 운용 실패 시 조기 면역 전환 → 기회비용
  → trigger 시점 결정의 어려움
  → 유동성 위기 시 즉시 면역 전환이 불가능할 수 있음
```

### 8.6 한국 시장에서의 면역 전략

## 9. 기간구조 이론 (Term Structure Theories)

### 9.1 서론: 수익률 곡선의 형태

수익률 곡선(Yield Curve)은 만기에 따른 수익률의 함수이다.
본 절에서는 수익률 곡선이 왜 특정 형태를 갖는지를 설명하는
**이론적 기초**를 다룬다.

```
주의:
  Doc 35 §2는 NSS(Nelson-Siegel-Svensson) 모형으로 곡선을 "피팅"하는
  실증적 도구를 다룸. 본 절은 그 곡선 형태의 이론적 근거만 다룸.
  NSS 수식, 파라미터 해석, Level/Slope/Curvature 분해는 Doc 35 §2 참조.

수익률 곡선의 4가지 대표적 형태:

(1) 정상형 (Normal / Upward-Sloping):
    수익률
    ↑        .....
    │      ..
    │    ..
    │  ..
    │ .
    │.
    +──────────→ 만기
    → 가장 일반적 형태 (역사적으로 ~70% 기간)
    → 장기 수익률 > 단기 수익률

(2) 역전형 (Inverted / Downward-Sloping):
    수익률
    ↑.
    │ .
    │  ..
    │    ..
    │      ...
    │         ....
    +──────────→ 만기
    → 경기침체 선행 지표 (Harvey, 1988)
    → 장기 수익률 < 단기 수익률

(3) 수평형 (Flat):
    수익률
    ↑
    │─────────────
    │
    +──────────→ 만기
    → 전환기에 나타남 (정상 ↔ 역전 사이)

(4) 혹형 (Humped):
    수익률
    ↑      .....
    │    ..      ..
    │  ..          ..
    │ .              ..
    │.                 ..
    +──────────→ 만기
    → 중기 수익률이 가장 높고 장기에서 하락
    → 드물지만 통화정책 전환기에 관찰
```

### 9.2 순수 기대 이론 (Pure Expectations Hypothesis)

```
핵심 주장 (Irving Fisher, 1896; Lutz, 1940):
  장기 수익률 = 미래 단기 수익률의 기대값의 기하평균

수학적 표현:
  (1 + y_n)^n = (1 + y₁) × (1 + E[r₂]) × ... × (1 + E[r_n])

  또는 선도금리(forward rate)로:
  f(t, t+1) = E[r_{t+1}]

  여기서:
    y_n     = n기간 현물(spot) 수익률
    y₁      = 현재 1기간 수익률
    E[r_t]  = t 시점에 기대되는 1기간 수익률
    f(t,t+1) = t에서 t+1까지의 내재 선도금리

해석:
  → 정상형 곡선: 시장이 미래 금리 상승을 예상
  → 역전형 곡선: 시장이 미래 금리 하락을 예상
  → 수평형 곡선: 시장이 금리 변동 없음을 예상

예시:
  1Y 금리 = 3.0%, 2Y 금리 = 3.5%
  내재 선도금리 f(1,2):
  (1.035)^2 = (1.03) × (1 + f)
  f = (1.035)^2 / 1.03 - 1 = 1.07123 / 1.03 - 1 = 4.00%
  → 시장은 1년 후 1Y 금리를 4.00%로 예상

비판:
  (1) 위험 프리미엄 무시: 장기 투자는 더 많은 불확실성 → 보상 필요
  (2) 경험적 실패: 정상형 곡선이 항상 금리 상승을 의미하지 않음
  (3) 장기채 수요 요인 무시: 보험/연기금의 구조적 장기채 수요
```

### 9.3 유동성 프리미엄 이론 (Liquidity Preference Theory)

John R. Hicks (1946), *Value and Capital*:

```
핵심 수정:
  장기 수익률 = 기대 미래 단기금리 + 유동성 프리미엄

수학적 표현:
  f(t, t+1) = E[r_{t+1}] + L(t+1)

  여기서:
    L(t) = 유동성 프리미엄 (Liquidity Premium / Term Premium)
    L(t) > 0 for all t
    L(t) < L(t+1)  (만기가 길수록 프리미엄 증가)

직관:
  → 투자자는 단기 투자를 선호 (유동성 선호)
  → 장기 투자를 유인하려면 추가 수익(프리미엄)을 제공해야 함
  → 프리미엄은 만기에 따라 체증

함의:
  → 정상형 곡선 ≠ 반드시 금리 상승 기대
    (유동성 프리미엄만으로도 정상형 가능)
  → 수평형 곡선 → 시장은 금리 소폭 하락 기대
    (하락 기대 + 유동성 프리미엄 = 수평)
  → 역전형 곡선 → 시장은 금리 대폭 하락 기대
    (대폭 하락 기대가 유동성 프리미엄을 압도)

유동성 프리미엄의 경험적 크기 (한국):
  KTB 기준 (Eo & Jeon, 2019 한국은행 추정):
  1Y → 3Y:   약 15-30bp
  3Y → 10Y:  약 30-60bp
  10Y → 30Y: 약 20-50bp
  → 10Y 부근에서 프리미엄 증가율이 체감하는 경향
```

### 9.4 시장분할 이론 (Market Segmentation Hypothesis)

John M. Culbertson (1957):

```
핵심 주장:
  수익률 곡선의 각 만기 구간은 독립적인 수급에 의해 결정된다.

가정:
  → 투자자들은 특정 만기에 고착 (habitat)
  → 다른 만기로의 이동 없음 (또는 극히 제한적)
  → 각 만기 구간은 별도의 시장

한국 시장의 기관별 선호 만기:

┌──────────────┬──────────────────────────────┐
│ 기관 유형     │ 선호 만기                     │
├──────────────┼──────────────────────────────┤
│ 은행         │ 단기 (1-3년)                  │
│              │ 이유: 예금 만기와 매칭          │
├──────────────┼──────────────────────────────┤
│ 자산운용사    │ 중기 (3-5년)                  │
│              │ 이유: 펀드 듀레이션 벤치마크    │
├──────────────┼──────────────────────────────┤
│ 보험사       │ 장기 (10-30년)                │
│              │ 이유: 보험 부채 매칭            │
├──────────────┼──────────────────────────────┤
│ 연기금       │ 초장기 (20-50년)              │
│              │ 이유: 연금 부채 매칭            │
├──────────────┼──────────────────────────────┤
│ 외국인       │ 단-중기 (1-10년)              │
│              │ 이유: 환 헤지 비용, 유동성 선호 │
└──────────────┴──────────────────────────────┘

비판:
  → 너무 극단적: 실제로는 기관들이 수익률 차이에 반응하여 이동
  → 곡선의 만기간 상관관계를 설명하지 못함
  → 통화정책의 장단기 전달 메커니즘 설명 불가
```

### 9.5 선호서식처 이론 (Preferred Habitat Theory)

Modigliani & Sutch (1966):

```
핵심: 기대이론 + 시장분할의 절충

가정:
  → 투자자들은 선호하는 만기 구간이 있다 (habitat)
  → 그러나 충분한 프리미엄이 주어지면 다른 만기로 이동 가능
  → 프리미엄의 크기는 수급에 따라 결정

수학적 표현:
  f(t, t+1) = E[r_{t+1}] + φ(t)

  여기서 φ(t) = 만기 프리미엄 (term premium)
  → 유동성 프리미엄 이론과 달리:
    φ(t)는 반드시 양(+)이 아닐 수 있고
    반드시 만기에 따라 증가하지 않을 수 있음
  → 수급 상황에 따라 혹형(humped) 곡선도 설명 가능

유동성 프리미엄 이론과의 차이:
  유동성 프리미엄: L(t) > 0, 단조 증가 → 항상 정상형 편향
  선호서식처:      φ(t)는 양수/음수 가능 → 다양한 형태 가능

한국 시장 사례:
  2024년 국고채 30년-50년 역전 현상:
  → 보험/연기금의 30Y 과도한 수요 → 30Y 수익률 하락
  → 50Y는 수요 부족 → 50Y 수익률 상대적 상승
  → 선호서식처 이론이 가장 잘 설명하는 현상
```

```
  → Kim & Wright (2005) 모형: Fed가 사용하는 기간 프리미엄 추정
  → Adrian, Crump & Moench (2013): NY Fed ACM 모형
  → 한국은행도 유사한 Term Premium 분해 모형 사용
```

### 9.7 현물금리, 선도금리, 파 수익률

기간구조 분석의 3가지 금리 개념:

```
(1) 현물금리 (Spot Rate, Zero Rate):
  제로쿠폰채의 만기수익률
  → t년 현물금리 s(t): P_zero = FV / (1+s(t))^t
  → 수익률 곡선의 가장 "순수한" 형태

(2) 선도금리 (Forward Rate):
  미래 특정 기간의 내재 금리
  → f(t₁, t₂): 시점 t₁에서 t₂까지의 내재 금리
  → (1+s(t₂))^t₂ = (1+s(t₁))^t₁ × (1+f(t₁,t₂))^(t₂-t₁)

  순간 선도금리(instantaneous forward rate):
  f(t) = lim_{Δ→0} f(t, t+Δ) = -d ln P(t) / dt

(3) 파 수익률 (Par Yield):
  파 채권(P = FV)의 쿠폰율
  → 가장 직관적이지만 쿠폰 재투자를 가정
  → 시장에서 호가되는 대부분의 수익률이 이 개념에 가까움

3개 곡선의 관계:
  정상형 곡선: Spot > Par > Forward 순서로 기울기 감소
    (Forward가 Spot보다 위에 위치)
  역전형 곡선: Forward < Spot < Par
    (Forward가 가장 낮음)
```

### 9.8 부트스트래핑 (Bootstrapping)

관측 가능한 쿠폰채 수익률에서 현물금리 곡선을 추출하는 방법:

```
알고리즘:

Step 1: 가장 짧은 만기의 현물금리
  1년 T-Bill(할인채): s(1) = 직접 관측 가능

Step 2: 2년 쿠폰채에서 s(2) 추출
  2년 파 채권 가격 P = FV (파이므로)
  FV = C/(1+s(1)) + (FV+C)/(1+s(2))^2
  → s(2) 해석적으로 구할 수 있음

Step 3: 3년 쿠폰채에서 s(3) 추출
  FV = C/(1+s(1)) + C/(1+s(2))^2 + (FV+C)/(1+s(3))^3
  → s(1), s(2)는 이미 알고 있으므로 s(3)만 미지수

Step N: 일반화
  FV = Σ_{t=1}^{N-1} C/(1+s(t))^t + (FV+C)/(1+s(N))^N
  → s(1)~s(N-1)은 이전 단계에서 결정
  → s(N)만 풀면 됨

수치 예시:
  1Y par yield = 3.00% → s(1) = 3.00%
  2Y par yield = 3.50% → 10,000 = 350/(1.03) + 10,350/(1+s(2))^2
    350/1.03 = 339.81
    10,350/(1+s(2))^2 = 10,000 - 339.81 = 9,660.19
    (1+s(2))^2 = 10,350/9,660.19 = 1.07139
    s(2) = 1.07139^0.5 - 1 = 3.506%
  → s(2) > par_yield(2Y): 정상형 곡선에서 현물금리 > 파 수익률

보간 (Interpolation):
  격자점 사이의 현물금리는 보간으로 추정
  → 선형 보간 (Linear): 단순하지만 불연속 선도금리
  → 3차 스플라인 (Cubic Spline): 매끄럽지만 변동 가능
  → NSS 모형 (Doc 35 §2): 파라메트릭 피팅으로 매끄러운 곡선
```

## 10. 유효듀레이션 (Effective Duration)

### 10.1 내재옵션과 듀레이션

Modified Duration은 현금흐름이 금리 변동에 따라 변하지 않는다는
가정 하에 유도된다. 그러나 내재옵션(Embedded Option)이 있는 채권은
금리에 따라 현금흐름 자체가 변한다:

```
내재옵션 유형:

(1) 콜러블 채권 (Callable Bond):
  발행자가 조기 상환할 권리
  → 금리 하락 → 조기상환 확률 ↑ → 유효 만기 단축

(2) 풋터블 채권 (Puttable Bond):
  투자자가 조기 상환을 요구할 권리
  → 금리 상승 → 조기 상환 요구 확률 ↑ → 유효 만기 단축

(3) MBS (주택저당증권):
  차주의 조기상환(prepayment) 옵션
  → 금리 하락 → 대환(refinancing) 증가 → 유효 만기 단축

문제:
  Modified Duration = Macaulay / (1+y/m)
  → 현금흐름 불변 가정에 기반
  → 콜러블/MBS: 금리 변동 → 현금흐름 변동 → D_mod가 부정확
```

### 10.2 유효듀레이션의 정의

```
유효듀레이션 (Effective Duration):

           P(y-Δy) - P(y+Δy)
D_eff = ─────────────────────
               2 · P₀ · Δy

여기서:
  P₀       = 현재 가격
  P(y-Δy)  = 수익률이 Δy만큼 하락했을 때의 가격
  P(y+Δy)  = 수익률이 Δy만큼 상승했을 때의 가격
  Δy       = 수익률 변동폭 (보통 25bp 또는 50bp)

핵심 차이:
  P(y-Δy)와 P(y+Δy)를 계산할 때, 금리 변동에 따른
  현금흐름 변화를 반영하여 가격을 재계산한다.
  → 옵션 가격결정 모형(이항트리, 몬테카를로) 사용

옵션미내재 채권:  D_eff ≈ D_mod
콜러블 채권:     D_eff < D_mod (저금리에서)
풋터블 채권:     D_eff < D_mod (고금리에서)
```

### 10.3 유효볼록성 (Effective Convexity)

```
              P(y-Δy) + P(y+Δy) - 2·P₀
C_eff = ────────────────────────────────
                   P₀ · (Δy)²

옵션미내재 채권: C_eff > 0 (양의 볼록성)
콜러블 채권 (저금리): C_eff < 0 (음의 볼록성)
  → 금리 하락 시 가격 상승이 제한 (콜 가격 한도)
  → 가격-수익률 곡선이 콜 가격 근처에서 오목해짐
```

### 10.4 Modified Duration vs Effective Duration 비교

```
┌──────────────────┬──────────────────┬──────────────────┐
│ 항목             │ Modified Duration│ Effective Duration│
├──────────────────┼──────────────────┼──────────────────┤
│ 현금흐름         │ 고정 가정        │ 변동 가능        │
│ 계산 방법        │ 해석적 공식      │ 수치적 근사      │
│ 필요 모형        │ 없음             │ 옵션 가격모형    │

│ 옵션미내재 결과   │ 동일             │ 동일 (≈ D_mod)  │

```

### 10.5 스프레드 체계 (Spread Measures)

채권 수익률을 벤치마크 대비 분해하는 방법:

```
(1) G-Spread (Government Spread):
  G-Spread = 채권 YTM - 동일 만기 국고채 YTM
  → 가장 단순, 선형 보간 국고채 기준
  → 문제: 쿠폰, 옵션 차이 미반영

(2) Z-Spread (Zero-Volatility Spread):
  국고채 현물금리 곡선(spot curve)에 일정한 스프레드를 더하여
  채권 가격을 재현하는 상수

  P = Σ CF_t / (1 + s(t) + z)^t

  여기서 s(t) = t년 국고채 현물금리, z = Z-spread
  → G-spread보다 정확 (spot curve 형태 반영)

(3) OAS (Option-Adjusted Spread):
  내재옵션 가치를 제거한 후의 스프레드
  → 이항트리/몬테카를로 모형에서 산출
  → OAS = Z-spread - Option Cost

  콜러블 채권: Option Cost > 0 → OAS < Z-spread
    → 발행자가 옵션을 보유 → 투자자에게 불리 → 스프레드에서 제거
  풋터블 채권: Option Cost < 0 → OAS > Z-spread
    → 투자자가 옵션을 보유 → 투자자에게 유리 → 스프레드에서 추가

(4) ASW Spread (Asset Swap Spread):
  채권의 고정금리를 변동금리(CD91 등)로 교환할 때의 스프레드
  → 채권 vs 스왑 시장의 상대가치 측정
  → 한국: CD91 + ASW bp 형태로 표시

스프레드 위계:
  단순 ←──────────────────────────→ 정교
  G-Spread  →  Z-Spread  →  OAS
```

### 10.6 한국 콜러블 채권 시장

```
한국의 내재옵션 채권 유형:

(1) 콜러블 회사채:
  → 발행 3-5년 후 콜 옵션 (조기상환 가능)
  → 주로 금융채(은행채, 보험채)에서 발행
  → 투자자에게 OAS 기준으로 추가 스프레드 요구

(2) 전환사채 (CB, Convertible Bond):
  → 주식으로 전환할 수 있는 권리 내재
  → 투자자 옵션 → OAS 관점에서는 풋터블과 유사
  → 주가 상승 시 전환 → 유효 만기 단축
  → Doc 35와의 관계: Merton DD 모형이 기업 자본구조를 분석

(3) 신종자본증권 (영구채):
  → 은행 BIS 비율 충족용
  → 5-10년 후 콜 옵션 (스텝업 조건 포함)
  → 콜 미행사 시 쿠폰 상향 (step-up) → 사실상 콜 행사 유인
  → 유효듀레이션: 콜 예상일까지의 기간에 가까움
```

## 부록 A: 채권 가격결정 주요 공식 요약

```
┌─────────────────────────────────────────────────────────────┐
│                 채권 분석 핵심 공식 모음                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 1. 가격:                                                    │
│    P = Σ_{t=1}^n  C/(1+y)^t  +  FV/(1+y)^n                │
│                                                             │
│ 2. 경과이자:                                                │
│    AI = C × (d / D)                                         │
│                                                             │
│ 3. 더티 가격:                                                │
│    P_dirty = P_clean + AI                                    │
│                                                             │
│ 4. Macaulay Duration:                                        │
│    D_mac = (1/P) × Σ  t × CF_t / (1+y)^t                   │
│                                                             │
│ 5. Modified Duration:                                        │
│    D_mod = D_mac / (1 + y/m)                                │
│                                                             │
│ 6. 가격 민감도 (1차):                                        │
│    ΔP/P ≈ -D_mod × Δy                                      │
│                                                             │
│ 7. DV01:                                                     │
│    DV01 = P × D_mod × 0.0001                                │
│                                                             │
│ 8. Convexity:                                                │
│    C = (1/P(1+y)²) × Σ  t(t+1) × CF_t / (1+y)^t           │
│                                                             │
│ 9. 가격 민감도 (2차):                                        │
│    ΔP/P ≈ -D_mod × Δy + (1/2) × C × (Δy)²                 │
│                                                             │
│ 10. Effective Duration:                                      │
│     D_eff = [P(y-Δy) - P(y+Δy)] / (2 × P₀ × Δy)           │
│                                                             │
│ 11. Key Rate Duration:                                       │
│     KRD_τ = -(1/P) × (∂P/∂y_τ)                             │
│                                                             │
│ 12. 영구채 듀레이션:                                         │
│     D_perp = (1+y) / y                                      │
│                                                             │
│ 13. 선도금리:                                                │
│     (1+s_n)^n = (1+s_m)^m × (1+f(m,n))^(n-m)               │
│                                                             │
│ 14. 면역 조건:                                               │
│     D_assets = D_liabilities,  C_assets ≥ C_liabilities     │
```

## 부록 B: 수익률 곡선 형태와 경기순환

```
수익률 곡선 형태 → 경기순환 위치 매핑:

(1) Bull Steepening (금리 하락 + 기울기 확대):
    장기금리 하락 > 단기금리 하락
    → 경기침체 초기, 중앙은행 금리인하 기대
    → 주식: 바닥 탐색기, 방어주 → 성장주 전환

(2) Bull Flattening (금리 하락 + 기울기 축소):
    단기금리 하락 > 장기금리 하락
    → 중앙은행 적극적 금리인하
    → 주식: 경기 회복 초기, 금융주 부담

(3) Bear Steepening (금리 상승 + 기울기 확대):
    장기금리 상승 > 단기금리 상승
    → 경기 확장기, 인플레이션 기대 상승
    → 주식: 가치주 우위, 성장주 부담

(4) Bear Flattening (금리 상승 + 기울기 축소):
    단기금리 상승 > 장기금리 상승
    → 중앙은행 긴축 (금리인상)
    → 주식: 경기 후반, 전반적 약세 가능성

한국 시장 특수성:
  → 미국 Treasury와 높은 동조화 (상관계수 0.6-0.8)
  → 그러나 한미 금리차 확대 시 외국인 자금 유출
  → BOK vs Fed 정책 괴리 → KTB 고유 변동 가능

  → Doc 35 §3에서 수익률 곡선 기울기 신호를 구체적으로 다룸
  → Doc 41 §3에서 RORO 레짐과 곡선 형태의 연결을 다룸
```
