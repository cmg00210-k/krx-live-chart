## 1. Miller (1977) 개인세 모형 (Personal Tax Model)

### 1.1 MM 정리 복습 — 왜 불충분한가

Doc 14 §3.1-3.2에서 MM 정리의 두 가지 버전을 다루었다:

```
MM 제1명제 (무세금, 1958):
  V_L = V_U
  → 완전 자본시장에서 자본구조는 기업가치와 무관

MM 제1명제 (법인세, 1963):
  V_L = V_U + Tc · D
  → 법인세가 존재하면 부채의 이자비용 세금절감(tax shield)으로
    기업가치가 증가. 극단적 결론: 100% 부채가 최적.

  Tc: 법인세율 (corporate tax rate)
  D: 부채의 시장가치
```

1963년 수정 모형의 문제는 명백하다. 만약 `V_L = V_U + Tc·D`가 옳다면,
모든 기업이 가능한 한 최대의 부채를 사용해야 한다. 그러나 현실에서:

1. 미국 기업의 평균 부채비율(D/(D+E))은 약 30-40%이다
2. 한국 기업의 평균 부채비율은 업종에 따라 20-60%로 다양하다
3. 많은 우량 기업(Apple, Samsung)이 순현금(net cash) 포지션을 유지한다

이 괴리를 설명하기 위해 두 가지 접근이 있다:

- **상충이론(Trade-off Theory)**: 파산비용이 세금절감 효과를 상쇄 (Doc 14 §3.3)
- **Miller (1977)**: 개인세를 도입하면 세금절감 효과 자체가 축소됨

### 1.2 Miller (1977) 모형의 직관

Merton Miller (1977), *Debt and Taxes*, Journal of Finance, 32(2), 261-275.
(1976년 미국재무학회 회장 취임 연설)

Miller의 핵심 통찰: **법인세만이 세금이 아니다.** 투자자는 이자소득과
배당소득(및 자본이득)에 대해 서로 다른 세율로 개인세를 낸다.

```
세금 흐름 분석 (Tax Flow Analysis):

  [1] 부채 투자자에게 흐르는 이자 1원의 세후 가치:
      이자 → 법인세 공제 → 투자자에게 지급 → 투자자 이자소득세
      세후 금액: 1 · (1 - Td)

      · 기업 단계: 이자비용은 세전 공제이므로 법인세 없음
      · 투자자 단계: 이자소득세율 Td 적용

  [2] 자기자본 투자자에게 흐르는 이익 1원의 세후 가치:
      영업이익 → 법인세 → 배당 또는 자본이득 → 투자자 개인세
      세후 금액: 1 · (1 - Tc) · (1 - Ts)

      · 기업 단계: 법인세율 Tc 적용
      · 투자자 단계: 자기자본 소득 유효세율 Ts 적용
        (배당세율과 자본이득세율의 가중평균)
```

여기서:
- **Tc** = 법인세율 (corporate tax rate)
- **Td** = 부채소득에 대한 개인세율 (personal tax on debt income)
- **Ts** = 자기자본 소득에 대한 유효 개인세율 (personal tax on equity income)

### 1.3 Miller (1977) 부채의 순이득 공식

부채 1원의 세금 측면 순이득(net gain from leverage)은 두 경로의 세후 가치
차이에서 도출된다:

```
부채의 세금 순이득 (Net Tax Gain from Debt):

  GL = [1 - (1-Tc)(1-Ts)/(1-Td)] · D

  이를 기업가치 공식에 대입하면:

  V_L = V_U + [1 - (1-Tc)(1-Ts)/(1-Td)] · D
```

**도출 과정:**

```
부채 D를 사용하는 기업의 영업이익 EBIT에서 시작:

부채기업(levered firm)의 총 세후 현금흐름:
  주주에게: (EBIT - rD·D)(1-Tc)(1-Ts)
  채권자에게: rD·D·(1-Td)

무부채기업(unlevered firm)의 총 세후 현금흐름:
  주주에게: EBIT·(1-Tc)(1-Ts)

차이(부채 사용의 순이득):
  GL = rD·D·(1-Td) - rD·D·(1-Tc)(1-Ts)
     = rD·D·[(1-Td) - (1-Tc)(1-Ts)]
     = rD·D·(1-Td)·[1 - (1-Tc)(1-Ts)/(1-Td)]

영구부채(perpetual debt) 가정 하에 현재가치:
  PV(GL) = D·[1 - (1-Tc)(1-Ts)/(1-Td)]

따라서:
  V_L = V_U + [1 - (1-Tc)(1-Ts)/(1-Td)] · D
```

### 1.4 특수 경우 분석

Miller 모형은 세율 조건에 따라 다양한 결론을 낳는다:

```
경우 1: 개인세 없음 (Td = Ts = 0)
  GL = [1 - (1-Tc)(1-0)/(1-0)] · D
     = [1 - (1-Tc)] · D
     = Tc · D
  → MM (1963)의 법인세 모형과 동일: V_L = V_U + Tc·D

경우 2: Ts = 0이고 Td = Tc (이자소득세 = 법인세)
  GL = [1 - (1-Tc)(1-0)/(1-Tc)] · D
     = [1 - 1] · D
     = 0
  → 부채의 세금 이점이 완전히 소멸!

경우 3: (1-Tc)(1-Ts) = (1-Td)  "Miller 균형"
  GL = [1 - 1] · D = 0
  → 역시 부채의 세금 이점 소멸

경우 4: Ts < Td (자본이득이 이자소득보다 유리하게 과세)
  (1-Ts) > (1-Td)이면 GL < Tc·D
  → 부채의 세금 이점이 법인세 모형보다 축소됨
```

### 1.5 Miller 균형 (Miller Equilibrium)

Miller (1977)의 가장 중요한 기여는 **균형 개념**의 도입이다.

```
Miller 균형의 논리:

투자자마다 개인세율(Td)이 다르다고 가정하자.
  - 면세 투자자(연기금, 비영리): Td = 0
  - 저세율 투자자: Td = 10%
  - 고세율 투자자: Td = 40%

기업이 부채를 발행하면:
  채권의 세전 수익률 = rD
  세후 수익률 = rD · (1 - Td)

[단계 1] 처음에는 면세 투자자(Td=0)가 채권을 매수
  → 이들에게 채권은 매력적 (세후 수익 = rD)

[단계 2] 기업들이 더 많은 부채를 발행하면
  채권 공급 증가 → 채권 수익률(rD) 상승 필요

[단계 3] 점점 더 높은 세율의 투자자를 채권 시장으로 끌어들여야 함
  Td가 높은 투자자에게는 rD가 충분히 높아야 매력적

[균형 조건] 한계 채권 투자자(marginal bondholder)의 세율 Td*에서:

  rD · (1 - Td*) = rE · (1 - Ts)    (세후 무차별)

  이때 한계 투자자의 세율이:
  (1 - Td*) = (1 - Tc)(1 - Ts)

  를 만족하면, 추가적인 부채 발행의 세금 이점이 0이 된다.

결론: 균형에서 개별 기업의 자본구조는 기업가치와 무관하다.
     (경제 전체의 부채 총량은 결정되지만, 개별 기업의 분배는 무관)
```

이것이 Miller의 "자본구조 퍼즐에 대한 퍼즐(puzzle about the puzzle)"이다:
개인세를 고려하면 부채의 세금 이점이 줄어들고, 균형에서는 사라질 수 있다.

### 1.6 한국 세제에의 적용

한국의 현행 세제(2024-2025 기준)를 Miller 모형에 대입한다:

```
한국 법인세 (Corporate Tax):
  과세표준           세율
  ≤ 2억 원          9%
  2억~200억 원      19%
  200억~3,000억 원  21%
  > 3,000억 원      24%

  실효세율(대기업 기준): Tc ≈ 20-24%
  (지방소득세 10% 부가 시 실질 Tc ≈ 22-26.4%)

배당소득세 (Tax on Dividends):
  기본: 14% + 지방소득세 1.4% = 15.4%
  금융소득종합과세(연간 금융소득 > 2,000만원):
    종합소득세 누진세율 적용 (최고 45% + 지방소득세 4.5% = 49.5%)

이자소득세 (Tax on Interest):
  기본: 14% + 지방소득세 1.4% = 15.4%
  금융소득종합과세: 배당과 동일하게 합산

양도소득세 (Capital Gains Tax):
  상장주식 대주주(10억 이상 보유 또는 일정 지분 이상):
    과세표준 3억 이하: 20% + 지방 2% = 22%
    과세표준 3억 초과: 25% + 지방 2.5% = 27.5%
  소액주주(2025 기준): 비과세
    (금융투자소득세 시행 유예 상태)
```

```
한국에서의 Miller 공식 적용:

[일반 투자자 — 금융소득종합과세 미해당]
  Tc = 0.22 (법인세 실효세율, 지방세 포함)
  Td = 0.154 (이자소득세)
  Ts = 0.154 (배당소득세) 또는 0 (소액주주 양도차익)

  Case A: 배당 중심 (Ts = 0.154)
    GL = [1 - (1-0.22)(1-0.154)/(1-0.154)] · D
       = [1 - (0.78)(0.846)/(0.846)] · D
       = [1 - 0.78] · D
       = 0.22 · D  (= Tc · D)

    → Ts = Td일 때 개인세가 상쇄되어 MM (1963)과 동일한 결론!
    → 한국의 배당소득세 = 이자소득세이므로 개인세 효과 중립

  Case B: 자본이득 중심 (Ts ≈ 0, 소액주주 비과세)
    GL = [1 - (1-0.22)(1-0)/(1-0.154)] · D
       = [1 - 0.78/0.846] · D
       = [1 - 0.922] · D
       = 0.078 · D

    → 부채의 세금 이점이 Tc·D(=0.22D)에서 0.078D로 대폭 축소!
    → 소액주주에게 자본이득이 비과세이면, 자기자본의 상대적 세금 이점 발생

  Case C: 고소득 투자자 (금융소득종합과세, Td = Ts = 0.40)
    GL = [1 - (1-0.22)(1-0.40)/(1-0.40)] · D
       = [1 - 0.78] · D
       = 0.22 · D

    → Case A와 동일: Ts = Td이면 항상 GL = Tc·D
```

### 1.7 한국 시장에 대한 함의

```
Miller 모형의 한국 시장 함의:

1. Ts ≈ Td인 한국 투자자 대다수에게:
   부채의 세금 이점 ≈ Tc · D (순수 법인세 절감)
   → 법인세 모형(MM 1963)이 상당히 유효한 근사

2. 그러나 소액주주(양도소득 비과세)에게는:
   부채의 세금 이점 < Tc · D
   → 기업이 부채보다 자기자본을 선호할 이론적 근거

3. 금융투자소득세(금투세) 시행 시 (Ts > 0):
   Ts가 Td에 가까워질수록 GL → Tc·D
   → 금투세는 역설적으로 부채 조달의 상대적 매력도를 높일 수 있음

4. Miller 균형의 실무적 의미:
   한국 채권시장의 세후 균형 수익률은
   면세 기관(국민연금, 퇴직연금)의 존재에 의해 결정됨
   → 개인 투자자의 세후 채권 수익률은 상대적으로 불리
```

### 1.8 DeAngelo-Masulis (1980) 확장

DeAngelo, H. & Masulis, R.W. (1980), *Optimal capital structure under corporate
and personal taxation*, Journal of Financial Economics, 8(1), 3-29.

Miller (1977)의 한계: 기업 간 자본구조 차이를 설명하지 못한다(균형에서 무관).
DeAngelo-Masulis는 **비부채 세금절감(non-debt tax shields)**을 도입하여
기업별 최적 자본구조가 존재함을 보인다.

```
비부채 세금절감 (Non-Debt Tax Shields, NDTS):
  — 감가상각비 (depreciation)
  — 투자세액공제 (investment tax credits)
  — 이월결손금 (tax loss carryforwards)

논리:
  NDTS가 큰 기업은 이미 과세소득이 줄어 있으므로
  부채의 이자비용 공제로 인한 추가 세금절감이 작다.

  NDTS ↑ → 부채의 한계 세금이점 ↓ → 최적 부채비율 ↓

실증적 함의:
  — 감가상각비/총자산 비율이 높은 기업(제조업) → 부채비율 낮음
  — 유형자산이 적은 기업(IT, 서비스) → 세금절감 위해 더 많은 부채 사용?
    (실제로는 유형자산이 담보가치가 높아 차입이 용이 → 반대 결과)
    → 상충이론과 대리인이론의 결합이 필요

한국 적용:
  — 한국 세법의 감가상각 제도: 정액법/정률법 선택 가능
  — R&D 세액공제: 중소기업 25-50%, 대기업 0-2%
  — 임시투자세액공제: 경기부양 시 한시적 적용
  → 대기업보다 중소기업의 NDTS가 상대적으로 크다
```

### 1.9 MM 정리에서 WACC로의 연결

법인세가 있는 MM 정리(1963)는 WACC 공식과 직접 연결된다. 레버리지 기업의 WACC는
무차입 자기자본비용($r_U$)보다 세금절감분만큼 낮아진다:

$$WACC_L = r_U \times \left(1 - T_c \cdot \frac{D}{V}\right)$$

여기서 $r_U$는 무차입 기업(all-equity firm)의 자기자본비용이자 WACC이고,
$T_c \cdot D/V$는 이자비용 세금절감(tax shield)의 가치 비중이다.

```text
도출 경로:

V_L = V_U + Tc·D  (MM 1963)

무차입 기업 가치: V_U = NOPAT / r_U  (영구현금흐름 가정)
레버리지 기업 가치: V_L = NOPAT / WACC_L

NOPAT / WACC_L = NOPAT / r_U + Tc·D
1 / WACC_L = 1/r_U + Tc·D / NOPAT

NOPAT/V_U = r_U  → NOPAT = r_U · V_U = r_U · V_L / (1 + Tc·D/V_U)

단순화 (D/V 기준 재정리):
  WACC_L = r_U · (1 - Tc · D/V_L)

함의:
  — D/V ↑ → WACC_L ↓ → 기업가치 V_L ↑ (단조 증가)
  — 100% 부채가 최적 → 비현실적 결론
  → 이것이 상충이론(Trade-off Theory)의 출발점:
    파산비용과 대리인비용이 세금절감 이점을 상쇄하는 최적점 존재
```

**실무 주의:** 이 공식은 MM 가정(영구부채, 고정 D 수준) 하에서 성립한다.
부채비율을 시장가 기준으로 재조정하는 경우 Miles-Ezzell (1980) 수정식을 사용해야 한다.

## 2. 대리인비용 자본구조 (Agency Costs & Capital Structure)

### 2.1 Jensen-Meckling (1976) 자본구조 맥락에서의 재해석

Doc 33 §2.1에서 대리인 비용의 기본 분해(AC = MC + BF + RL)를 다루었다.
본 절에서는 이를 **자본구조 최적화의 프레임워크**로 확장한다.

Jensen, M.C. & Meckling, W.H. (1976), *Theory of the firm: Managerial behavior,
agency costs and ownership structure*, Journal of Financial Economics, 3(4), 305-360.

```
자기자본의 대리인비용 (Agency Cost of Equity):

경영자의 지분율 α가 하락할수록:
  — 경영자의 소비에서 기업 자원 이용 비율 증가 (perquisite consumption)
  — 경영자 노력의 한계편익이 α만큼만 자신에게 귀속
  — 외부 주주의 감시 필요성 증가 → MC 상승

  AC_equity(α) = MC(α) + BF(α) + RL(α)

  특성: α ↓ → AC_equity ↑ (단조 증가)

자기자본의 대리인비용을 줄이는 방법:
  (1) 경영자 지분 확대 (α ↑)
  (2) 배당 증가 → 잉여현금흐름 축소 (Jensen 1986)
  (3) 부채 확대 → 원리금 상환 의무로 경영자 재량 제한
```

```
부채의 대리인비용 (Agency Cost of Debt):

주주-채권자 갈등에서 발생하는 두 가지 핵심 문제:

[1] 자산 대체 문제 (Asset Substitution / Risk Shifting):
    Jensen & Meckling (1976)

    — 부채 발행 후 주주는 고위험 프로젝트를 선호
    — 성공 시: 주주가 초과이익 향유 (채권자 수익은 고정)
    — 실패 시: 유한책임(limited liability)으로 손실 제한
    — 주주의 지분은 콜옵션 → 변동성 증가가 주주에게 유리

    수학적 표현 (옵션 비유):
      E = C(V, D, σ)    주주의 지분 = 기업가치에 대한 콜옵션
      ∂C/∂σ > 0         변동성 증가 → 콜옵션 가치 증가 → 주주 이익
      B = V - C(V, D, σ) 채권자 = 풋옵션 매도
      ∂B/∂σ < 0         변동성 증가 → 채권 가치 하락 → 채권자 손실

[2] 과소투자 문제 (Underinvestment / Debt Overhang):
    Myers (1977), Determinants of corporate borrowing, JFE, 5(2), 147-175.

    — 부채가 과다한 기업에서 NPV > 0 프로젝트도 기각될 수 있음
    — 이유: 투자의 이익이 채권자에게 먼저 귀속 (채무 우선)
    — 주주 입장: 투자 비용은 100% 부담, 이익은 채권자와 공유
    — 특히 재무적 곤경(financial distress)에 처한 기업에서 심각

    수치 예시:
      기업 부채 D = 100, 기업가치 V = 80 (부도 상태)
      새 프로젝트: 투자 I = 20, 확실한 수익 30 → NPV = 10
      주주 수익: max(V + 30 - D - I, 0) = max(80 + 30 - 100 - 20, 0)
              = max(-10, 0) = 0
      → NPV > 0인데도 주주가 투자를 거부 (이익이 채권자에게 귀속)
```

### 2.2 총 대리인비용과 최적 자본구조

자기자본과 부채의 대리인비용을 결합하면:

```
총 대리인비용 (Total Agency Costs):

  AC_total(D/E) = AC_equity(D/E) + AC_debt(D/E)

  AC_equity: D/E ↑ → ↓ (부채가 경영자 재량 제한)
  AC_debt:   D/E ↑ → ↑ (자산 대체, 과소투자 심화)

  최적 D/E*: dAC_total/d(D/E) = 0

                  AC_total
                 /         \
  AC_equity ----/            \---- AC_debt
               /              \
              * (최적 D/E)

  D/E → 0: AC_equity 최대, AC_debt ≈ 0
  D/E → ∞: AC_equity ≈ 0, AC_debt + 파산비용 최대
```

### 2.3 확장된 상충이론 (Extended Trade-off Theory)

세금 효과(§1)와 대리인비용을 결합한 기업가치 공식:

```
확장된 기업가치 공식:

  V_L = V_U + PV(Tax Shield) - PV(Financial Distress)
            - PV(AC_debt) + PV(AC_equity 절감)

상세 분해:

  V_L = V_U
      + [1 - (1-Tc)(1-Ts)/(1-Td)] · D          ... Miller (1977) 세금효과
      - PV(직접 파산비용)                          ... 법률비용, 관리비용
      - PV(간접 파산비용)                          ... 고객 이탈, 인재 유출
      - PV(자산 대체 비용)                         ... Jensen-Meckling (1976)
      - PV(과소투자 비용)                          ... Myers (1977)
      + PV(부채의 규율 효과)                       ... Jensen (1986)
```

### 2.4 Jensen (1986) 잉여현금흐름 가설과 부채의 규율

Doc 33 §2.1에서 FCF 가설의 기본 개념을 다루었다. 여기서는 자본구조와의
직접적 연결을 형식화한다.

```
Jensen (1986) 잉여현금흐름 가설 (Free Cash Flow Hypothesis):

정의:
  FCF = 영업현금흐름 - 양(+)의 NPV 프로젝트 투자액

FCF > 0인 기업에서 경영자의 유인:
  (1) 제국 건설 (empire building): NPV < 0 프로젝트 투자
  (2) 과도한 보상, 비필수적 지출 (perquisite consumption)
  (3) 비효율적 다각화 (conglomerate discount)

부채의 규율 효과 (Disciplinary Role of Debt):
  — 원리금 상환 의무 → FCF를 경영자가 아닌 채권자에게 지급
  — 부도 위협 → 경영자 노력 증가 (일자리 상실 위험)
  — "부채는 경영자에게 칼을 들이대는 것" (Jensen)

  최적 부채 수준:
    FCF가 큰 기업(성숙, 저성장) → 부채비율 높게 유지
    FCF가 작은 기업(고성장, 투자 많음) → 부채비율 낮게 유지

실증적 증거:
  — LBO(leveraged buyout): 부채 급증 후 경영 효율화 관찰
  — 석유 산업(1980년대): 유가 하락 후 FCF 축소 → 과잉투자 감소
```

### 2.5 한국 재벌 구조와 대리인비용의 특수성

Doc 33 §3.3에서 재벌의 터널링 문제를 다루었다. 여기서는 이것이 자본구조에
미치는 영향을 분석한다.

```
한국 재벌의 대리인비용 특수성:

[1] 전통적 대리인 문제 (경영자 vs 주주):
    — 서구: 소유 분산 → 경영자가 주주 이익에 반하여 행동
    — 한국: 지배주주(오너)가 경영권 보유 → 전통적 대리인 문제 약화

[2] 한국형 대리인 문제 (지배주주 vs 소액주주):
    — 지배주주의 지배권(voting rights) >> 현금흐름권(cash flow rights)
    — 피라미드 출자, 순환출자로 적은 자본으로 그룹 전체 지배
    — 터널링(tunneling): 계열사 간 부당 내부거래로 가치 이전
    — Bae, Kang & Kim (2002): 한국 M&A에서 가치 파괴 실증

    형식화:
      지배주주의 지분율: α (자기자본 기준)
      지배주주의 지배력: C (의결권 기준)
      과잉지배 비율: C/α >> 1 (피라미드/순환출자 효과)

      지배주주의 사적이익 추출 유인:
        B(α) = (1 - α) · S(α)    S: 사적이익(private benefits)
        α ↓ → B ↑ (자기 부담 줄어듦 → 터널링 유인 증가)

[3] 자본구조에의 영향:
    — 지배주주는 지분 희석을 극도로 꺼림 → 부채 선호
    — 그러나 과도한 부채 → 파산 시 경영권 상실 위험
    — 결과: 재벌 계열사의 부채비율은 비재벌보다 높은 경향
    — 1997 외환위기: 재벌의 과도한 부채가 위기를 심화

[4] 제도적 변화:
    — 부채비율 200% 규제 (1999-2009)
    — 순환출자 금지 (2014 공정거래법 개정)
    — 최근: 지배구조 개선 → 소액주주 보호 강화 추세
```

### 2.6 대리인비용 측정의 실무적 접근

```
대리인비용 프록시 변수 (Proxy Variables):

[직접 측정 불가 → 간접 추정]

(1) 자산 회전율 (Asset Utilization Ratio):
    AUR = 매출액 / 총자산
    → AUR이 낮을수록 자산의 비효율적 활용 → AC 높음

(2) 판관비율 (SGA Ratio):
    SGA_ratio = 판매관리비 / 매출액
    → SGA_ratio가 높을수록 경영자의 과도한 지출 → AC 높음

(3) Tobin's Q:
    Q = (시가총액 + 부채 장부가) / 총자산 장부가
    → Q < 1: 자산이 과소평가 또는 비효율적 운용 → AC 높음 시사

(4) 경영자 보상 대비 성과:
    Compensation_efficiency = 주주 수익률 / 경영자 총보상
    → 보상은 높은데 성과는 낮으면 → AC 높음

CheeseStock 적용:
  financials.js의 투자 점수(investment score)에서
  자산 효율성 지표를 통해 간접적으로 대리인비용을 반영
  → Doc 14 §9 참조
```

## 3. 시그널링 이론 (Signaling Theory in Corporate Finance)

### 3.1 정보 비대칭의 문제 — 기업재무 맥락

Doc 31 §3.4에서 Akerlof (1970)의 역선택과 Spence (1973)의 신호 이론을
일반적 시장 맥락에서 다루었다. 본 절에서는 이를 **기업의 재무 의사결정**에
특화하여 확장한다.

```
기업재무에서의 정보 비대칭 구조:

  경영자(내부자)     →  알고 있음: 기업의 진정한 가치, 미래 현금흐름
  외부 투자자       →  알지 못함: 재무제표와 공시만 관찰 가능

  이 비대칭이 만드는 문제들:
  (1) 역선택(Adverse Selection):
      저품질 기업이 고품질 기업으로 위장 → 시장 전체 저평가
  (2) 도덕적 해이(Moral Hazard):
      자금 조달 후 경영자가 약속과 다른 행동
  (3) 시그널링(Signaling):
      고품질 기업이 비용을 수반하는 행동으로 자신의 유형을 드러냄
```

### 3.2 Ross (1977) 재무 시그널링 모형

Ross, S.A. (1977), *The determination of financial structure: The incentive-signalling
approach*, Bell Journal of Economics, 8(1), 23-40.

Ross는 부채 수준 자체가 기업 품질의 **신호(signal)**가 될 수 있음을 보였다.

```
Ross (1977) 모형의 구조:

가정:
  — 경영자는 기업의 미래 현금흐름 분포를 알고 있음
  — 투자자는 모름 → 기업가치를 정확히 평가할 수 없음
  — 경영자의 보상은 기업가치에 연동

경영자 보상 함수:
  W = γ₀ + γ₁ · V_t    (기업이 생존할 때, t=1)
  W = γ₀ + γ₁ · V_t - L (기업이 파산할 때)

  γ₀: 고정 보상
  γ₁: 기업가치 연동 계수
  V_t: t기의 기업가치 (시장 평가)
  L: 파산 시 패널티 (L > 0, 충분히 큼)

시그널링 메커니즘:
  — 경영자가 부채 수준 D를 선택
  — 높은 D → 파산 확률 증가 → 패널티 L의 기대비용 증가
  — 고품질 기업: 미래 현금흐름이 충분 → 파산 위험 낮음 → 높은 D 감당 가능
  — 저품질 기업: 높은 D를 선택하면 파산 확률 높음 → 패널티 기대비용 과다

분리 균형 (Separating Equilibrium):
  D_H > D_L (고품질 기업이 더 많은 부채 사용)

  투자자의 합리적 추론:
    D ≥ D* → 고품질로 판단 → 높은 기업가치 부여
    D < D* → 저품질로 판단 → 낮은 기업가치 부여

  D*는 저품질 기업이 모방할 수 없는 수준:
    E[W(D*, high quality)] ≥ E[W(D*, low quality)]
    → 저품질 기업의 파산 기대비용이 시그널의 이익을 초과

핵심 조건 (Spence 교차 조건의 기업재무 버전):
  시그널 비용이 품질에 반비례해야 함
  → 고품질 기업의 부채 유지 비용 < 저품질 기업의 부채 유지 비용
  → 이는 고품질 기업의 파산 확률이 낮기 때문에 자연스럽게 성립
```

### 3.3 Bhattacharya (1979) 배당 시그널링 모형

Bhattacharya, S. (1979), *Imperfect information, dividend policy, and "the bird in
the hand" fallacy*, Bell Journal of Economics, 10(1), 259-270.

```
Bhattacharya (1979) 배당 시그널링:

핵심 직관:
  배당은 세금 비용을 수반한다 (배당소득세 > 자본이득세인 경우).
  이 세금 비용이 시그널을 credible하게 만든다.
  → Spence (1973)의 교육 시그널링과 동일한 논리

모형 구조:
  — 기업의 투자 수익률 θ는 경영자만 알고 있음
  — 경영자가 배당 수준 D를 선택 (사전적 약속)
  — 실현 현금흐름 < D이면 외부 자금 조달 필요 (거래비용 τ 발생)

  경영자의 목적함수:
    max_D  V(D) - Tax_cost(D) - Expected_transaction_cost(D)

  Tax cost:
    배당의 세금 비용 = Td · D  (배당소득세)

  Transaction cost (배당 약속을 이행하기 위한 외부 조달):
    E[τ · max(D - CF, 0)]
    → 현금흐름이 배당보다 적을 확률에 비례

  고품질 기업 (θ 높음):
    — CF > D일 확률 높음 → 거래비용 기대값 낮음
    — 높은 배당 약속이 가능

  저품질 기업 (θ 낮음):
    — CF < D일 확률 높음 → 거래비용 기대값 높음
    — 높은 배당을 모방하면 비용이 과다

분리 균형:
  D*(θ): θ의 단조 증가 함수
  → 높은 배당 = 높은 품질의 신호

한국 시장 적용:
  한국에서 Ts ≈ Td (배당소득세 ≈ 이자소득세 = 15.4%)
  → 배당의 세금 비용이 상대적으로 낮음
  → 시그널링 비용이 낮다면, 시그널의 credibility도 약화?
  → 대안: 한국에서는 배당보다 자사주 매입이 더 강한 시그널일 수 있음
    (자사주 매입 → 경영자가 시장에서 주식을 매수 → 직접적 자금 투입)
```

### 3.4 Lintner (1956) 배당 평활화 모형

Lintner, J. (1956), *Distribution of incomes of corporations among dividends,
retained earnings, and taxes*, American Economic Review, 46(2), 97-113.

이론적 모형이 아닌 **실증적 관찰**에서 출발한 고전적 연구이다.

```
Lintner (1956) 배당 조정 모형 (Partial Adjustment Model):

  D_t - D_{t-1} = c · (D*_t - D_{t-1}) + u_t

  D_t: t기의 배당
  D_{t-1}: 전기의 배당
  D*_t: 목표 배당 = τ · E_t  (τ: 목표 배당성향, E: 이익)
  c: 조정 속도 (0 < c < 1, speed of adjustment)
  u_t: 오차항

정리하면:
  D_t = c · τ · E_t + (1-c) · D_{t-1} + u_t

Lintner의 발견:
  (1) 기업은 안정적 배당 증가를 선호 (배당 평활화)
  (2) 이익 증가가 영구적(permanent)이라고 판단될 때만 배당 증가
  (3) 배당 삭감에 극도로 저항적 (하방 경직성)
  (4) 조정 속도 c ≈ 0.3-0.5 (미국 기업 평균)
      → 이익 변동의 30-50%만 당기 배당에 반영

시그널링과의 연결:
  — 배당 증가: 경영자가 미래 이익의 영구적 개선을 확신 → 강한 양의 시그널
  — 배당 유지: 중립 (기존 이익 수준 유지 전망)
  — 배당 삭감: 경영자도 더 이상 유지 불가 → 매우 강한 부정적 시그널
  — 비대칭성: 삭감의 정보 함량 > 증가의 정보 함량

한국 실증:
  — 한국 기업의 c ≈ 0.2-0.4 (미국보다 느린 조정)
  — 재벌 계열사: 배당보다 내부유보 선호 → c가 더 낮은 경향
  — 금융주(은행): c가 상대적으로 높음 (규제 산업, 안정적 이익)
```

### 3.5 Myers-Majluf (1984) 자본조달 서열이론의 정식 모형

Doc 14 §3.4에서 서열이론의 직관과 기본 결론을 다루었다. 여기서는
원논문의 모형을 정식으로 전개한다.

```
Myers & Majluf (1984) 모형 구조:

타이밍:
  t=0: 경영자가 기업가치(자산 a)와 신규 투자 기회(NPV b)를 알고 있음
       외부 투자자는 이를 모름 (a와 b의 분포만 알고 있음)
  t=1: 정보 공개, 진정한 가치 실현

경영자의 의사결정:
  — 신규 투자에 필요한 자금 I (자기자금 부족)
  — 주식 발행으로 I를 조달할지 여부를 결정

기존 주주의 관점 (경영자는 기존 주주의 이익을 대리):

  주식 발행 시 기존 주주의 부(wealth):
    W_issue = (N_old / (N_old + N_new)) · (a + b + I)

    N_old: 기존 주식 수
    N_new: 신규 발행 주식 수
    a: 기존 자산의 진정한 가치
    b: 신규 투자의 NPV
    I: 투자 금액 (= 주식 발행 수익)

  주식 미발행 시 기존 주주의 부:
    W_no_issue = a   (투자 기회 포기)

  경영자의 발행 조건:
    W_issue ≥ W_no_issue
    → (N_old / (N_old + N_new)) · (a + b + I) ≥ a

핵심 결과:

  [1] 역선택 문제:
    — a가 높은(자산이 고평가된 것이 아닌, 진정 가치가 높은) 기업:
      → 시장은 평균적 a를 가정하여 주식 가격 산정
      → 주식이 저평가 → 신규 주주에게 부의 이전 발생
      → 기존 주주 손해 → 발행 꺼림

    — a가 낮은 기업:
      → 주식이 고평가 → 기존 주주 이익 → 발행 선호

  [2] 투자자의 합리적 추론:
    — 주식 발행 공시 → "경영자가 주가가 고평가라 판단"으로 해석
    — 주가 하락 (음의 공시효과)

  [3] 과소투자 문제 (Underinvestment due to information asymmetry):
    — 고품질 기업이 NPV > 0 프로젝트를 포기할 수 있음
    — 주식 발행의 역선택 비용 > 투자의 NPV일 때
    — 사회적 후생 관점에서 비효율적

  [4] 자본조달 서열 (Pecking Order):
    — 내부 자금: 정보 비대칭 문제 없음 (1순위)
    — 부채: 역선택 비용이 작음 (채권 가치는 기업가치에 덜 민감) (2순위)
    — 주식: 역선택 비용이 가장 큼 (3순위)

    정보 민감도(information sensitivity):
      내부 자금(0) < 안전 부채 < 위험 부채 < 전환사채 < 주식
      → 정보 비대칭이 클수록 정보 민감도가 낮은 증권 선호
```

### 3.5a Myers-Majluf (1984) 역선택 균형 조건 — 정식 수식

§3.5의 서술 모형을 수식으로 정식화한다.

경영자가 기업의 진정가치($V_{true}$)를 알지만 시장은 시장가($V_{market}$)만 관찰할 수
있을 때, 기존 주주 관점에서 주식 발행이 합리적인 조건:

$$NPV_{\text{project}} > I \times \frac{V_{true} - V_{market}}{V_{market} + I}$$

여기서 $I$는 신규 투자 금액(= 주식 발행 조달액)이다.

```text
도출 경로:

주식 발행 시 신규 투자자가 수령하는 지분 비율:
  α = I / (V_market + I)   (시장 가격 기준 발행가 결정)

주식 발행 후 기존 주주의 부:
  W_issue = (1 - α) · (V_true + NPV_project + I)
           = (V_market / (V_market + I)) · (V_true + NPV_project + I)

주식 미발행 시 기존 주주의 부:
  W_no_issue = V_true

발행 조건 (W_issue ≥ W_no_issue):
  (V_market / (V_market + I)) · (V_true + NPV + I) ≥ V_true

정리하면:
  NPV ≥ I · (V_true - V_market) / (V_market + I)

결론 분석:
  [A] V_true > V_market (저평가): 우변 양수 → NPV가 충분히 크지 않으면 발행 기각
      → 과소투자(underinvestment) 발생
  [B] V_true = V_market (공정평가): 우변 = 0 → NPV > 0이면 항상 발행
      → 정보 비대칭이 없으면 투자 효율 달성
  [C] V_true < V_market (고평가): 우변 음수 → NPV < 0 프로젝트도 발행 가능
      → 고평가 시 주식 발행으로 기존 주주 이익 극대화
```

이것이 자본조달 서열이론(Pecking Order)의 수학적 기초이다:
내부자금(정보 비대칭 비용 0) > 부채(우변 분자 ≈ 0) > 주식(우변 양수 위험).

## 4. MM 제3명제 — 배당무관련 (Dividend Irrelevance)

### 4.1 Miller-Modigliani (1961) 배당무관련 명제

Miller, M.H. & Modigliani, F. (1961), *Dividend policy, growth, and the valuation
of shares*, Journal of Business, 34(4), 411-433.

MM의 배당무관련 명제는 1958년 자본구조 무관련 명제의 논리적 연장선이다.

```
MM 배당무관련 명제 (Dividend Irrelevance Proposition):

완전 자본시장 가정:
  (1) 세금 없음 (법인세, 소득세 모두)
  (2) 거래비용 없음 (발행비용, 매매수수료)
  (3) 정보 비대칭 없음 (경영자와 투자자 동일 정보)
  (4) 대리인비용 없음
  (5) 기업의 투자 정책이 배당 정책과 독립적으로 결정됨

명제:
  기업의 가치는 투자 정책에 의해 결정되며,
  배당 정책은 기업가치에 영향을 미치지 않는다.

  V = Σ_{t=1}^{∞} E[CF_t] / (1 + r)^t

  여기서 CF_t = 영업현금흐름_t - 투자지출_t = 총 배분가능 현금흐름
  → 이 중 얼마를 배당으로, 얼마를 자사주 매입으로 배분하든 V는 불변
```

### 4.1a MM (1961) 배당무관련 핵심 항등식

MM 배당무관련 명제의 수식적 핵심은 단일 기간 주가 결정 항등식이다:

$$V_0 = \frac{D_1 + P_1}{1 + r}$$

여기서 $V_0$는 현재 주가, $D_1$은 기말 배당, $P_1$은 배당 지급 후 주가,
$r$은 주주의 요구수익률이다.

```text
배당 증가 시 자기 상쇄 메커니즘:

배당 D_1 증가 → 현금 유출 → 기업이 신주 발행으로 부족분 조달

신주 발행 주식 수 ΔN:
  P_1 · ΔN = D_1_증가분   (신주 발행가 = 배당 후 주가 P_1)

기존 주주 지분 희석:
  P_1_new = P_1_old · (N / (N + ΔN))   (희석 효과)

항등식 정리:
  D_1↑ 만큼 P_1↓  →  D_1 + P_1 = 불변  →  V_0 = 불변

수치 예시:
  현재 주가 V_0 = 100, r = 10%
  배당 D_1 = 0일 때: P_1 = 110  →  V_0 = 110/1.1 = 100
  배당 D_1 = 10일 때: P_1 = 100  →  V_0 = (10+100)/1.1 = 100
  배당 D_1 = 20일 때: P_1 = 90  →  V_0 = (20+90)/1.1 = 100
```

결론: 배당이 증가하면 $D_1$↑이지만 신주 발행 희석으로 $P_1$↓하여 $V_0$는 불변이다.
투자자는 주식 매도로 "자가 배당(homemade dividend)"을 생성할 수 있으므로,
완전시장에서 배당정책은 기업가치에 영향을 미치지 않는다.

### 4.2 증명 스케치 — 자가 배당 (Homemade Dividends)

```
자가 배당 논증 (Homemade Dividend Argument):

투자자가 원하는 배당 수준과 기업의 실제 배당 수준이 다를 때:

Case 1: 기업이 배당을 적게 지급 (투자자는 더 원함)
  → 투자자가 주식 일부를 매도하여 현금 확보
  → 매도 대금 = "자가 배당" (homemade dividend)
  → 세금·거래비용이 없으므로 비용 없이 가능

Case 2: 기업이 배당을 많이 지급 (투자자는 적게 원함)
  → 투자자가 배당금으로 추가 주식 매수
  → 실질적으로 배당을 재투자한 효과
  → 세금·거래비용이 없으므로 비용 없이 가능

수치 예시:
  기업 A: 주가 100, 배당 0 (무배당)
  기업 B: 주가 100, 배당 5 → 배당 후 주가 95

  투자자가 5의 현금이 필요할 때:
    기업 A 보유: 주식 5% 매도 → 현금 5, 잔여 주식가치 95
    기업 B 보유: 배당 5 수령, 주식가치 95
    → 두 경우 모두 현금 5 + 주식 95 = 100 (동일)

  투자자가 현금이 불필요할 때:
    기업 A 보유: 주식가치 100
    기업 B 보유: 배당 5 재투자 → 주식가치 100 (동일)

결론: 투자자가 자유롭게 거래할 수 있다면, 기업의 배당 정책은 무관하다.
```

### 4.3 배당무관련 명제의 핵심 전제 — 투자 정책의 독립성

```
배당무관련 명제의 핵심 전제:

"기업의 투자 정책이 배당 정책과 독립적으로 결정됨"

이것이 의미하는 바:
  — 기업은 NPV > 0인 모든 프로젝트에 투자 (최적 투자 정책)
  — 투자에 필요한 자금은 내부 자금 또는 외부 조달로 확보
  — 배당은 투자 후 잔여 현금흐름에서 지급

잔여 배당 모형 (Residual Dividend Model):
  D = 순이익 - (자기자본비율 × 투자지출)
    = NI - (E/(D+E)) × CAPEX

  → 배당은 투자 결정의 부산물이지, 독립적 의사결정이 아님

이 전제가 위반되면 배당이 중요해진다:
  (1) 배당 지급으로 투자 자금 부족 → 양의 NPV 프로젝트 포기
      → 배당 증가 = 기업가치 감소 (투자 정책 왜곡)
  (2) 배당 미지급으로 잉여현금 축적 → 음의 NPV 프로젝트 투자
      → 배당 감소 = 기업가치 감소 (대리인비용, Jensen 1986)
```

### 4.4 세금 고객효과 (Tax Clientele Effect)

Elton, E.J. & Gruber, M.J. (1970), *Marginal stockholder tax rates and the
clientele effect*, Review of Economics and Statistics, 52(1), 68-74.

세금을 도입하면 배당무관련 명제는 성립하지 않는다. 그러나 **고객효과**에
의해 배당 정책은 여전히 중립적일 수 있다.

```
세금 고객효과 (Tax Clientele Effect):

전제:
  — 배당소득세율(Td) > 자본이득세율(Tg)인 세제 (미국, 대부분의 국가)
  — 투자자마다 한계세율이 다름

세율별 선호:
  고세율 투자자 (개인 고소득자):
    → 배당의 세후 수익 = D · (1 - Td_high)
    → 자본이득의 세후 수익 = G · (1 - Tg_high)
    → Td_high > Tg_high이면 저배당주 선호

  저세율 투자자 (비영리, 연기금):
    → Td ≈ Tg ≈ 0
    → 배당과 자본이득에 무차별 → 고배당주도 거부하지 않음

  중간 세율 투자자:
    → 세율에 따라 적절한 배당 수준의 주식 선택

균형 (Clientele Equilibrium):
  — 고배당 기업 → 저세율 투자자 고객층 형성
  — 저배당 기업 → 고세율 투자자 고객층 형성
  — 각 고객층이 해당 배당 수준에 만족 → 균형

  균형에서:
    ∂V/∂(D/E) = 0  (배당성향 변화가 가치에 영향 없음)
    → 가치에 영향이 없는 것은 배당 자체가 아니라,
      배당 수준이 고객층과 이미 매칭되어 있기 때문

Elton-Gruber 검증 방법:
  배당 낙일(ex-dividend day)의 주가 하락폭을 관찰

  이론:
    주가 하락 = D · (1 - Td) / (1 - Tg)

  완전 시장: 주가 하락 = D (배당금만큼 정확히 하락)
  세금 존재: 주가 하락 < D (배당의 세금 불이익 반영)
  → 주가 하락폭 / 배당금 비율로 한계 투자자의 세율 추정 가능

한국 적용:
  — 한국 배당소득세 = 15.4% (분리과세 기준)
  — 소액주주 양도소득: 비과세 (2025년 기준, 금투세 유예)
  → Td(15.4%) > Tg(0%) → 이론적으로 저배당 선호
  — 그러나 기관(연기금, 보험)의 배당 수요 존재
  → Clientele 효과 작동: 고배당주는 기관 중심, 저배당주는 개인 중심
```

### 4.5 배당 퍼즐 (The Dividend Puzzle)

Black, F. (1976), *The dividend puzzle*, Journal of Portfolio Management, 2(2), 5-8.

```
배당 퍼즐 (The Dividend Puzzle):

"The harder we look at the dividend picture, the more it seems like a puzzle,
with pieces that just don't fit together."  — Fischer Black (1976)

질문: 배당이 세금상 불리하다면, 왜 기업들은 배당을 지급하는가?

설명 1: 대리인비용 감소 (Jensen 1986)
  — 배당 → FCF 감소 → 경영자의 비효율적 투자 방지
  — 특히 성숙 기업, 저성장 산업에서 중요
  → Doc 33 §2.1 참조

설명 2: 시그널링 (Bhattacharya 1979, §3.3 참조)
  — 배당 증가 → 미래 이익에 대한 경영자 확신의 시그널
  — 세금 비용이 시그널을 credible하게 만듦

설명 3: 행동재무학 — "새장 속의 새" 선호 (Bird-in-Hand)
  — Graham & Dodd (1934): 배당 1원 > 유보이익 1원
  — 투자자들이 확실한 배당을 불확실한 자본이득보다 선호
  — Shefrin & Statman (1984): 자기 통제(self-control) 문제
    → 배당은 소비를 위한 "mental accounting" 규칙 제공
    → "원금은 건드리지 않고 이자(배당)로만 생활"
  — Doc 04 §2 (전망이론: 확실성 효과) 참조

설명 4: 제도적 요인
  — 일부 기관(뮤추얼펀드, 보험사)은 배당 수익만 배분 가능
  — 법적으로 자본이득을 소득으로 인식하지 않는 규정
  — "prudent investor" 규칙: 배당이 없는 주식은 투자 부적격?

설명 5: 거래비용
  — 자가 배당(주식 매도)에는 거래비용 발생
  — 소액 투자자: 주식 매도의 최소 단위(1주) 제약
  — 배당은 이 비용 없이 현금을 제공

퍼즐의 현재 상태:
  완전한 단일 설명은 없으며, 위 요인들의 복합적 작용으로 이해됨
```

## 5. 고급 자본예산 (Advanced Capital Budgeting)

### 5.1 수익성지수 (Profitability Index, PI)

Doc 14 §1.3에서 NPV 규칙을 다루었다. PI는 NPV를 보완하는 도구이다.

```
수익성지수 (Profitability Index):

  PI = PV(미래 현금흐름) / I₀

     = Σ_{t=1}^{n} CF_t / (1+r)^t
       ─────────────────────────────
                  I₀

  의사결정 규칙:
    PI > 1 → 투자 수용 (NPV > 0과 동치)
    PI = 1 → 무차별 (NPV = 0)
    PI < 1 → 투자 기각 (NPV < 0과 동치)

  관계:
    PI = 1 + NPV/I₀
    → PI > 1 ⟺ NPV > 0

수치 예시:
  프로젝트 A: I₀ = 100, PV(CF) = 150 → NPV = 50, PI = 1.50
  프로젝트 B: I₀ = 200, PV(CF) = 280 → NPV = 80, PI = 1.40

  NPV 기준: B > A (80 > 50) → B 선택
  PI 기준:  A > B (1.50 > 1.40) → A 선택

  → 자본 제약이 없으면: NPV 기준 (B 선택)
  → 자본 제약이 있으면: PI 기준으로 순위 결정 (A 우선)
```

### 5.2 자본 할당 (Capital Rationing)

```
자본 할당 문제 (Capital Rationing):

상황: 총 투자 예산 K가 한정되어 있을 때, 어떤 프로젝트 조합을 선택할 것인가?

단일 기간 자본 할당 (Single-Period Rationing):
  max  Σ_j NPV_j · x_j
  s.t. Σ_j I_j · x_j ≤ K
       x_j ∈ {0, 1}  (정수 조건: 프로젝트는 전부 또는 불참)

  — 이것은 0-1 배낭 문제(knapsack problem)와 동일
  — 프로젝트가 분할 가능(divisible)하면: PI 순위로 해결 가능
  — 프로젝트가 분할 불가능하면: 정수 계획법 필요

PI 순위법 (연속 이완):
  (1) 모든 프로젝트를 PI 내림차순으로 정렬
  (2) 예산 K가 소진될 때까지 순서대로 선택
  (3) 경계의 프로젝트는 비율적으로 참여 (분할 가능 시)

수치 예시:
  예산 K = 300

  | 프로젝트 | I₀ | NPV | PI |
  |----------|-----|-----|----|
  | A | 100 | 50 | 1.50 |
  | B | 200 | 80 | 1.40 |
  | C | 150 | 45 | 1.30 |
  | D | 100 | 20 | 1.20 |

  PI 순위: A → B → C → D
  선택: A(100) + B(200) = 300 (예산 소진)
  총 NPV: 50 + 80 = 130

  대안: A(100) + C(150) + 0.5D(50) = 300
  총 NPV: 50 + 45 + 10 = 105 < 130

  → PI 순위법은 분할 가능 시 최적, 불가능 시 근사
```

### 5.3 수정내부수익률 (MIRR)

Doc 14 §1.4에서 IRR의 한계(다중 IRR 문제, 재투자율 가정)를 언급하였다.
MIRR은 이 두 문제를 해결한다.

```
수정내부수익률 (Modified Internal Rate of Return, MIRR):

IRR의 두 가지 문제:
  (1) 현금흐름 부호가 여러 번 바뀌면 다중 IRR 발생
  (2) 중간 현금흐름의 재투자율 = IRR (비현실적 가정)

MIRR의 접근:
  — 양의 현금흐름: WACC(또는 적정 재투자율)로 미래가치 계산
  — 음의 현금흐름: WACC(또는 자본비용)로 현재가치 계산
  — MIRR = 이 둘을 연결하는 수익률

공식:
  Σ_{t|CF_t>0} CF_t · (1+r_reinvest)^(n-t)
  MIRR = [ ──────────────────────────────────── ]^(1/n) - 1
          Σ_{t|CF_t<0} |CF_t| / (1+r_finance)^t

  r_reinvest: 재투자율 (통상 WACC)
  r_finance: 자금조달 비용 (통상 WACC)
  n: 프로젝트 기간

간편 공식 (r_reinvest = r_finance = WACC):
  MIRR = (TV / PV_costs)^(1/n) - 1

  TV = Σ CF_t(positive) · (1+WACC)^(n-t)   ... Terminal Value
  PV_costs = Σ |CF_t(negative)| / (1+WACC)^t

수치 예시:
  현금흐름: [-100, +230, -132]  (부호 변경 2회 → IRR 다중 가능)

  IRR 방정식:
    -100 + 230/(1+r) - 132/(1+r)² = 0
    → r = 10% 또는 r = 20% (두 개의 IRR!)

  MIRR (WACC = 12%):
    TV = 230·(1.12)¹ + 0 = 257.6  (양의 현금흐름의 미래가치)
    PV_costs = 100 + 132/(1.12)² = 100 + 105.18 = 205.18
    MIRR = (257.6/205.18)^(1/2) - 1 = 1.2555^(0.5) - 1 = 12.1%

  → 단일 MIRR = 12.1% (WACC 12%와 비교 가능, NPV ≈ 0에 근접)
```

### 5.4 등가연간연금 (Equivalent Annual Annuity, EAA)

```
등가연간연금 (Equivalent Annual Annuity, EAA):

문제: 수명이 다른 프로젝트 비교 시 NPV만으로는 부적절

예시:
  프로젝트 X: 수명 3년, NPV = 30
  프로젝트 Y: 수명 5년, NPV = 40

  단순 비교: Y > X (40 > 30)
  그러나 X는 3년 후 교체하여 재실행 가능 → 공정한 비교 필요

방법 1: 공통 수명 (Replacement Chain) — 최소공배수로 연장
  X: 3+3+3+3+3 = 15년 (5회 반복), NPV_15 = ?
  Y: 5+5+5 = 15년 (3회 반복), NPV_15 = ?
  → 계산 복잡, 수명의 최소공배수가 클 수 있음

방법 2: EAA — NPV를 연간 등가 금액으로 환산

  EAA = NPV / PVIFA(r, n)

  PVIFA(r, n) = [1 - (1+r)^(-n)] / r    ... 연금현가 계수

  프로젝트 X (r=10%): EAA_X = 30 / PVIFA(10%, 3)
    = 30 / [(1-1.1^(-3))/0.1] = 30 / 2.4869 = 12.06

  프로젝트 Y (r=10%): EAA_Y = 40 / PVIFA(10%, 5)
    = 40 / [(1-1.1^(-5))/0.1] = 40 / 3.7908 = 10.55

  EAA 비교: X > Y (12.06 > 10.55) → X 선택!
  → NPV만으로는 Y가 우월해 보였으나, 연간 기준으로 X가 우월

주의사항:
  — EAA는 프로젝트가 동일 조건으로 무한 반복된다는 가정
  — 기술 변화, 인플레이션, 시장 환경 변화 고려 불가
  — 실무에서는 EAA + 정성적 판단 병행
```

### 5.5 실물옵션 (Real Options) — 개관

```
실물옵션 (Real Options):

핵심 통찰:
  NPV 분석은 "지금 투자" vs "영원히 투자 안 함"의 이분법
  그러나 현실에서 경영자는 미래에 의사결정을 변경할 수 있음
  → 이 유연성(flexibility)에도 가치가 있다

  프로젝트 가치 = NPV + 옵션 가치

주요 실물옵션 유형:

[1] 연기 옵션 (Option to Delay / Defer):
    — 투자를 미룰 수 있는 권리
    — 불확실성이 클수록 연기의 가치 증가
    — NPV > 0이라도 연기가 최적일 수 있음
    — 비유: 미국식 콜옵션 — 만기 전 행사가 항상 최적은 아님

    가치:
      V_delay = max(NPV_now, E[NPV_later · e^(-rt)])
      → Doc 10 §2 (HJB 방정식) 참조: 최적 정지 문제(optimal stopping)

[2] 확장 옵션 (Option to Expand):
    — 초기 투자 후 수요가 좋으면 규모 확대 가능
    — 초기 투자 = 콜옵션의 프리미엄
    — 비유: 유럽식 콜옵션 (행사가 = 추가 투자비용)

    V_expand = NPV_initial + C(V_additional, I_expand, σ, T)

[3] 축소/포기 옵션 (Option to Contract / Abandon):
    — 사업이 부진하면 규모 축소 또는 완전 철수 가능
    — 잔존가치(salvage value) = 풋옵션의 행사가
    — 비유: 미국식 풋옵션

    V_abandon = NPV_initial + P(V_salvage, V_continue, σ, T)

[4] 전환 옵션 (Option to Switch):
    — 투입물(input)이나 산출물(output)을 전환할 수 있는 유연성
    — 예: 석유/가스 전환 발전소, 다품종 생산 공장

[5] 성장 옵션 (Growth Option):
    — 현재의 R&D 투자가 미래 사업 기회를 창출
    — 바이오/IT 기업의 높은 PER은 성장옵션의 가치를 반영

실물옵션의 한국 적용:
  — 반도체 (Samsung, SK Hynix):
    설비 투자 결정에서 연기 옵션 + 확장 옵션이 핵심
    DRAM 업황 사이클에 따른 최적 투자 타이밍
  — 바이오:
    임상 단계별 투자 = 복합 옵션(compound option)
    Phase I → Phase II → Phase III → 상업화
    각 단계에서 중단 가능 = 포기 옵션의 연쇄
  — 건설/부동산:
    토지 보유 = 개발 연기 옵션
    분양 시점 선택 = 연기 옵션의 행사

수학적 프레임워크:
  — Black-Scholes 응용: 간단한 유럽식 옵션 유형
  — 이항 모형 (Binomial Model): 이산적 의사결정 나무
  — 몬테카를로 시뮬레이션: 복잡한 경로 의존적 옵션
  — 편미분방정식: 연속 시간 모형 (Doc 10 §2-3 참조)
  → 상세한 수학적 전개는 향후 Doc 45 (실물옵션 심화)에서 다룰 예정
```

## 6. 배당정책 심화 (Advanced Dividend Policy)

### 6.1 배당 vs 자사주 매입 — 세금 효율성 비교

```
배당과 자사주 매입의 세금 효율성 비교 (한국 세제, 2025):

[배당 (Cash Dividend)]

경로:
  기업 영업이익 → 법인세(Tc) → 세후이익 → 배당 → 배당소득세(Td)

투자자 세후 수령액 (이익 1원 기준):
  1 · (1 - Tc) · (1 - Td)

수치 (Tc=22%, 일반 투자자 Td=15.4%):
  1 · 0.78 · 0.846 = 0.660

수치 (Tc=22%, 금종 종합과세 Td=40%):
  1 · 0.78 · 0.60 = 0.468


[자사주 매입 → 소각 (Share Repurchase → Cancellation)]

경로:
  기업 영업이익 → 법인세(Tc) → 세후이익 → 자사주 매입
  → 주식 수 감소 → 주당 가치 상승 → 양도 시 양도소득세(Tg)

투자자 세후 수령액 (이익 1원, 투자자가 매도 시):
  1 · (1 - Tc) · (1 - Tg)

수치 (Tc=22%, 소액주주 양도소득 비과세 Tg=0%):
  1 · 0.78 · 1.00 = 0.780

수치 (Tc=22%, 대주주 Tg=22%):
  1 · 0.78 · 0.78 = 0.608


세금 효율성 비교표 (이익 1원 기준):

| 방법 | 소액주주 | 일반 투자자(15.4%) | 금종(40%) | 대주주(22%) |
|------|---------|-------------------|----------|------------|
| 배당 | 0.660 | 0.660 | 0.468 | — |
| 자사주→소각 | 0.780 | 0.780 | 0.780 | 0.608 |
| 차이 | +0.120 | +0.120 | +0.312 | — |

결론:
  — 소액주주(양도소득 비과세): 자사주 매입이 절대적으로 유리
    (배당 대비 세후 수령액 18.2% 증가)
  — 금종 대상 고소득 투자자: 자사주 매입의 세금 이점이 극대화
    (배당 대비 세후 수령액 66.7% 증가!)
  — 대주주: 양도소득세율에 따라 다르나, 일반적으로 자사주 매입이 유리

주의: 자사주 매입의 세금 이점은 투자자가 주식을 매도할 때만 실현됨
     (매도 전까지는 미실현 이익 → 세금 이연 효과 추가)
```

### 6.2 한국 상법상 자사주 매입 제한

```
한국 상법 제341조 (자기주식의 취득):

제341조 (자기주식의 취득)
  ① 회사는 다음 각 호의 어느 하나에 해당하는 경우에만
     자기의 명의와 계산으로 자기의 주식을 취득할 수 있다.
     1. 주식을 소각하기 위한 경우
     2. 합병 또는 다른 회사의 영업 전부의 양수로 인한 경우
     ...

제341조의 핵심: 배당가능이익 한도 내에서만 자사주 매입 가능

  배당가능이익 = 순자산액 - (자본금 + 자본준비금 + 이익준비금
                            + 그 결산기에 적립하여야 할 이익준비금
                            + 미실현이익)

  구성요소:
    순자산액: 재무상태표상 자산총계 - 부채총계
    자본금: 발행주식의 액면총액
    자본준비금: 주식발행초과금 등 (상법 459조)
    이익준비금: 배당의 1/10 이상 적립, 자본금의 1/2 한도
    미실현이익: K-IFRS 적용 시 공정가치 평가이익 등

실무적 함의:
  — 배당가능이익이 부족한 기업은 자사주 매입 불가
  — 특히 적자 누적 기업: 배당가능이익 < 0 → 자사주 매입 불가
  — 회계적 이익과 현금의 괴리: 이익이 있어도 현금 부족 시 어려움
  — 대규모 자사주 매입 후에는 배당 여력도 함께 감소

자사주 처분 방법:
  (1) 소각 (Cancellation): 주식 수 영구 감소 → 가장 강한 시그널
  (2) 재매각 (Resale): 시장에 다시 매도 → 시그널 효과 약화
  (3) 임직원 보상 (Treasury stock method): 스톡옵션 행사 시 사용
  → 소각 공시 여부가 시그널 효과를 크게 좌우

최근 한국 동향 (2024-2025):
  — 기업 밸류업 프로그램 (Corporate Value-Up Program)
  — 자사주 매입·소각 활성화 유도
  — PBR 1배 미만 기업의 주주환원 계획 공시 의무화 논의
```

### 6.3 주식배당 vs 현금배당

```
주식배당 (Stock Dividend) vs 현금배당 (Cash Dividend):

[현금배당]
  — 기업에서 주주에게 현금 유출
  — 기업의 현금 감소, 유보이익 감소
  — 주가: 배당금만큼 하락 (이론적)
  — 투자자: 현금 수취, 배당소득세 납부

[주식배당]
  — 추가 주식을 무상으로 배분
  — 기업의 현금 변동 없음
  — 유보이익 → 자본금 전환 (회계적 재분류)
  — 주가: 주식 수 증가에 비례하여 하락 (이론적)
  — 투자자: 주식 수 증가, 주당 가치 하락 (순 효과 = 0, 이론적)

주식배당의 의미:
  — 이론적으로 무가치(MM 배당무관련 하에서)
  — 실무적 동기:
    (1) 현금 보존: 성장 투자에 현금 필요
    (2) 유동성 효과: 주식 수 증가 → 주당 가격 하락 → 거래 활성화
    (3) 심리적 효과: "무언가를 받았다"는 인식
    (4) 기업의 성장 기대 시그널: 현금 대신 재투자한다는 메시지

한국 세법상 주식배당:
  — 주식배당도 배당소득으로 과세 (액면가 기준)
  — 예: 액면 5,000원 × 10주 배당 = 50,000원 배당소득
  — 현금 유입 없이 세금 납부 의무 발생 → 투자자 불만 가능
  — 세금 관점에서는 현금배당보다 불리할 수 있음
```

### 6.6 배당 재투자 계획 (DRIP)

```
배당 재투자 계획 (Dividend Reinvestment Plan, DRIP):

정의:
  배당금을 현금으로 수령하지 않고, 자동으로 해당 기업의 주식을
  추가 매수하는 프로그램

장점 (투자자):
  — 복리 효과 극대화 (배당금이 자동으로 재투자)
  — 거래비용 절감 (수수료 없이 주식 매수)
  — 소수점 단위 주식 매수 가능 (fractional shares)
  — 규율적 투자 습관 형성

장점 (기업):
  — 현금 유출 감소 (배당금이 다시 자본으로 유입)
  — 충성도 높은 장기 주주 확보
  — 주가 안정 효과

한국의 DRIP:
  — 한국에서는 DRIP이 활성화되지 않음
  — 이유:
    (1) 소수점 주식 거래 제도 미비 (미국은 fractional shares 일반화)
    (2) 배당소득세: DRIP에서도 배당 수령 시 과세됨
        → 재투자했는데 세금은 내야 함 → 현금 부담
    (3) 한국 증권 인프라: 자동 재투자 시스템 부족
```

## 요약 (Summary)

### 6개 절의 핵심 메시지 정리

```
§1 Miller (1977) 개인세 모형:
  — MM (1963)의 법인세 모형은 개인세를 무시한 불완전한 분석
  — Miller 공식: V_L = V_U + [1 - (1-Tc)(1-Ts)/(1-Td)] · D
  — 한국: Ts ≈ Td이면 GL ≈ Tc·D (법인세 모형이 합리적 근사)
  — 소액주주(Tg=0): GL < Tc·D → 자기자본의 세금 이점 존재

§2 대리인비용 자본구조:
  — 자기자본의 대리인비용 ↔ 부채의 대리인비용의 균형에서 최적 D/E 결정
  — Jensen (1986): 부채의 규율 효과 → 성숙 기업의 고레버리지 정당화
  — 한국 재벌: 지배주주-소액주주 대리인비용이 핵심 이슈
  — 확장 상충이론: 세금 + 파산비용 + 대리인비용의 동시 최적화

§3 시그널링 이론:
  — Ross (1977): 부채 수준이 기업 품질의 시그널
  — Bhattacharya (1979): 배당의 세금 비용이 시그널을 credible하게 함
  — Lintner (1956): 배당 평활화 → 삭감의 정보 함량 > 증가의 정보 함량
  — Myers-Majluf (1984): 정보 비대칭 → 자본조달 서열의 이론적 정당화
  — KRX: 유상증자(-), 자사주 매입(+), 배당 변경(비대칭적) 공시효과

§4 MM 제3명제 — 배당무관련:
  — 완전 자본시장에서 배당 정책은 기업가치에 영향 없음
  — 자가 배당(homemade dividends)으로 투자자가 선호하는 현금흐름 생성 가능
  — 세금 고객효과: 세율별 투자자 그룹이 선호하는 배당 수준에 매칭
  — 배당 퍼즐: 대리인비용, 시그널링, 행동적 요인의 복합 설명

§5 고급 자본예산:
  — PI: NPV의 보완, 자본 제약 하에서 프로젝트 순위 결정
  — MIRR: IRR의 다중해·재투자율 문제 해결
  — EAA: 수명이 다른 프로젝트의 공정 비교
  — 실물옵션: NPV를 넘어서는 경영 유연성의 가치 인식

§6 배당정책 심화:
  — 한국 세제 하에서 자사주 매입이 배당보다 세금 효율적
  — 상법상 자사주 매입 제한: 배당가능이익 한도
  — KRX 섹터별 배당 패턴: 잔여 배당, FCF 가설, 시그널링의 복합 작용
  — 주주환원 정책 변화: Korea Discount 해소를 위한 구조적 전환기
```
